Introduction

Group actions formalize the idea of a group "acting" on a set. The orbit-stabilizer theorem and class equation are powerful counting tools. Sylow's theorems use these to probe the subgroup structure of any finite group — they are among the most applicable results in finite group theory.

Group Actions

Group Action

A group action of G on a set X is a map G × X → X, (g,x) ↦ g·x, satisfying:

  1. e·x = x for all x ∈ X (identity acts trivially)
  2. (gh)·x = g·(h·x) for all g,h ∈ G, x ∈ X (compatibility)

Orbits & Stabilizers

Orbit and Stabilizer

Orbit of x ∈ X: Orb(x) = G·x = {g·x : g ∈ G}

Stabilizer (isotropy group) of x: Stab(x) = Gₓ = {g ∈ G : g·x = x} ≤ G

Orbit-Stabilizer Theorem
|G| = |Orb(x)| · |Stab(x)|

Equivalently: |Orb(x)| = [G : Stab(x)] = |G|/|Stab(x)|.

Burnside's Lemma (counting orbits)

The number of orbits of G acting on X is:

|X/G| = (1/|G|) · Σ_{g∈G} |X^g|

where X^g = {x ∈ X : g·x = x} = fixed points of g. Used for combinatorial counting.

Conjugacy Classes

Under conjugation action g·x = gxg⁻¹: orbits are conjugacy classes. Stabilizer of x is the centralizer C(x) = {g: gx=xg}.

|conj class of x| = [G:C(x)] = |G|/|C(x)|.

Cauchy's Theorem & Class Equation

Class Equation

Partition G into conjugacy classes. The center Z(G) consists of all size-1 classes. For the remaining classes with representatives x₁,...,xₖ (outside Z(G)):

|G| = |Z(G)| + Σᵢ [G : C(xᵢ)]

Each term [G:C(xᵢ)] > 1 divides |G|.

Cauchy's Theorem

If p is a prime and p | |G|, then G has an element of order p (equivalently, a subgroup of order p).

p-groups

G is a p-group if |G| = pⁿ for some prime p and n ≥ 1.

  • Every p-group has non-trivial center: Z(G) ≠ {e}.
  • Every group of order p² is abelian (isomorphic to ℤ_{p²} or ℤₚ × ℤₚ).

Sylow's Theorems

Let |G| = pⁿm where p is prime and gcd(p,m) = 1. A Sylow p-subgroup is a subgroup of order pⁿ.

Sylow's First Theorem (Existence)

Every finite group G has a Sylow p-subgroup for each prime p dividing |G|.

More generally: for every k with 0 ≤ k ≤ n, G has a subgroup of order pᵏ.

Sylow's Second Theorem (Conjugacy)

All Sylow p-subgroups of G are conjugate to each other: if P, Q are both Sylow p-subgroups, then Q = gPg⁻¹ for some g ∈ G.

Sylow's Third Theorem (Number of Sylow Subgroups)

Let nₚ = number of Sylow p-subgroups of G. Then:

  • nₚ ≡ 1 (mod p)
  • nₚ | m (where |G| = pⁿm)
  • nₚ = [G : N_G(P)] where N_G(P) is the normalizer of any Sylow p-subgroup P

Unique Sylow ⟹ Normal: if nₚ = 1, the unique Sylow p-subgroup is normal in G.

Application: Groups of order 15

|G|=15=3·5. n₃|5 and n₃≡1(mod 3), so n₃=1. n₅|3 and n₅≡1(mod 5), so n₅=1. Both Sylow subgroups are unique, hence normal. G≅ℤ₃×ℤ₅≅ℤ₁₅.

Application: No simple group of order 56

|G|=56=2³·7. n₇|8 and n₇≡1(mod 7), so n₇∈{1,8}. If n₇=1, the Sylow 7-subgroup is normal → G not simple. If n₇=8: 8 Sylow 7-subgroups give 8·6=48 elements of order 7. The remaining 8=2³ elements form a unique Sylow 2-subgroup, which is normal → G not simple.
