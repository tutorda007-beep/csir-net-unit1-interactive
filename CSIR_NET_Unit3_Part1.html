Types of First Order ODEs

Separable Equations Basic

Form: dy/dx = f(x)·g(y) Method: g(y)⁻¹ dy = f(x) dx → integrate both sides Example: dy/dx = xy dy/y = x dx ln|y| = x²/2 + C y = Ae^(x²/2) (A = ±e^C)

Autonomous: dy/dx = g(y) — separable with f(x)=1. Equilibria at g(y)=0.

Linear First Order Basic

Form: dy/dx + P(x)y = Q(x) Integrating Factor: μ(x) = e^(∫P dx) Solution: y = (1/μ)∫μQ dx + C/μ Example: dy/dx + y = eˣ μ = eˣ, eˣy = ∫e²ˣ dx = e²ˣ/2 + C y = eˣ/2 + Ce⁻ˣ

The integrating factor μ makes the left side d(μy)/dx.

Exact Equations Intermediate

Form: M(x,y)dx + N(x,y)dy = 0 Exact iff: ∂M/∂y = ∂N/∂x Solution F(x,y) = C where: ∂F/∂x = M, ∂F/∂y = N F = ∫M dx + g(y), then find g(y) from ∂F/∂y = N Non-exact: find integrating factor μ(x) or μ(y)

Test: If M_y = N_x, the equation is exact and an F exists.

Bernoulli & Homogeneous Intermediate

Bernoulli: dy/dx + P(x)y = Q(x)yⁿ Substitution: v = y^(1−n) → linear in v dv/dx + (1−n)P(x)v = (1−n)Q(x) Homogeneous: dy/dx = f(y/x) Substitution: v = y/x, y = vx v + x dv/dx = f(v) → separable

Note: Bernoulli with n=0 is linear; n=1 is separable.

Electric field lines and equipotential curves are orthogonal trajectories of each other.

The substitution to reduce dy/dx + y/x = y³ to a linear ODE is:
Av = y²
Bv = y⁻²
Cv = y³
Dv = 1/y
✓ Correct Answer: (B) — v = y⁻²
Bernoulli with n=3: substitute v = y^(1−n) = y^(1−3) = y⁻². Then dv/dx = −2y⁻³ dy/dx. Multiply the ODE by −2y⁻³: dv/dx − 2v/x = −2. This is linear in v with P(x)=−2/x, Q(x)=−2. Integrating factor: x⁻², giving d(x⁻²v)/dx = −2x⁻², so v = Cx² − x² → y⁻² = Cx² − x².

Initial Value ProblemAdvanced
The solution of dy/dx = y², y(0) = 1, exists on the interval:
A(−∞, ∞)
B(−1, 1)
C(−∞, 1)
D(0, 2)
✓ Correct Answer: (C) — (−∞, 1)
Separate: dy/y² = dx → −1/y = x + C. IVP y(0)=1: −1=C, so 1/y = 1−x, i.e., y = 1/(1−x). This blows up (finite-time blow-up) at x=1. Solution exists for x<1, i.e., on (−∞,1). This illustrates that even for smooth f(x,y)=y², global existence may fail. The Picard-Lindelöf theorem guarantees only local existence.

Method of Undetermined Coefficients

Particular Solution Forms Intermediate

Applies to: ay''+by'+cy = g(x) when g(x) is: Polynomial pₙ(x): Yₚ = Aₙxⁿ+…+A₀ Exponential e^(αx): Yₚ = Ae^(αx) Trig cos(βx)/sin(βx): Yₚ = Acos(βx)+Bsin(βx) Product combinations: multiply above forms Modification Rule: if trial Yₚ overlaps y_h, multiply by x (or x² if root has mult. 2)

Cannot use for: tan x, ln x, sec x — use variation of parameters instead.

Variation of Parameters Intermediate

For y''+P(x)y'+Q(x)y = R(x) with y₁,y₂ known: W = y₁y₂'−y₂y₁' (Wronskian, W≠0) Yₚ = −y₁∫(y₂R/W)dx + y₂∫(y₁R/W)dx Works for ANY R(x) (not just special forms). More general but often harder to integrate.

System form: u₁'y₁ + u₂'y₂ = 0, u₁'y₁' + u₂'y₂' = R. Solve for u₁', u₂', then integrate.

Operator Method & Reduction

Operator Method (D-operator) Advanced

D ≡ d/dx; Dⁿ ≡ dⁿ/dxⁿ f(D)y = R(x) Key shortcuts: 1/f(D) e^(ax) = e^(ax)/f(a), if f(a)≠0 1/f(D) e^(ax) = x·e^(ax)/f'(a), if f(a)=0 1/(D²+a²) sin(ax) = −x·cos(ax)/(2a) 1/(D²+a²) cos(ax) = x·sin(ax)/(2a)

The operator method gives a systematic shortcut for computing Yₚ without integration tables.

Reduction of Order Advanced

If y₁ is one known solution of y''+Py'+Qy=0: Set y₂ = v(x)·y₁(x), substitute: v'' y₁ + v'(2y₁'+Py₁) = 0 Let w = v': w'y₁ + w(2y₁'+Py₁) = 0 Separable! Gives w, then v = ∫w dx Abel's formula: W(x) = W(x₀)·e^(−∫P dx)

Reduction of order always produces y₂ (independent of y₁) using Abel's identity.

Wronskian & Linear Independence

Wronskian Definition Basic

For functions y₁, y₂: W(y₁,y₂) = |y₁ y₂ | |y₁' y₂'| = y₁y₂' − y₂y₁' For y₁,y₂,...,yₙ: W = det([yᵢ^(j−1)]) (i,j = 1 to n) Key theorem: W ≡ 0 on I → y₁,y₂ linearly DEPENDENT W ≠ 0 at some point → INDEPENDENT

Caution: W=0 at a point does NOT imply dependence. W must be identically zero.

Abel's Identity Intermediate

For y''+P(x)y'+Q(x)y=0 with solutions y₁,y₂: Abel's identity: W(x) = W(x₀) · exp(−∫_{x₀}^{x} P(t) dt) Consequence: W is either always 0 or never 0 on I (for solutions of a linear ODE — not arbitrary functions!) Example: y''−3y'+2y=0, P(x)=−3 W(x) = Ce^(3x) (always nonzero for C≠0)

Abel's identity confirms that W(x)=0 at one point iff W≡0 (for ODE solutions).

Euler-Cauchy Equation

Euler-Cauchy ODE Intermediate

Form: x²y'' + pxy' + qy = 0 (x > 0) Substitution: x = eᵗ (i.e., t = ln x), let D = d/dt: x²y'' = D(D−1)y xy' = Dy Transformed: [D(D−1) + pD + q]y = 0 → (D²+(p−1)D+q)y = 0 (constant coefficients!) Alternatively: try y = xʳ Indicial equation: r(r−1)+pr+q = r²+(p−1)r+q = 0

Solutions by root type:

  • Real distinct r₁≠r₂: y = C₁x^(r₁) + C₂x^(r₂)
  • Repeated r₁=r₂=r: y = x^r(C₁ + C₂ ln x)
  • Complex r = α±βi: y = x^α[C₁cos(β ln x)+C₂sin(β ln x)]
Existence & Uniqueness

Picard-Lindelöf Theorem Intermediate

IVP: y' = f(x,y), y(x₀) = y₀ If f is continuous and satisfies Lipschitz in y: |f(x,y₁)−f(x,y₂)| ≤ L|y₁−y₂| in a rectangle R: |x−x₀|≤a, |y−y₀|≤b Then ∃ unique solution on |x−x₀| ≤ min(a, b/M) where M = max|f| on R.

Peano's theorem: Continuity alone (no Lipschitz) → existence but possibly NOT uniqueness.

Singular Points & Regularity Advanced

For y''+P(x)y'+Q(x)y=0: Ordinary point: P, Q analytic at x₀ Regular singular point: (x−x₀)P and (x−x₀)²Q are analytic at x₀ (Frobenius method applies) Irregular singular point: otherwise Euler-Cauchy x²y''+pxy'+qy=0: x=0 is a regular singular point (xP(x)=p, x²Q(x)=q are analytic)

Frobenius method (see Part 2) handles regular singular points via power series y = x^r Σaₙxⁿ.
