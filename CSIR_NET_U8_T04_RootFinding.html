Introduction

This finale covers root finding (solving f(x)=0), numerical linear algebra (Ax=b), and numerical ODE solvers. These are core computational mathematics topics tested in CSIR NET.

Root Finding Methods

Bisection Method
Given f(a)f(b) < 0, set c=(a+b)/2. If f(c)=0: done. If f(a)f(c)<0: set b=c. Else: set a=c. Error after n steps: ≤(b−a)/2ⁿ. Order: p=1, rate C=1/2.
Newton-Raphson Method
x_{n+1} = xₙ − f(xₙ)/f′(xₙ) Order p=2 (quadratic) near simple root. Fails if f′(x*)=0. Requires: good initial guess, f′≠0.
Secant & Fixed-Point
Secant: x_{n+1}=xₙ − f(xₙ)(xₙ−x_{n−1})/(f(xₙ)−f(x_{n−1})) Order p=(1+√5)/2≈1.618 Fixed-point: xₙ₊₁=g(xₙ) Converges if |g′(x*)|<1, rate C=|g′(x*)|

Numerical Linear Algebra

Gaussian Elimination

Forward elimination to upper triangular form, then back substitution. Complexity: O(n³). With partial pivoting: swap rows to put largest |aᵢⱼ| in pivot position (improves stability).

LU Decomposition
A = LU (L lower triangular, U upper triangular) Solve Ax=b: Ly=b (forward sub), then Ux=y (back sub) Advantage: once A=LU, solve for multiple b efficiently.
Iterative Methods
Jacobi: x_i^(k+1) = (bᵢ − Σ_{j≠i}aᵢⱼxⱼ^(k))/aᵢᵢ Gauss-Seidel: uses updated xⱼ^(k+1) as soon as computed (faster) Converge if A is diagonally dominant: |aᵢᵢ| > Σ_{j≠i}|aᵢⱼ|

Numerical ODEs

Euler Methods for y′=f(t,y), y(t₀)=y₀
Forward Euler: y_{n+1} = yₙ + h·f(tₙ,yₙ) Order O(h) global Backward Euler: y_{n+1} = yₙ + h·f(t_{n+1},y_{n+1}) (implicit, O(h)) Midpoint: y_{n+1} = yₙ + h·f(tₙ+h/2, yₙ+h/2·f(tₙ,yₙ)) O(h²)
Runge-Kutta 4 (RK4)
k₁ = h·f(tₙ, yₙ) k₂ = h·f(tₙ+h/2, yₙ+k₁/2) k₃ = h·f(tₙ+h/2, yₙ+k₂/2) k₄ = h·f(tₙ+h, yₙ+k₃) y_{n+1} = yₙ + (k₁+2k₂+2k₃+k₄)/6 Global error: O(h⁴). Local truncation error: O(h⁵).
Stability

For y′=λy (test equation, Re(λ)<0): Euler stable if |1+hλ|<1. For hλ real and negative, need h|λ|<2. Backward Euler stable for all h>0 (A-stable). RK4 has a finite stability region.
