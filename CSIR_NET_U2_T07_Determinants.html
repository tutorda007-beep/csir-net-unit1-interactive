Introduction

The theory of linear systems Ax=b is completely governed by the rank of the coefficient matrix A and the augmented matrix [A|b]. Determinants provide a powerful computational and theoretical tool for characterizing invertibility and solving small systems.

Determinants

Definition (Leibniz Formula)

For an n×n matrix A, the determinant is:

det(A) = Σ_{σ∈Sₙ} sgn(σ) · a_{1,σ(1)} · a_{2,σ(2)} · ... · a_{n,σ(n)}

where the sum runs over all permutations σ of {1,...,n} and sgn(σ) = ±1 (sign of permutation).

Cofactor Expansion

Expansion along Row i (Laplace Expansion)
det(A) = Σⱼ aᵢⱼ · Cᵢⱼ where Cᵢⱼ = (−1)^(i+j) Mᵢⱼ

Mᵢⱼ = minor (det of matrix with row i and col j removed). Cᵢⱼ = cofactor.

2×2 and 3×3 Determinants

2×2

det([[a,b],[c,d]]) = ad − bc

3×3 (Sarrus' Rule)

det(A) = a(ei−fh) − b(di−fg) + c(dh−eg)

for A = [[a,b,c],[d,e,f],[g,h,i]].

Properties of Determinants

Fundamental Properties
  1. Multilinear: Linear in each row (or column) separately
  2. Alternating: det changes sign when two rows are swapped
  3. Normalization: det(I) = 1
  4. Row of zeros: det = 0 if A has a zero row
  5. Proportional rows: det = 0 if two rows are proportional
  6. Row operation: Adding multiple of one row to another leaves det unchanged
  7. Scalar factor: det(αA) = αⁿ det(A) for n×n matrix A

Linear Systems Ax = b

Types of Solutions
  • Unique solution: rank(A) = rank([A|b]) = n (number of unknowns)
  • Infinitely many solutions: rank(A) = rank([A|b]) < n
  • No solution (inconsistent): rank(A) < rank([A|b])

Gaussian Elimination

Row Reduction to REF/RREF

Perform elementary row operations on the augmented matrix [A|b]:

  1. R_i ↔ R_j: Swap two rows (det changes sign)
  2. R_i ← α·R_i: Scale a row by α≠0 (det multiplied by α)
  3. R_i ← R_i + α·R_j: Add multiple of one row to another (det unchanged)

Reduce to Row Echelon Form (REF): zeros below each pivot. Then back-substitute. RREF (reduced REF): additionally, each pivot = 1 and zeros above as well.

Homogeneous Systems

Ax = 0 (Homogeneous)

Always has the trivial solution x=0. Has non-trivial solutions iff rank(A) < n iff det(A)=0 (square matrix case). The solution set (null space) has dimension n−rank(A) = nullity(A).

Rank & Rouché-Capelli Theorem

Rank of a Matrix

rank(A) = number of pivots in REF = dimension of column space = dimension of row space. By rank-nullity: rank(A) + nullity(A) = n (number of columns).

Rouché-Capelli (Kronecker-Capelli) Theorem

The system Ax=b (m equations, n unknowns) is consistent iff:

rank(A) = rank([A|b])

When consistent, the number of free variables = n − rank(A), and the solution space is an affine subspace of dimension n − rank(A).

rank(A)rank([A|b])Solutions
rr∞^(n−r) solutions (n−r free variables)
rr+1 (> r)No solution (inconsistent)
r = nr = nUnique solution
Example

3×3 system Ax=b with rank(A)=2, rank([A|b])=2, n=3: consistent with n−r=1 free variable → infinitely many solutions (a line in ℝ³).

Cramer's Rule & Matrix Inverse

Cramer's Rule

If A is n×n with det(A) ≠ 0, the unique solution to Ax=b is:

xⱼ = det(Aⱼ)/det(A)

where Aⱼ is A with the j-th column replaced by b. Useful for theoretical purposes; inefficient for large n.

Inverse via Adjugate
A⁻¹ = (1/det(A)) · adj(A)

where adj(A) = [Cⱼᵢ] is the adjugate (classical adjoint) = transpose of the cofactor matrix.

2×2: A⁻¹ = (1/(ad−bc))·[[d,−b],[−c,a]]

Key results for CSIR NET
📊

Determinant

det(AB)=det(A)det(B). Row swap: ×(−1). Scale row by c: ×c. Add multiple: unchanged.

🔍

Rouché-Capelli

Consistent iff rank(A)=rank([A|b]). Free vars = n−rank(A). Inconsistent: ranks differ.

📐

Rank-Nullity

rank(A) + nullity(A) = n (cols). Row rank = Col rank. Max rank = min(m,n).

⚡

Invertibility

Equivalent: det(A)≠0, rank=n, trivial null space, unique solution to Ax=b for all b.

🔑

Cramer's Rule

xⱼ=det(Aⱼ)/det(A). Only for square invertible A. Geometric: area/volume via det.

⁻¹

Inverse Formula

A⁻¹=(1/det(A))adj(A). For 2×2: swap diagonal, negate off-diagonal, divide by det.
