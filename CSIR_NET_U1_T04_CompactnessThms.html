Unit 1 — Analysis · Topic 04 of 11
Bolzano-Weierstrass & Heine-Borel
Two pillars of compactness theory — subsequential limits, open covers, and closed-bounded sets in ℝⁿ
CSIR NET Core Compactness Theory Intermediate 65 min Study

Learning Objectives

  1. Define accumulation point (cluster point) and identify them for given sets
  2. State and prove the Bolzano-Weierstrass Theorem (BWT) for sequences and sets
  3. Define compactness via open covers and understand its importance
  4. State and apply the Heine-Borel Theorem: compact ⟺ closed + bounded in ℝⁿ
  5. Distinguish where BWT and HBT hold vs fail (in general metric spaces vs ℝⁿ)
🎯
CSIR NET Frequency: BWT and HBT together appear in 2–3 questions per exam. Common patterns: "which sets are compact?", "does this sequence have a convergent subsequence?", and connections to continuity (EVT). HBT is the key characterization of compact sets in ℝⁿ.

📍 Accumulation Points

·
Concept 1
Cluster Points & Derived Sets
Accumulation Point (Cluster Point / Limit Point)
A point p is an accumulation point of a set S ⊆ ℝ if every neighborhood of p contains a point of S different from p:

∀ε > 0: (p−ε, p+ε) ∩ (S ∖ {p}) ≠ ∅

The set of all accumulation points of S is the derived set S'.
⚡
Key Difference: An accumulation point need NOT belong to S. The point 0 is a cluster point of S = (0,1) but 0 ∉ S. A set is closed iff it contains all its accumulation points (S' ⊆ S).

{1/n : n ∈ ℕ}{0}No (0 is a cluster point not in S)
S = {0} ∪ {1/n : n∈ℕ}{0}Yes (0 ∈ S)
Bolzano-Weierstrass for Sets
Every infinite bounded subset of ℝ has at least one accumulation point.
This is the "set version" of BWT. Example: the infinite bounded set {1/n : n ∈ ℕ} ⊂ [0,1] has 0 as an accumulation point. An unbounded infinite set may have no accumulation point (e.g., ℕ is unbounded and has none).

🔄 Bolzano-Weierstrass Theorem

BW
Concept 2
The BW Theorem for Sequences
Bolzano-Weierstrass Theorem (Sequence Version)
Every bounded sequence in ℝ has a convergent subsequence.
Proof via Nested Intervals: Let (aₙ) be bounded: aₙ ∈ [a, b] for all n.
Bisect [a, b] into [a, (a+b)/2] and [(a+b)/2, b]. At least one half contains infinitely many terms — call it I₁.
Bisect I₁, pick the half with infinitely many terms — call it I₂. Continue: get nested closed intervals I₁ ⊇ I₂ ⊇ … with |Iₙ| = (b−a)/2ⁿ → 0.
By NIT, ⋂ Iₙ = {p} for some p. Pick aₙₖ ∈ Iₖ — this is a convergent subsequence with aₙₖ → p. □

Subsequences — Key Definitions

Subsequence
A subsequence of (aₙ) is (aₙₖ) where n₁ < n₂ < n₃ < … is a strictly increasing sequence of indices.

Key fact: If aₙ → L then every subsequence also converges to L.
Converse: If (aₙ) has two subsequences converging to different limits, then (aₙ) diverges.

🗜 Compactness

⊂
Concept 3
Open Covers and Compact Sets
Open Cover
A collection {Uα} of open sets is an open cover of K if K ⊆ ⋃ Uα.
Compact Set
A set K is compact if every open cover has a finite subcover:
For every open cover {Uα} of K, there exist finitely many α₁,…,αₙ such that K ⊆ Uα₁ ∪ … ∪ Uαₙ.

Compactness is the "finite approximation" property — no matter how you cover the set with open sets, finitely many suffice. This is what allows "global" arguments from "local" information.

Properties of Compact Sets
1. Every compact set is bounded and closed.
2. A closed subset of a compact set is compact.
3. The continuous image of a compact set is compact.
4. A compact set in a Hausdorff space is closed.
5. A finite union of compact sets is compact.
6. An arbitrary intersection of compact sets is compact.
🔑
Why compactness matters: On a compact set, continuous functions attain their maximum and minimum (EVT), are uniformly continuous, and many other nice properties hold that fail on non-compact sets. Compactness is the "taming" condition for infinite sets.

📦 Heine-Borel Theorem

HB
Concept 4
The Heine-Borel Theorem
Heine-Borel Theorem
A subset K of ℝⁿ is compact if and only if K is both closed AND bounded.

In ℝ: K ⊂ ℝ is compact ⟺ K is closed and bounded.
In ℝⁿ: Same — every closed bounded subset of ℝⁿ is compact.
Proof idea (⇒): Compact ⟹ bounded: cover K with balls {B(x,1): x∈K} — finite subcover gives boundedness. Compact ⟹ closed: take limit point p of K; every neighborhood contains points of K; build open cover excluding a shrinking neighborhood of p — no finite subcover unless p ∈ K.
(⇐): Closed and bounded ⟹ compact (in ℝ): K ⊆ [a,b] for some a,b. If {Uα} covers K with no finite subcover, bisect [a,b] — at least one half has no finite subcover. Repeat → Nested Interval → point in K not covered — contradiction.

Compact vs Non-Compact Sets in ℝ

SetClosed?Bounded?Compact?Reason

[0, ∞)YesNoNoNot bounded; {[0,n)} has no finite subcover
ℝYesNoNoNot bounded

⚠️
HBT is special to ℝⁿ! In general metric spaces, closed + bounded does NOT imply compact. Example: In ℓ² (infinite-dimensional Hilbert space), the unit ball is closed and bounded but NOT compact. HBT characterizes "finite-dimensional" compactness.

⚖️ BWT vs HBT — Full Comparison

Bolzano-Weierstrass Theorem
  • Statement: Every bounded sequence in ℝ has a convergent subsequence
  • Also: Every infinite bounded set in ℝ has an accumulation point
  • Key tool: Nested interval bisection
  • Uses: Extracting convergent subsequences; proving Cauchy criterion
  • Extends to: ℝⁿ (every bounded sequence in ℝⁿ has a convergent subsequence)
  • Fails for: Unbounded sequences; infinite-dimensional spaces
Heine-Borel Theorem
  • Statement: K ⊂ ℝⁿ compact ⟺ K closed and bounded
  • Also: K compact ⟺ every sequence in K has a convergent subsequence in K (sequential compactness)
  • Key tool: Open cover finite subcover argument
  • Uses: Extreme Value Theorem; uniform continuity
  • Extends to: ℝⁿ for all n ∈ ℕ
  • Fails for: Infinite-dimensional spaces (unit ball not compact in ℓ²)
Equivalence in ℝⁿ
For K ⊆ ℝⁿ, the following are equivalent:
(i) K is compact (every open cover has finite subcover)
(ii) K is closed and bounded (Heine-Borel)
(iii) Every sequence in K has a subsequence converging to a point in K (sequential compactness)
(iv) Every infinite subset of K has an accumulation point in K (Bolzano-Weierstrass property)
In general metric spaces, (i) ⟺ (iii) ⟺ (iv) always hold. But (i) ⟺ (ii) only holds in finite-dimensional spaces like ℝⁿ.

✏️ Worked Examples

Example 1 — Apply BWT
Show that the sequence aₙ = sin(n) has a convergent subsequence.
Since −1 ≤ sin(n) ≤ 1 for all n, the sequence (sin(n)) is bounded. By the Bolzano-Weierstrass Theorem, every bounded sequence in ℝ has a convergent subsequence. Hence (sin(n)) has a convergent subsequence.
Note: Finding the actual subsequence is hard (requires density of {n mod 2π} in [0, 2π]), but BWT guarantees its existence without construction.
Example 2 — Compact Set Identification
Which of the following subsets of ℝ are compact?
(a) [−3, 5] (b) (0, 1] (c) {2, 4, 6, 8} (d) ℤ (e) [0,1] ∪ [2,3]
(a) [−3, 5]: Closed ✓, Bounded ✓ → Compact (HBT)
(b) (0, 1]: Not closed (0 is a limit point not in the set) → Not compact. Cover: {(1/n, 2)} has no finite subcover.
(c) {2, 4, 6, 8}: Finite set → Always closed and bounded → Compact
(d) ℤ: Closed ✓ but unbounded → Not compact
(e) [0,1] ∪ [2,3]: Closed ✓ (finite union of closed sets), Bounded ✓ → Compact
Example 3 — Open Cover with No Finite Subcover
Show directly that (0, 1) is not compact by exhibiting an open cover with no finite subcover.
Consider the open cover: Uₙ = (1/n, 1) for n = 2, 3, 4, …
It covers (0,1): For any x ∈ (0,1), by Archimedean property ∃n with 1/n < x, so x ∈ Uₙ. Thus ⋃Uₙ ⊇ (0,1). ✓
No finite subcover: Any finite subcollection {Uₙ₁, …, Uₙₖ} has max index N = max(n₁,…,nₖ). Then UN covers only (1/N, 1), missing (0, 1/N]. So no finite subcover covers all of (0,1). □
Example 4 — Accumulation Point
Find all accumulation points of S = {(−1)ⁿ + 1/n : n ∈ ℕ}.
Terms: For odd n: (−1)ⁿ + 1/n = −1 + 1/n → −1. For even n: 1 + 1/n → 1.
The terms of S form two sequences approaching −1 and +1.
Accumulation points: S' = {−1, 1}.
Check: Every neighborhood of −1 contains −1 + 1/n for large odd n (which is in S and ≠ −1). Similarly for 1. These are the only cluster points.
Example 5 — CSIR NET Style
Let K = {(x,y) ∈ ℝ² : x² + y² ≤ 1}. Is K compact?
K is the closed unit disk in ℝ².
Closed: K = f⁻¹([0,1]) where f(x,y) = x²+y² is continuous and [0,1] is closed. So K is closed. ✓
Bounded: Every point satisfies ‖(x,y)‖ = √(x²+y²) ≤ 1, so K ⊆ B(0,1). ✓
By the Heine-Borel Theorem (in ℝ²), K is compact. Consequence: any continuous function on K attains its maximum and minimum.

📌 Summary & CSIR NET Tips

BWT — Core

  • Bounded sequence ⟹ convergent subsequence
  • Infinite bounded set ⟹ has accumulation point
  • Proved via nested interval bisection
  • Holds in ℝⁿ for all finite n
  • BWT is equivalent to completeness of ℝ

HBT — Core

  • In ℝⁿ: compact ⟺ closed + bounded
  • Compact ⟹ every sequence has convergent subseq. in K
  • FAILS in infinite-dimensional spaces
  • Compact ⟹ EVT holds; f continuous ⟹ unif. continuous
  • Finite union of compact sets is compact

CSIR Exam Q-Types

  • Identify compact/non-compact sets in ℝ and ℝ²
  • Apply BWT to show convergent subsequence exists
  • Open cover examples with no finite subcover
  • EVT applications on compact sets
  • Accumulation point calculations

Common Mistakes

  • Open intervals are NOT compact (missing boundary)
  • Unbounded closed sets NOT compact (e.g., [0,∞))
  • Bounded ≠ compact (need closed too)
  • HBT fails in infinite dimensions!
  • FIP for compact sets → non-empty intersection
