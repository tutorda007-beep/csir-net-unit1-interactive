Introduction

Sturm-Liouville (S-L) theory provides a unified framework for boundary value problems arising in mathematical physics (heat equation, wave equation, Schrödinger equation). The key results are: real eigenvalues, orthogonal eigenfunctions, and completeness — allowing arbitrary functions to be expanded in eigenfunctions.

The Sturm-Liouville Problem

S-L Equation
−(p(x)y′)′ + q(x)y = λw(x)y, a < x < b

with boundary conditions: α₁y(a)+α₂y′(a)=0 and β₁y(b)+β₂y′(b)=0.

Here: p(x)>0, w(x)>0 (weight function), q(x) real, p,q,w continuous on [a,b].

Self-Adjoint / Symmetric Operator

The S-L operator Ly = −(py′)′+qy is self-adjoint (symmetric) in the Hilbert space L²([a,b], w dx):

⟨Lu, v⟩_w = ⟨u, Lv⟩_w where ⟨f,g⟩_w = ∫_a^b f(x)g(x)w(x)dx

This follows from integration by parts plus the boundary conditions.

Converting to S-L Form

Any second-order linear ODE y′′+P(x)y′+(Q(x)+λR(x))y=0 can be written in S-L form by multiplying by the integrating factor μ=e^(∫P dx).

Eigenvalue Properties

Main Theorem (Regular S-L Problem)
  1. Eigenvalues are real.
  2. Eigenvalues form a discrete sequence λ₁ < λ₂ < λ₃ < … → +∞.
  3. Each eigenvalue has exactly one (linearly independent) eigenfunction φₙ (up to scalar multiple).
  4. Eigenfunctions for distinct eigenvalues are orthogonal w.r.t. weight w:
    ⟨φₘ, φₙ⟩_w = ∫_a^b φₘ(x)φₙ(x)w(x)dx = 0 if m≠n
  5. The eigenfunctions are complete: any piecewise smooth f on [a,b] can be expanded as f(x) = Σcₙφₙ(x) where cₙ = ⟨f,φₙ⟩_w/‖φₙ‖²_w.
Oscillation Theorem

The nth eigenfunction φₙ has exactly n−1 zeros in the open interval (a,b). (φ₁ has 0 zeros, φ₂ has 1 zero, etc.)

Eigenfunction Expansion

Generalized Fourier Series

Given a regular S-L problem with eigenfunctions {φₙ} and weight w, any piecewise smooth f has:

f(x) = Σ_{n=1}^∞ cₙφₙ(x), cₙ = ∫_a^b f(x)φₙ(x)w(x)dx / ∫_a^b φₙ²(x)w(x)dx

The series converges in the L²_w norm (Parseval's identity: Σcₙ²‖φₙ‖² = ‖f‖²_w).

Example: Fourier Sine Series

y′′+λy=0, y(0)=y(π)=0: eigenfunctions sin(nx), n=1,2,… f(x)=Σbₙsin(nx) where bₙ=(2/π)∫₀^π f(x)sin(nx)dx.

Parseval's Identity
‖f‖²_w = ∫_a^b |f(x)|²w(x)dx = Σ_{n=1}^∞ |cₙ|²‖φₙ‖²_w

This is the completeness of {φₙ} as an orthonormal basis in L²_w([a,b]).

Green's Function Connection

The S-L BVP Ly=f (with λ not an eigenvalue) has a unique solution y=∫G(x,t)f(t)dt where G is the Green's function. Green's function is symmetric: G(x,t)=G(t,x).

Summary

S-L Problem

−(py′)′+qy=λwy with separated BCs. p>0, w>0. Self-adjoint in L²_w.

Eigenvalues

Real, discrete λ₁<λ₂<…→+∞. Each has one eigenfunction (up to scalar).

Orthogonality

∫φₘφₙw dx=0 for m≠n. Inner product weighted by w(x).

Completeness

f=Σcₙφₙ in L²_w. cₙ=⟨f,φₙ⟩_w/‖φₙ‖²_w. Parseval holds.

Oscillation

φₙ has n−1 zeros in (a,b). Between consecutive zeros of φₙ, φₙ₊₁ has exactly one zero.

Green's Function

G(x,t)=G(t,x). Unique solution y=∫G(x,t)f(t)dt when λ≠eigenvalue.
