Unit 1 — Analysis · Topic 02 of 11
Real Number System & Completeness
The complete ordered field ℝ, Archimedean property, supremum & infimum — foundations of real analysis
CSIR NET Core Analysis Foundation Intermediate 65 min Study

Learning Objectives

  1. State the field axioms, order axioms, and identify ℝ as a complete ordered field
  2. Define supremum (least upper bound) and infimum (greatest lower bound) and compute them
  3. State and use the Completeness Axiom (Least Upper Bound Property)
  4. Apply the Archimedean Property to prove density results
  5. Use the Nested Interval Theorem and understand its connection to completeness
🎯
CSIR NET Frequency: Sup/inf computations and Archimedean property appear in 3–4 questions per exam. Completeness distinguishes ℝ from ℚ — this distinction is tested both in Part B and Part C. Know the LUB property inside-out.

⚖️ Ordered Field Axioms

𝔽
Concept 1
Field Axioms of ℝ

A field is a set F with two operations (+, ·) satisfying the following axioms. ℝ, ℚ, and ℂ are all fields; ℤ is NOT (no multiplicative inverses for integers other than ±1).

A1 — Closure
a + b ∈ F and a · b ∈ F for all a, b ∈ F
A2 — Commutativity
a + b = b + a and a · b = b · a
A3 — Associativity
(a+b)+c = a+(b+c) and (a·b)·c = a·(b·c)
A4 — Identity
∃ 0: a+0 = a and ∃ 1≠0: a·1 = a
A5 — Inverses
∃ −a: a+(−a) = 0 and ∃ a⁻¹ (a≠0): a·a⁻¹ = 1
A6 — Distributivity
a·(b+c) = a·b + a·c
📊
Concept 2
Order Axioms — ℝ as an Ordered Field

An ordered field has a total order < compatible with the field operations. ℝ and ℚ are ordered fields; ℂ is NOT (no total order compatible with multiplication).

Order Axioms
O1 (Trichotomy): Exactly one holds: a < b, a = b, or a > b
O2 (Transitivity): a < b and b < c ⟹ a < c
O3 (Addition): a < b ⟹ a + c < b + c for any c
O4 (Multiplication): a < b and 0 < c ⟹ ac < bc

Consequences: a² ≥ 0 for all a ∈ ℝ; 0 < 1; n > 0 for all n ∈ ℕ.

⚠️
Key fact: ℂ cannot be made into an ordered field. Reason: i² = −1 < 0, but in an ordered field all squares are ≥ 0 — contradiction. This is a common CSIR NET question!

✅ Completeness & Supremum/Infimum

🎯
Concept 3
Upper Bounds, Lower Bounds, Sup and Inf
Bounded Sets
A set S ⊂ ℝ is bounded above if ∃ M ∈ ℝ such that x ≤ M for all x ∈ S.
A set S ⊂ ℝ is bounded below if ∃ m ∈ ℝ such that x ≥ m for all x ∈ S.
Bounded: Both bounded above and below.
Supremum — Least Upper Bound (LUB)
sup(S) = α ⟺
  (i) α is an upper bound: x ≤ α for all x ∈ S
  (ii) α is the LEAST: if β is any upper bound, then α ≤ β

Equivalent condition: α = sup(S) iff for every ε > 0, there exists x ∈ S with x > α − ε.
Infimum — Greatest Lower Bound (GLB)
inf(S) = β ⟺
  (i) β is a lower bound: x ≥ β for all x ∈ S
  (ii) β is the GREATEST: if γ is any lower bound, then β ≥ γ

Equivalent condition: β = inf(S) iff for every ε > 0, there exists x ∈ S with x < β + ε.

Important Properties of Sup and Inf

PropertyStatement
sup(−S) = −inf(S)−S = {−x : x ∈ S}
sup(S+T) = sup S + sup TS+T = {s+t : s∈S, t∈T}
If S ⊆ Tsup S ≤ sup T and inf S ≥ inf T
max(S)sup S, if sup S ∈ S (maximum exists)
Unbounded abovesup S = +∞ (by convention)
⭐
Concept 4
The Completeness Axiom — What Distinguishes ℝ from ℚ
Completeness Axiom (Least Upper Bound Property)
Every non-empty subset S of ℝ that is bounded above has a supremum in ℝ.

sup(S) ∈ ℝ whenever S ≠ ∅ and S is bounded above.
This axiom FAILS in ℚ: Consider S = {x ∈ ℚ : x² < 2}. S is bounded above (e.g., by 2) and non-empty. But sup(S) = √2 ∉ ℚ. So ℚ does not satisfy the LUB property. ℝ is defined to be the unique complete ordered field (up to isomorphism).
🔑
Key insight: The completeness axiom is what "fills in the gaps" of the rational line. Every "hole" in ℚ (like √2, π, e) is filled by a real number defined as the sup or inf of some rational set. This is the essence of ℝ.

🏛 Archimedean Property

∀
Concept 5
Archimedean Property of ℝ
Archimedean Property
For any x ∈ ℝ, there exists n ∈ ℕ such that n > x.

Equivalently: For any ε > 0, there exists n ∈ ℕ such that 1/n < ε.
Proof: Suppose not — then ℕ is bounded above. By LUB property, sup(ℕ) = α exists. Then α − 1 is not an upper bound, so ∃ n ∈ ℕ with n > α − 1, i.e., n + 1 > α. But n+1 ∈ ℕ and n+1 > α = sup(ℕ) — contradiction. □

Key Consequences

Three Equivalent Formulations
(i) ∀ x ∈ ℝ, ∃ n ∈ ℕ such that n > x (ℕ is unbounded above in ℝ)
(ii) ∀ ε > 0, ∃ n ∈ ℕ such that 1/n < ε
(iii) lim_{n→∞} (1/n) = 0

Floor function: For x ∈ ℝ, ∃! n ∈ ℤ with n ≤ x < n+1 (denoted ⌊x⌋)

🔍 Density & Nested Interval Theorem

🔬
Concept 6
Density of ℚ and ℝ∖ℚ in ℝ
Density Theorem
Between any two distinct real numbers there exists both a rational number and an irrational number.

Formally: if a < b (a, b ∈ ℝ), then ∃ r ∈ ℚ with a < r < b, and ∃ ρ ∈ ℝ∖ℚ with a < ρ < b.
Proof of ℚ dense in ℝ: By Archimedean property, ∃ n ∈ ℕ with n(b−a) > 1. Let m = ⌈na⌉ (ceiling). Then m ≤ na+1 ≤ nb and m > na, so a < m/n < b, and m/n ∈ ℚ. □
For irrationals: take r rational in (a,b) and r' rational in (a, r). Then r + (√2)/n lies between a and b for appropriate n.

Nested Interval Theorem

Nested Interval Theorem (Cantor)
Let [aₙ, bₙ] be a sequence of closed bounded intervals with [a₁,b₁] ⊇ [a₂,b₂] ⊇ … (nested). Then ⋂_{n=1}^∞ [aₙ, bₙ] ≠ ∅.

If additionally bₙ − aₙ → 0, then ⋂ [aₙ, bₙ] = {p} for exactly one point p.
Key: The set A = {aₙ} is bounded above (by b₁). By LUB property, p = sup(A) exists. One verifies p ∈ [aₙ, bₙ] for all n. This theorem FAILS for open intervals (⋂ (0,1/n) = ∅) and in ℚ (holes). It uses the Completeness Axiom essentially.
🧠
Why the theorem fails for open intervals: ⋂_{n=1}^∞ (0, 1/n) = ∅ because no x > 0 is in all intervals (by Archimedean: 1/n < x for large n), and 0 ∉ (0, 1/n) (open interval). Closedness is essential!
↔️
Concept 7
Equivalent Forms of Completeness

All the following are equivalent to the LUB property (each can replace the others as an axiom for ℝ):

#Equivalent Statement
1LUB Property: Every non-empty bounded-above set has a supremum
2GLB Property: Every non-empty bounded-below set has an infimum
3Monotone Convergence: Every bounded monotone sequence converges
4Bolzano-Weierstrass: Every bounded sequence has a convergent subsequence
5Nested Interval: Nested closed bounded intervals have non-empty intersection
6Cauchy Completeness: Every Cauchy sequence in ℝ converges
7Heine-Borel: A subset of ℝ is compact iff it is closed and bounded

📜 Key Theorems

Theorem 1 — Supremum ε-Characterization
α = sup(S) iff α is an upper bound of S AND for every ε > 0, there exists s ∈ S with s > α − ε.
This means α is "as small as possible" — no element below α is still an upper bound. Use this to prove equalities like sup{x² : 0 ≤ x < 1} = 1.
Theorem 2 — sup and inf via Sequences
α = sup(S) iff there exists a sequence (sₙ) in S with sₙ → α.
This connects the LUB property to sequences — fundamental for the rest of analysis.
Theorem 3 — inf(-S) = -sup(S)
For any bounded set S, inf(−S) = −sup(S), where −S = {−x : x ∈ S}.
This allows the GLB property to be derived from the LUB property, showing they're equivalent.
Theorem 4 — Well-Ordering of ℕ
Every non-empty subset of ℕ has a smallest element (least element).
This is a consequence of the ordering on ℝ restricted to ℕ, and is equivalent to the Principle of Mathematical Induction. Used to prove Archimedean property directly.
Theorem 5 — Existence of nth Roots
For any x > 0 and n ∈ ℕ, there exists a unique y > 0 with yⁿ = x (written y = x^{1/n}).
Proved using the LUB property: let S = {t ∈ ℝ : t ≥ 0, tⁿ ≤ x}. Then sup(S) = x^{1/n}. This shows ℝ contains all positive real nth roots — unlike ℚ (e.g., √2 ∉ ℚ).

✏️ Worked Examples

Example 1 — Computing Supremum
Find sup S and inf S for S = {n/(n+1) : n ∈ ℕ}.
Elements: S = {1/2, 2/3, 3/4, 4/5, …}
Pattern: n/(n+1) = 1 − 1/(n+1) is strictly increasing and approaches 1.
Upper bound: n/(n+1) < 1 for all n, so 1 is an upper bound.
Least upper bound: For any α < 1, choose n large enough so 1/(n+1) < 1 − α, then n/(n+1) > α. So no number less than 1 is an upper bound. Thus sup S = 1 ∉ S (no maximum).
Lower bound: Smallest element is 1/2 (at n=1). So inf S = 1/2 ∈ S (minimum exists).
Example 2 — LUB Property Fails in ℚ
Show that S = {x ∈ ℚ : x² < 2} has no supremum in ℚ.
S is bounded above in ℚ: e.g., 3/2 is an upper bound (since (3/2)² = 9/4 > 2).
sup S ∉ ℚ: In ℝ, sup S = √2 (since x² < 2 iff x < √2 for x ≥ 0). But √2 ∉ ℚ.
Conclusion: Every rational upper bound of S is not the least one (one can always find a smaller rational upper bound closer to √2). So S has no supremum in ℚ, proving ℚ is NOT complete. In ℝ, sup S = √2 exists, showing the power of completeness.
Example 3 — Archimedean Property Application
Prove that for any ε > 0, there exists n ∈ ℕ with 0 < 1/n < ε.
By the Archimedean property applied to x = 1/ε > 0: there exists n ∈ ℕ with n > 1/ε. Taking reciprocals (and reversing inequality since n, ε > 0): 1/n < ε. Also 1/n > 0 trivially. □
Consequence: This proves lim_{n→∞} 1/n = 0 rigorously.
Example 4 — ε-Characterization of Sup
Show that sup{x ∈ ℝ : x² ≤ 3} = √3.
Let S = {x ∈ ℝ : x² ≤ 3}. Note x ∈ S iff |x| ≤ √3, so S = [−√3, √3].
√3 is an upper bound: For x ∈ S, x ≤ √3. ✓
Least: For any ε > 0, take x = √3 − ε/2 (for small ε). Then x < √3 and x ∈ S (x² < 3). So √3 − ε is not an upper bound (our x > √3 − ε is in S).
By the ε-characterization, sup S = √3. ✓
Example 5 — Nested Interval Theorem
Apply the Nested Interval Theorem to Iₙ = [1/n, 1 − 1/n] and find ⋂ Iₙ.
Note: I₁ = [1, 0] is empty (wait — 1/1 = 1 > 1 − 1/1 = 0), so the intervals are only non-empty for n ≥ 2.
For n ≥ 2: Iₙ = [1/n, 1 − 1/n] ⊇ I_{n+1} = [1/(n+1), 1−1/(n+1)] ✓ (since 1/n > 1/(n+1) and 1−1/n < 1−1/(n+1)).
Length of Iₙ = (1 − 1/n) − 1/n = 1 − 2/n → 1 as n→∞... Hmm, length → 1, not 0.
⋂_{n=2}^∞ [1/n, 1−1/n] = [sup{1/n}, inf{1−1/n}] = [0, 1] (as n→∞, 1/n→0 and 1−1/n→1).
So the intersection is the closed interval [0,1].

📌 Summary & CSIR NET Tips

Core Facts

  • ℝ is the unique complete ordered field
  • ℚ is ordered field but NOT complete
  • ℂ is a field but NOT ordered
  • ℤ is an ordered ring, not a field
  • Every non-empty bounded set in ℝ has sup and inf

Key Results

  • Archimedean: ∀x ∈ ℝ, ∃n ∈ ℕ with n > x
  • 1/n → 0 (Archimedean consequence)
  • ℚ dense in ℝ; ℝ∖ℚ dense in ℝ
  • Nested closed bounded intervals: ⋂ Iₙ ≠ ∅
  • sup(A∪B) = max(sup A, sup B)

CSIR Exam Q-Types

  • Compute sup/inf of standard sets
  • True/False about ordered fields
  • Identify where LUB fails in ℚ
  • Archimedean applications (1/n < ε)
  • Nested interval theorem applications

Common Mistakes

  • sup S need not be in S (no max necessary)
  • Same sup/inf does NOT mean same sets
  • sup(A ∩ B) ≠ min(sup A, sup B) in general
  • Open intervals can have empty ⋂ (not NIT)
  • ℤ ≠ field (no reciprocals for most integers)
