Introduction

Ordinary differential equations (ODEs) involve derivatives of a function of one variable. First-order ODEs involve only y and y'. The main solution techniques — separation of variables, integrating factors, exactness — and the fundamental existence-uniqueness theorem (Picard-Lindelöf) are core CSIR NET topics.

Types & Classification

ODE Order & Degree

The order of an ODE is the order of the highest derivative. The degree is the power of the highest-order derivative (after clearing fractions/radicals).

Separable & Linear Equations

Separable ODEs

Method

dy/dx = f(x)g(y): separate as dy/g(y) = f(x)dx, then integrate both sides.

∫ dy/g(y) = ∫ f(x) dx + C
Example

dy/dx = xy: Separate → dy/y = x dx → ln|y| = x²/2 + C → y = Ae^(x²/2).

First-Order Linear ODEs

Integrating Factor Method

y' + P(x)y = Q(x). Multiply by μ(x) = e^(∫P(x)dx):

d/dx[μy] = μQ(x) ⟹ μy = ∫μQ dx + C ⟹ y = (1/μ)∫μQ dx + C/μ
Example

y' + (2/x)y = x² (x>0). μ=e^(∫2/x dx)=x². Multiply: d/dx[x²y]=x⁴. Integrate: x²y = x⁵/5+C → y = x³/5 + C/x².

Bernoulli Equation

Reduction

y' + P(x)y = Q(x)yⁿ (n≠0,1). Let z = y^(1−n). Then z' + (1−n)P(x)z = (1−n)Q(x) — linear in z.

Exact Equations

Exact ODE

M(x,y)dx + N(x,y)dy = 0 is exact if ∂M/∂y = ∂N/∂x (on a simply connected domain).

Then ∃ F(x,y) with ∂F/∂x = M, ∂F/∂y = N, and solution is F(x,y) = C.

Finding F

F(x,y) = ∫M dx + g(y), then ∂F/∂y = N gives g'(y), so integrate to find g(y).

Integrating Factors for Non-Exact

If not exact, find μ such that μM dx + μN dy = 0 is exact.

  • If (∂M/∂y − ∂N/∂x)/N depends only on x: μ = e^(∫h(x)dx).
  • If (∂N/∂x − ∂M/∂y)/M depends only on y: μ = e^(∫k(y)dy).

Homogeneous Substitution

v = y/x

If dy/dx = f(y/x), let v=y/x (so y=vx). Then dy/dx = v + x·dv/dx. Substitute and separate in x,v.

Existence & Uniqueness Theorem

Picard-Lindelöf Theorem

Consider IVP: y' = f(x,y), y(x₀) = y₀.

If f is continuous on a rectangle R = {|x−x₀|≤a, |y−y₀|≤b} and satisfies a Lipschitz condition in y:

|f(x,y₁) − f(x,y₂)| ≤ L|y₁ − y₂|

then ∃ unique solution on |x−x₀| ≤ h, where h = min(a, b/M) and M = max|f| on R.

Peano Existence Theorem

If f is merely continuous (no Lipschitz), then at least one solution exists (but uniqueness may fail).

Example: y' = y^(1/3), y(0)=0 has infinitely many solutions: y≡0 and y=(2x/3)^(3/2).

Singular Solutions

Solutions not obtainable from the general solution by fixing the constant C are singular solutions. Example: y'=y^(2/3) has singular solution y=0 and general solution y=(x/3+C)³.

Picard Iteration

Define y₀(x)=y₀ and yₙ₊₁(x)=y₀+∫_{x₀}^x f(t,yₙ(t))dt. Under Lipschitz, yₙ→y uniformly (contraction mapping on C[x₀-h, x₀+h]).

Summary

Separable

dy/g(y)=f(x)dx. Integrate both sides. Works when RHS factors as f(x)g(y).

Linear 1st Order

y'+Py=Q. Multiply by μ=e^(∫P dx). Then d(μy)/dx=μQ. Integrate.

Exact

∂M/∂y=∂N/∂x ⟹ exact. Find F with Fx=M, Fy=N. Solution: F=C.

Bernoulli

y'+Py=Qyⁿ. z=y^(1−n) linearizes. Then solve linear ODE for z.

Picard-Lindelöf

f Lipschitz in y ⟹ unique local solution. Iteration converges via Banach FPT.

Singular Solutions

Not in general solution. Arise when Lipschitz condition fails (e.g., y^(1/3) at y=0).
