Introduction

Mathematical statistics uses probability theory to draw inference from data. The two main branches are estimation (point and interval) and hypothesis testing. Key concepts: sufficiency, completeness, unbiasedness, consistency, and the Cramér-Rao lower bound on variance.

Point Estimation

Estimator Properties
  • Unbiased: E[T(X)]=θ for all θ
  • Consistent: T(X₁,…,Xₙ) →^P θ as n→∞
  • Efficient: attains Cramér-Rao lower bound
  • Sufficient: T(X) contains all information about θ in X (Fisher-Neyman factorization)
Cramér-Rao Lower Bound
Var(T) ≥ 1/I(θ) for any unbiased estimator T I(θ) = E[(∂/∂θ log f(X;θ))²] (Fisher information) = −E[∂²/∂θ² log f(X;θ)] For n iid observations: I_n(θ) = n·I₁(θ)

An estimator achieving this bound is called efficient (UMVUE — Uniformly Minimum Variance Unbiased Estimator).

Maximum Likelihood Estimation (MLE)
L(θ;x) = Πᵢ f(xᵢ;θ) (likelihood) ℓ(θ) = Σᵢ log f(xᵢ;θ) (log-likelihood) MLE: θ̂ = argmax ℓ(θ) ⟺ ∂ℓ/∂θ = 0

MLE is asymptotically efficient (achieves CR bound), invariant under reparametrization, consistent.

MLE Examples
  • Normal: μ̂=X̄, σ̂²=(1/n)Σ(Xᵢ−X̄)² (biased; unbiased S²=n/(n−1)·σ̂²)
  • Exponential(λ): λ̂=1/X̄
  • Poisson(λ): λ̂=X̄
  • Binomial(n,p): p̂=X̄/n

Interval Estimation

Confidence Interval

A (1−α)×100% CI for θ is a random interval [L(X),U(X)] such that P(L≤θ≤U)=1−α.

Common critical values: z_{α/2}: z_{0.025}=1.96, z_{0.005}=2.576.

Hypothesis Testing

Framework

H₀: null hypothesis vs H₁: alternative. Choose a test statistic T and rejection region R.

  • Type I error (α): P(reject H₀ | H₀ true) — significance level
  • Type II error (β): P(fail to reject H₀ | H₁ true)
  • Power: 1−β = P(reject H₀ | H₁ true)
  • p-value: smallest α at which H₀ is rejected; reject H₀ if p-value<α
Neyman-Pearson Lemma

Most powerful test of size α for simple H₀ vs simple H₁ uses the likelihood ratio:

Reject H₀ if L(x;θ₁)/L(x;θ₀) > k_α (where k_α gives size α)

For composite alternatives use uniformly most powerful (UMP) tests when they exist.
