Introduction

The Laplace transform converts an ODE in t into an algebraic equation in s, making it especially powerful for IVPs with discontinuous or impulsive forcing (Heaviside steps, Dirac deltas). Existence is guaranteed for functions of exponential order.

The Laplace Transform

Definition
ℒ{f(t)} = F(s) = ∫₀^∞ e^(−st) f(t) dt, s > s₀

Existence: if |f(t)| ≤ Me^(αt) (exponential order α), then F(s) exists for Re(s) > α.

Heaviside / Unit Step Function
u_c(t) = H(t−c) = { 0 if t < c; 1 if t ≥ c } ℒ{u_c(t)} = e^(−cs)/s (c ≥ 0)
Dirac Delta
δ(t−c): ∫f(t)δ(t−c)dt = f(c) ℒ{δ(t−c)} = e^(−cs) (c ≥ 0) ℒ{δ(t)} = 1

Key Properties (Linearity)

ℒ{af+bg} = aF+bG  |  ℒ⁻¹{aF+bG} = af+bg. Use partial fractions for the inverse.

Solving ODEs via Laplace Transform

Method (IVP with constant coefficients)
  1. Apply ℒ to both sides; use differentiation rules with initial conditions.
  2. Solve algebraically for Y(s)=ℒ{y}.
  3. Compute y(t)=ℒ⁻¹{Y(s)} via partial fractions / table.
Example

y″+4y=0, y(0)=1, y′(0)=0.

s²Y−s+4Y=0 ⟹ Y(s)=s/(s²+4) ⟹ y(t)=cos(2t)
Discontinuous Forcing (Heaviside)

y″+y=u_π(t), y(0)=0, y′(0)=0:

Y = e^(−πs)/[s(s²+1)] y(t) = u_π(t)[1−cos(t−π)] = u_π(t)[1+cos(t)]
Impulsive Forcing (Delta)

y″+y=δ(t−π), y(0)=y′(0)=0:

Y = e^(−πs)/(s²+1) ⟹ y(t) = u_π(t)sin(t−π) = −u_π(t)sin(t)

Convolution & Inverse Laplace

Convolution
(f*g)(t) = ∫₀^t f(t−τ)g(τ)dτ = ∫₀^t f(τ)g(t−τ)dτ
Convolution Theorem
ℒ{f*g} = ℒ{f}·ℒ{g} = F(s)·G(s) ℒ⁻¹{F(s)G(s)} = f*g

This allows computing ℒ⁻¹{F(s)G(s)} without partial fractions when the inverse of individual factors is known.

Initial & Final Value Theorems
Initial: lim_{t→0⁺} f(t) = lim_{s→∞} sF(s) Final: lim_{t→∞} f(t) = lim_{s→0⁺} sF(s) (if limit exists)
Transfer Function

For Ly=g (linear constant-coeff ODE, zero ICs), Y(s)=H(s)G(s) where H(s) is the transfer function. The impulse response h(t)=ℒ⁻¹{H(s)} gives y=h*g.
