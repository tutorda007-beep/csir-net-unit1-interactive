Introduction

The maximum modulus principle is a profound geometric property of holomorphic functions: they cannot achieve interior maxima (in modulus). Rouché's theorem tells us how many zeros a function has inside a contour. Together these results form the cornerstone of geometric function theory.

Maximum Modulus Principle

Maximum Modulus Principle

If f is holomorphic and non-constant on a domain D, then |f| cannot attain a maximum at an interior point of D.

Equivalently: if D is bounded and f is continuous on D̄ and holomorphic on D, then max_{z∈D̄}|f(z)| = max_{z∈∂D}|f(z)|.

The maximum of |f| occurs on the boundary, not inside.
Key Idea of Proof

By the mean value property, f(z₀) = (1/2π)∫₀^(2π) f(z₀+re^(iθ)) dθ. If |f(z₀)| were a maximum, applying the modulus inequality shows f must be constant on each circle around z₀. By connectedness, f is constant everywhere — contradiction.

Minimum Modulus Principle

Minimum Modulus

If f is holomorphic and non-vanishing on domain D, then |f| cannot attain a minimum at an interior point.

Equivalently: min |f| on D̄ is achieved on ∂D, or f has a zero inside D.

Application

If f is holomorphic on the closed unit disk D̄, |f| = 1 on ∂D = unit circle, then f must be a Blaschke product (finite product of Möbius transformations).

Schwarz Lemma

Schwarz Lemma

Let f: D→D be holomorphic where D=unit disk, with f(0)=0. Then:

  • |f(z)| ≤ |z| for all z ∈ D
  • |f'(0)| ≤ 1
  • If |f(z₀)| = |z₀| for some z₀ ≠ 0, or |f'(0)| = 1, then f(z) = e^(iθ)z (rotation)
Proof Sketch

Define g(z) = f(z)/z for z≠0, g(0)=f'(0). Then g is holomorphic on D (removable singularity at 0). On |z|=r, |g(z)|=|f(z)|/r≤1/r. By max modulus, |g(z)|≤1/r on entire disk of radius r. Letting r→1 gives |g(z)|≤1, i.e., |f(z)|≤|z|.

Schwarz-Pick Lemma (Generalization)

For holomorphic f: D→D (not necessarily f(0)=0):

|f'(z)| ≤ (1−|f(z)|²)/(1−|z|²) for all z ∈ D

In terms of the hyperbolic metric, f is a contraction. Equality holds iff f is a Möbius transformation mapping D to itself (a "disk automorphism").

Rouché's Theorem

Rouché's Theorem

Let f, g be holomorphic on and inside a simple closed curve γ. If |g(z)| < |f(z)| on γ, then f and f+g have the same number of zeros inside γ (counted with multiplicity).

Why it works

Argument principle: Z_h = (1/2πi)∮ h'/h dz counts zeros. Consider h(t)=f+tg for t∈[0,1]. Since |tg|≤|g|<|f| on γ, h(t)≠0 on γ, so Z_{h(t)} is a continuous integer function of t — thus constant. Z_{h(0)}=Z_f = Z_{h(1)}=Z_{f+g}.

Classic Applications

Example 1: Zeros of z⁷+6z³+1 inside |z|=2

Let f=z⁷, g=6z³+1 (or let f=6z³, compare). On |z|=2: |z⁷|=128, |6z³+1|≤6·8+1=49. Since 49<128, f+g=z⁷+6z³+1 has same zeros as z⁷ inside |z|=2, which is 7.

Example 2: Fundamental Theorem of Algebra (via Rouché)

p(z) = zⁿ + aₙ₋₁zⁿ⁻¹ + ... + a₀. Take f=zⁿ, g=lower terms. For large R: |f|=Rⁿ > |g| on |z|=R. So p has same zeros as zⁿ inside |z|=R, which is n zeros. Thus every degree-n polynomial has exactly n roots.

Hurwitz's Theorem

Hurwitz's Theorem

If {fₙ} is a sequence of holomorphic functions converging uniformly to f on compact subsets of D, and each fₙ has no zeros in D, then either f≡0 or f has no zeros in D.

More generally: if each fₙ has at most k zeros in D, so does the limit f.

Open Mapping Theorem & Related Results

Open Mapping Theorem

If f is holomorphic and non-constant on a domain D, then f is an open map: it maps open sets to open sets.

Corollary

A non-constant holomorphic function has the property that f(D) is an open set whenever D is a domain. In particular, holomorphic functions "spread out" — they cannot map all of D into a curve or a single point.
