Introduction

A vector space is a set with two operations — vector addition and scalar multiplication — satisfying eight axioms. This abstraction unifies geometric vectors in ℝⁿ, polynomials, matrices, and function spaces under one framework.

Key Structures

  • ℝⁿ, ℂⁿ — coordinate n-space
  • Mₘₓₙ(𝔽) — matrices over field 𝔽
  • Pₙ(𝔽) — polynomials of degree ≤ n
  • C[a,b] — continuous functions

Vector Space Axioms

Definition — Vector Space

A vector space V over a field 𝔽 is a set with operations + : V×V → V and · : 𝔽×V → V satisfying:

  1. Commutativity: u + v = v + u
  2. Associativity: (u+v)+w = u+(v+w)
  3. Zero vector: ∃ 0 ∈ V: v + 0 = v for all v
  4. Additive inverse: ∀ v ∈ V, ∃ −v: v + (−v) = 0
  5. Scalar identity: 1·v = v
  6. Scalar associativity: (αβ)v = α(βv)
  7. Distributivity over vectors: α(u+v) = αu + αv
  8. Distributivity over scalars: (α+β)v = αv + βv
Consequences of Axioms
  • 0·v = 0 (zero scalar → zero vector)
  • α·0 = 0 (any scalar times zero vector = zero vector)
  • (−1)·v = −v (additive inverse from scalar −1)
  • If αv = 0, then α = 0 or v = 0

Subspaces

Definition — Subspace

A non-empty subset W ⊆ V is a subspace of V if W is closed under addition and scalar multiplication. The subspace test: W is a subspace iff:

  1. 0 ∈ W (zero vector)
  2. u, v ∈ W ⟹ u + v ∈ W (closed under addition)
  3. α ∈ 𝔽, v ∈ W ⟹ αv ∈ W (closed under scalar multiplication)

Equivalently: W is a subspace iff ∀ u,v ∈ W and α,β ∈ 𝔽: αu + βv ∈ W.

Sum and Intersection of Subspaces

Theorem
  • W₁ ∩ W₂ is always a subspace of V
  • W₁ ∪ W₂ is a subspace iff W₁ ⊆ W₂ or W₂ ⊆ W₁
  • W₁ + W₂ = {w₁ + w₂ : w₁ ∈ W₁, w₂ ∈ W₂} is always a subspace
Dimension Formula (Grassmann)
dim(W₁ + W₂) = dim W₁ + dim W₂ − dim(W₁ ∩ W₂)

This is the subspace analogue of the inclusion-exclusion principle.

Span and Linear Independence

Definition — Linear Combination & Span

A linear combination of v₁,...,vₖ ∈ V is α₁v₁ + α₂v₂ + ... + αₖvₖ (αᵢ ∈ 𝔽).

Span: span{v₁,...,vₖ} = set of all linear combinations of v₁,...,vₖ. This is the smallest subspace containing all vᵢ.

Definition — Linear Independence

Vectors v₁,...,vₖ are linearly independent if α₁v₁ + ... + αₖvₖ = 0 ⟹ α₁ = α₂ = ... = αₖ = 0.

They are linearly dependent if some non-trivial combination equals zero (some αᵢ ≠ 0).

Key Facts About Linear Independence

  • A single non-zero vector is always linearly independent
  • The zero vector makes any set linearly dependent
  • If vᵢ ∈ span{v₁,...,vᵢ₋₁}, the set is linearly dependent
  • Any subset of a linearly independent set is linearly independent
  • In ℝⁿ, any set of more than n vectors is linearly dependent
Testing Linear Independence via Row Reduction

To test if v₁,...,vₖ ∈ ℝⁿ are linearly independent, form the matrix [v₁ | v₂ | ... | vₖ] and row reduce. The vectors are linearly independent iff the reduced matrix has no free variables (rank = k).

Basis and Dimension

Definition — Basis

A basis for V is a set B = {v₁,...,vₙ} that is both:

  • Linearly independent, AND
  • Spanning: span(B) = V

Every vector in V has a unique representation as a linear combination of basis vectors.

Definition — Dimension

The dimension dim(V) is the number of vectors in any basis of V. If V has no finite basis, dim(V) = ∞.

Key Theorems

Basis Extension Theorem

Any linearly independent set in a finite-dimensional V can be extended to a basis of V. Any spanning set contains a basis as a subset.

Dimension Theorem

All bases of a finite-dimensional vector space have the same cardinality. This common number is the dimension.

Consequences:

  • A set of exactly dim(V) vectors that spans V is a basis
  • A set of exactly dim(V) linearly independent vectors is a basis
  • Any set of more than dim(V) vectors is linearly dependent
  • No spanning set has fewer than dim(V) vectors
Finding a Basis by Row Reduction

Given vectors v₁,...,vₖ, form the matrix with these as rows. Row reduce to echelon form. The vectors corresponding to pivot rows form a basis for span{v₁,...,vₖ}. The rank = number of pivots = dimension of span.

Coordinates and Isomorphisms

Coordinate Vector

Given basis B = {b₁,...,bₙ} for V, every v ∈ V has a unique representation v = α₁b₁ + ... + αₙbₙ. The coordinate vector of v with respect to B is [v]_B = (α₁,...,αₙ) ∈ 𝔽ⁿ.

Theorem — Isomorphism

Every n-dimensional vector space over 𝔽 is isomorphic to 𝔽ⁿ. The isomorphism is given by the coordinate map v ↦ [v]_B.

Two vector spaces over the same field are isomorphic iff they have the same dimension.

Change of Basis

Change of Basis Matrix

Let B = {b₁,...,bₙ} and C = {c₁,...,cₙ} be two bases of V. The change of basis matrix from B to C is P = [[b₁]_C | [b₂]_C | ... | [bₙ]_C].

Then [v]_C = P·[v]_B. The matrix P is invertible, and P⁻¹ is the change of basis matrix from C to B.

If A = [T]_B (matrix of T in basis B), then [T]_C = P⁻¹·A·P (similar matrices)

Direct Sums

Definition — Direct Sum

V = W₁ ⊕ W₂ (direct sum) if:

  • V = W₁ + W₂ (every v = w₁ + w₂)
  • W₁ ∩ W₂ = {0} (decomposition is unique)

Equivalently: each v ∈ V has a unique representation v = w₁ + w₂ with w₁ ∈ W₁, w₂ ∈ W₂.

Dimension of Direct Sum

If V = W₁ ⊕ W₂, then dim V = dim W₁ + dim W₂. More generally:

V = W₁ ⊕ W₂ ⊕ ... ⊕ Wₖ ⟹ dim V = Σ dim Wᵢ

Complement of a Subspace

For any subspace W ⊆ V (finite-dimensional), there exists a complement W' such that V = W ⊕ W'. (Note: the complement is NOT unique unless dim W = 0 or dim W = dim V.)

Examples
  • ℝ² = {(x,0)} ⊕ {(0,y)}: x-axis and y-axis
  • M₂ₓ₂ = Sym₂ ⊕ SkewSym₂: symmetric + skew-symmetric matrices (A = (A+Aᵀ)/2 + (A−Aᵀ)/2)
  • Pₙ = (even polys) ⊕ (odd polys)
