Basic Definitions

Complex Number z = x + iy Basic

x = Re(z) (real part), y = Im(z) (imaginary part), i² = −1.

|z| = √(x²+y²) (modulus/magnitude) arg(z) = θ = arctan(y/x) (principal: θ∈(−π,π])
  • z = 0 iff x = 0 and y = 0
  • z real iff Im(z) = 0; z pure imaginary iff Re(z)=0
  • Conjugate: z̄ = x − iy.   zz̄ = |z|²
  • Re(z) = (z+z̄)/2,   Im(z) = (z−z̄)/(2i)

Polar Form & Euler's Formula Basic

z = r·e^(iθ) = r(cosθ + i sinθ) where r = |z|, θ = arg(z)

Euler's formula: e^(iθ) = cosθ + i sinθ

  • e^(iπ) + 1 = 0  (Euler's identity)
  • |z₁z₂| = |z₁|·|z₂|
  • arg(z₁z₂) = arg(z₁) + arg(z₂) (mod 2π)
  • z₁/z₂: divide moduli, subtract arguments

Algebraic Operations Basic

Addition: (a+ib)+(c+id) = (a+c) + i(b+d)

Multiplication: (a+ib)(c+id) = (ac−bd) + i(ad+bc)

Division: (a+ib)/(c+id) = [(a+ib)(c−id)] / (c²+d²)

Key identities:

  • |z̄| = |z|,   arg(z̄) = −arg(z)
  • |Re(z)| ≤ |z|,   |Im(z)| ≤ |z|
  • Triangle ineq: |z₁+z₂| ≤ |z₁|+|z₂|
  • Reverse: | |z₁|−|z₂| | ≤ |z₁−z₂|

de Moivre & nth Roots Intermediate

de Moivre: (cosθ + i sinθ)ⁿ = cos(nθ) + i sin(nθ)

nth roots of z = re^(iθ):

zₖ = r^(1/n) · e^(i(θ+2πk)/n), k=0,1,…,n−1

nth roots of unity: ωₖ = e^(2πik/n), k = 0,…,n−1

  • All lie on unit circle |z|=1
  • Equally spaced: angle 2π/n between consecutive
  • Sum = 0, Product = (−1)ⁿ⁺¹

Complex Derivative · CR Equations · Entire Functions · Harmonic Functions · Singularities

Complex Derivative & Analyticity

Complex Derivative Basic

f'(z) = lim[Δz→0] [f(z+Δz) − f(z)] / Δz

The limit must be the same regardless of direction Δz→0 in ℂ. This is far stronger than real differentiability!

  • f analytic at z₀: f' exists in a neighbourhood of z₀
  • f entire: analytic on all of ℂ (e.g., eˢ, sin z, polynomials)
  • Analytic ≠ just differentiable at one point
  • Rules: sum, product, quotient, chain — same as real

Cauchy-Riemann Equations Basic

Write f(z) = u(x,y) + iv(x,y). Then f is differentiable at z = x+iy iff:

∂u/∂x = ∂v/∂y (CR₁) ∂u/∂y = −∂v/∂x (CR₂)

Necessary: f differentiable at z ⟹ CR satisfied there.

Sufficient: CR satisfied + partial derivatives continuous ⟹ f analytic.

Harmonic Functions Intermediate

A real function φ(x,y) is harmonic if ∇²φ = 0:

∂²φ/∂x² + ∂²φ/∂y² = 0

Key theorem: If f = u+iv is analytic, then BOTH u and v are harmonic.

Conversely, given harmonic u, there exists harmonic v (the harmonic conjugate) s.t. f = u+iv is analytic. Find v using CR equations.

Example: u = x²−y² → v = 2xy (since f(z)=z²)

Types of Singularities Advanced

At an isolated singularity z₀ of f:

  • Removable: lim f(z) exists finite (e.g., sin(z)/z at 0)
  • Pole of order m: lim (z−z₀)ᵐ f(z) = finite ≠ 0. Simple pole: m=1
  • Essential: neither removable nor pole (e.g., e^(1/z) at 0)
Picard's Great Thm: near essential singularity, f takes EVERY complex value (except possibly one) infinitely often.
Key Theorems

CR in Polar Form Advanced

If f = u+iv, z = re^(iθ), the CR equations become:

∂u/∂r = (1/r)·∂v/∂θ and ∂v/∂r = −(1/r)·∂u/∂θ

Useful for functions like f(z) = Log z = ln|z| + i·arg(z). Here u = ln r, v = θ. Check: ∂u/∂r = 1/r = (1/r)·∂v/∂θ = 1/r ✓.

Analytic Function is Infinitely Differentiable Advanced

If f is analytic in domain D, then f has derivatives of ALL orders in D (unlike real analysis where once-differentiable doesn't imply twice!). This follows from Cauchy's Integral Formula.

Also: analytic function is represented by its power series (convergent Taylor series) in any disk within D.

AEverywhere in ℂ
BOnly at z = 0
CNowhere

✓ Correct Answer: (B) — Only at z = 0

Harmonic ConjugateMedium
If u(x,y) = x³ − 3xy² is harmonic, its harmonic conjugate v(x,y) is:
A3x²y − y³
B3x²y + y³
Cx³ + 3xy²
D3xy − y³
✓ Correct Answer: (A) — 3x²y − y³
∂u/∂x = 3x²−3y² = ∂v/∂y, so v = 3x²y − y³ + g(x). Then ∂v/∂x = 6xy + g'(x) must equal −∂u/∂y = 6xy → g'(x)=0 → g = constant. So v = 3x²y − y³. Check: f(z) = z³ = (x+iy)³ = (x³−3xy²) + i(3x²y−y³) ✓.

✓ Correct Answer: (D) — 1/(z²+1)
1/(z²+1) has singularities at z=±i (where z²+1=0), so it is NOT entire. eᶻ, sin z, and polynomials are entire (analytic on ALL of ℂ). The function 1/(z²+1) is meromorphic (analytic except at poles ±i).

BPole of order 1

✓ Correct Answer: (C) — Removable singularity

Harmonic FunctionAdvanced
If f(z) = u+iv is analytic and non-constant in a domain D, then:
Au is harmonic but v need not be
BBoth u and v satisfy Laplace's equation
Cu²+v² is always constant
D|f| has a maximum in D
✓ Correct Answer: (B)
Both u and v satisfy ∇²φ = 0 (Laplace equation) — they are harmonic conjugates. Proof: from CR, ∂²u/∂x²=∂²v/∂x∂y=−∂²u/∂y², so ∇²u=0. Similarly ∇²v=0. Option D is FALSE — by the Maximum Modulus Principle, |f| attains its maximum on the BOUNDARY, not inside D.

Power Series & Transcendental Functions

Radius of Convergence · e^z · sin z · cos z · sinh/cosh · Periodicity · Zeros

📊 Interactive: Disk of Convergence

The disk of convergence is the largest open disk |z−z₀|<R in which the power series converges absolutely and uniformly.

Power Series

Radius of Convergence Basic

Σ aₙ(z−z₀)ⁿ converges for |z−z₀| < R R = 1 / limsup |aₙ|^(1/n) [Cauchy-Hadamard] Also: R = lim |aₙ/aₙ₊₁| [if limit exists]
  • Converges absolutely inside the disk |z−z₀|<R
  • Diverges outside |z−z₀|>R
  • On the boundary |z−z₀|=R: test each point separately
  • Inside disk: convergence is uniform on compact subsets

Exponential & Trig Functions Basic

eᶻ = Σ zⁿ/n! = 1+z+z²/2!+… (R=∞) sin z = Σ (−1)ⁿz^(2n+1)/(2n+1)! (R=∞) cos z = Σ (−1)ⁿz^(2n)/(2n)! (R=∞)

Key relations:

  • e^(iz) = cos z + i sin z (Euler)
  • sin z = (e^(iz)−e^(−iz))/(2i)
  • cos z = (e^(iz)+e^(−iz))/2
  • sin²z + cos²z = 1 (still holds in ℂ!)

Hyperbolic Functions Intermediate

sinh z = (eᶻ − e^(−z))/2 cosh z = (eᶻ + e^(−z))/2 tanh z = sinh z / cosh z

Link to trig: sin(iz) = i·sinh(z) and cos(iz) = cosh(z)

  • cosh²z − sinh²z = 1
  • sinh(z+w) = sinh z·cosh w + cosh z·sinh w
  • Zeros of sinh z: z = nπi (n ∈ ℤ)
  • Zeros of cosh z: z = (n+½)πi

Properties of eᶻ Advanced

For z = x + iy: |eᶻ| = eˣ, arg(eᶻ) = y eᶻ = eˣ(cos y + i sin y)
  • eᶻ is periodic with period 2πi: e^(z+2πi) = eᶻ
  • eᶻ ≠ 0 for all z ∈ ℂ
  • eᶻ is entire, eᶻ = 1 iff z = 2nπi
  • Zeros of sin z: z = nπ (n ∈ ℤ)
  • Zeros of cos z: z = (n+½)π
  • sin z, cos z unbounded on ℂ (unlike real case!)

Periodicity of eᶻMedium
The fundamental period of e^z as a function ℂ→ℂ is:
A2π
B2πi
Cπi

✓ Correct Answer: (B) — 2πi
e^(z+2πi) = eᶻ·e^(2πi) = eᶻ·1 = eᶻ. So period = 2πi. Note: on ℝ, eˣ has no real period (strictly increasing). The complex periodicity comes from the imaginary part: e^(iy) = cos y + i sin y which has period 2π in y. This periodicity makes eᶻ map horizontal strips of height 2π onto ℂ\{0}.

Zeros of sin zAdvanced
The zeros of sin z in ℂ are:
Az = nπ, n ∈ ℤ
Bz = nπi, n ∈ ℤ
Cz = (n+½)π, n ∈ ℤ
Dz = nπ/2, n ∈ ℤ
✓ Correct Answer: (A) — z = nπ, n ∈ ℤ
sin z = (e^(iz)−e^(−iz))/(2i) = 0 ⟺ e^(iz) = e^(−iz) ⟺ e^(2iz) = 1 ⟺ 2iz = 2nπi ⟺ z = nπ. The zeros of sin z are exactly the same as on ℝ! But note: sin(i) = i·sinh(1) ≠ 0, confirming the zeros are only real. Zeros of cos z: z=(2n+1)π/2.

Complex Trig IdentityAdvanced
The value of sin(π/2 + i) is:
Ai·sinh(1)
Bcosh(1)
Ccos(1)
Di·cosh(1)
✓ Correct Answer: (B) — cosh(1)
sin(π/2+i) = sin(π/2)cos(i) + cos(π/2)sin(i) = 1·cos(i) + 0·sin(i) = cos(i). Now cos(i) = (e^(i·i)+e^(−i·i))/2 = (e^(−1)+e)/2 = cosh(1). General formula: cos(iy) = cosh(y). Since cosh(1) ≈ 1.543, this is a real number greater than 1 — impossible for real cosine!

📌 CSIR Key Tips — Complex Analysis I

  • |eᶻ| = eˣ (only real part matters). Period = 2πi.
  • CR equations: necessary for diff; CR + continuous partials = analytic. Check |z|² → not analytic except z=0.
  • Harmonic conjugate: integrate CR equations. f(z)=z³ gives u=x³−3xy², v=3x²y−y³.
  • Singularity types: removable (limit exists), pole (lim (z−z₀)ᵐf finite), essential (neither). sin(z)/z has removable sing at 0.
  • Roots of unity: 1+ω+ω²+…+ωⁿ⁻¹ = 0. Sum=0, Product=(−1)ⁿ⁺¹.
