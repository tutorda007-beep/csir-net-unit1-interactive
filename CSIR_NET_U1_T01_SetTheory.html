Unit 1 — Analysis · Topic 01 of 11
Elementary Set Theory
Foundation of all mathematics — sets, operations, countability, and Cantor's cardinality theory
CSIR NET Core Analysis Foundation Beginner–Intermediate 60 min Study

Learning Objectives

  1. Define a set and use set-builder notation fluently
  2. Apply all set operations (union, intersection, difference, complement) and De Morgan's laws
  3. Distinguish between finite, countably infinite, and uncountably infinite sets with examples
  4. Understand cardinality and prove two sets have the same cardinality using bijections
  5. State and apply Cantor's theorem, the Schroeder-Bernstein theorem, and Cantor's diagonal argument
🎯
CSIR NET Frequency: Set theory questions appear in 2–3 questions each exam, usually testing countability of standard sets (ℚ, ℝ∖ℚ, ℝ) or cardinality comparisons. Master the bijection technique — it's the key to 4.75-mark Part C questions.

📦 Sets & Operations

📘
Concept 1
What Is a Set?

A set is a well-defined collection of distinct objects called elements. The phrase "well-defined" is crucial — for any object, we can determine unambiguously whether it belongs to the set or not.

Notation
• Roster form: A = {1, 2, 3, 4, 5}
• Set-builder form: A = {x ∈ ℤ : 1 ≤ x ≤ 5}
• Membership: 3 ∈ A means "3 is an element of A"
• Non-membership: 6 ∉ A means "6 is not an element of A"

Concept 2
Set Operations
Formal Definitions
Union: A ∪ B = {x : x ∈ A or x ∈ B}
Intersection: A ∩ B = {x : x ∈ A and x ∈ B}
Difference: A ∖ B = {x : x ∈ A and x ∉ B}
Complement: Aᶜ = {x ∈ U : x ∉ A} (relative to universal set U)
Symmetric Difference: A △ B = (A ∖ B) ∪ (B ∖ A)

De Morgan's Laws — Extremely Important!

(A ∪ B)ᶜ = Aᶜ ∩ Bᶜ
(A ∩ B)ᶜ = Aᶜ ∪ Bᶜ

These generalize to arbitrary (even infinite) collections: (⋃ Aᵢ)ᶜ = ⋂ Aᵢᶜ and (⋂ Aᵢ)ᶜ = ⋃ Aᵢᶜ.

🗂 Types of Sets

📁
Concept 3
Special Sets
Empty Set ∅
The set containing no elements. Unique. ∅ ⊆ A for every set A. |∅| = 0.
Note: {∅} is NOT empty — it contains one element (the empty set itself).
Subset & Proper Subset
A ⊆ B ⟺ (∀x)(x ∈ A ⟹ x ∈ B)
A ⊊ B ⟺ A ⊆ B and A ≠ B (proper subset)
Power Set
𝒫(A) = {S : S ⊆ A}. If |A| = n, then |𝒫(A)| = 2ⁿ.
Example: 𝒫({a,b}) = {∅, {a}, {b}, {a,b}} → has 4 = 2² elements.
Cartesian Product
A × B = {(a,b) : a ∈ A, b ∈ B}.
If |A| = m and |B| = n, then |A × B| = mn.
ℝ² = ℝ × ℝ is the usual coordinate plane.
⚡
CSIR Trap: The number of subsets of A is 2^|A|, but the number of PROPER subsets is 2^|A| − 1. Don't confuse these!
🔗
Concept 4
Relations and Functions (Key Link)

A relation R from A to B is a subset of A × B. A function f : A → B is a relation where every element of A is paired with exactly one element of B.

Types of Functions
Injective (one-to-one): f(a₁) = f(a₂) ⟹ a₁ = a₂
Surjective (onto): ∀b ∈ B, ∃a ∈ A such that f(a) = b
Bijective: Both injective and surjective (one-to-one correspondence)

Bijections are the tool to compare infinite set sizes — if a bijection f : A → B exists, then A and B have the same cardinality.

♾ Countability

🔢
Concept 5
Finite vs Infinite Sets

A set A is finite if it is empty or can be put in bijection with {1, 2, …, n} for some n ∈ ℕ. Otherwise A is infinite.

Countably Infinite
A set A is countably infinite if there exists a bijection f : ℕ → A. This means elements of A can be listed in a sequence: a₁, a₂, a₃, …
Such a set has cardinality ℵ₀ (aleph-null).
Countable
A set is countable if it is either finite or countably infinite.
Uncountably Infinite
A set is uncountable if it is infinite but NOT countably infinite. No bijection with ℕ exists. The set ℝ is uncountably infinite with cardinality 𝔠 = 2^ℵ₀.

🔬 Cantor's Theory & Cardinality

∞
Concept 6
Cardinality and Comparing Infinite Sets
Cardinality (|A|)
|A| = |B| ⟺ ∃ a bijection f : A → B
|A| ≤ |B| ⟺ ∃ an injection f : A → B
|A| < |B| ⟺ |A| ≤ |B| but |A| ≠ |B|

Cantor's Diagonal Argument — Proof that ℝ is Uncountable

Suppose for contradiction that all real numbers in (0,1) can be listed: r₁, r₂, r₃, … Write each in decimal:

r₁ = 0. d₁₁ d₁₂ d₁₃ d₁₄ … r₂ = 0. d₂₁ d₂₂ d₂₃ d₂₄ … r₃ = 0. d₃₁ d₃₂ d₃₃ d₃₄ … r₄ = 0. d₄₁ d₄₂ d₄₃ d₄₄ …

Construct x = 0.x₁x₂x₃… where xₙ ≠ dₙₙ (e.g., xₙ = 5 if dₙₙ ≠ 5, else xₙ = 4). Then x differs from every rₙ in the n-th decimal place. So x ∉ {r₁, r₂, r₃, …} — contradiction!

📜 Key Theorems

Theorem 1 — Cantor's Theorem
For any set A, the power set 𝒫(A) has strictly greater cardinality than A:
|A| < |𝒫(A)|
Proof idea: The function f : A → 𝒫(A) given by f(a) = {a} is injective, so |A| ≤ |𝒫(A)|. Suppose g : A → 𝒫(A) were surjective; define D = {a ∈ A : a ∉ g(a)}. Then D ∈ 𝒫(A) but D ≠ g(a) for any a — contradiction. Hence no surjection exists, so |A| < |𝒫(A)|. □
Theorem 2 — Schroeder-Bernstein Theorem
If |A| ≤ |B| and |B| ≤ |A|, then |A| = |B|.
Equivalently: if there exist injections f : A → B and g : B → A, then there exists a bijection h : A → B.
Why it matters: To show two sets have equal cardinality, it suffices to find injections in both directions — you don't need to explicitly construct a bijection. Used to prove |(0,1)| = |[0,1]|.
Theorem 3 — Countable Union of Countable Sets
A countable union of countable sets is countable.
If A₁, A₂, A₃, … are countable sets, then ⋃_{n=1}^∞ Aₙ is countable.
Application: ℚ = ⋃_{n=1}^∞ {p/n : p ∈ ℤ} is a countable union of countable sets → ℚ is countable.
Theorem 4 — Characterization of Uncountable Sets
Every infinite subset of a countable set is countable. Hence if A contains an uncountable subset B (with |B| ≤ |A|), then A itself is uncountable.
Use: (a,b) ⊂ ℝ is uncountable because ℝ is uncountable and injections exist both ways.
Theorem 5 — Finite Intersection Property (FIP)
A collection 𝒞 of sets has the FIP if every finite subcollection has non-empty intersection. A topological space is compact iff every collection of closed sets with the FIP has non-empty total intersection.
Relevance: Connects pure set theory to topology and analysis — concepts built in Unit 5.

✏️ Worked Examples

Example 1 — CSIR NET 2019 Style
Show that the set of all polynomials with integer coefficients is countable.
Solution: Let Pₙ = set of all polynomials with integer coefficients of degree exactly n.
Each such polynomial is determined by (n+1) integer coefficients, so Pₙ is in bijection with ℤⁿ⁺¹. Since ℤ is countable, ℤⁿ⁺¹ is a finite product of countable sets → countable. Hence Pₙ is countable.
The set of all polynomials with integer coefficients = ⋃_{n=0}^∞ Pₙ, a countable union of countable sets → countable. □
Example 2 — Bijection Construction
Show that |ℕ| = |{n² : n ∈ ℕ}| (the set of perfect squares has the same cardinality as ℕ).
Solution: Define f : ℕ → {n² : n ∈ ℕ} by f(n) = n².
Injective: If f(m) = f(n), then m² = n², so m = n (since m, n > 0).
Surjective: Any perfect square k² is f(k) for k ∈ ℕ.
So f is bijective → |ℕ| = |{perfect squares}| = ℵ₀. This counterintuitive result (a proper subset has the same cardinality) is characteristic of infinite sets.
Example 3 — Power Set Cardinality
If A = {1, 2, 3}, list 𝒫(A) and verify |𝒫(A)| = 2³.
Solution: 𝒫({1,2,3}) = {∅, {1}, {2}, {3}, {1,2}, {1,3}, {2,3}, {1,2,3}}
Count: 8 elements = 2³ ✓
Note: ∅ and A itself are always in 𝒫(A).
Example 4 — Schroeder-Bernstein Application
Show |(0,1)| = |[0,1]| without explicitly constructing a bijection.
Solution: We use Schroeder-Bernstein.
• Injection f : (0,1) → [0,1]: The inclusion map f(x) = x is injective.
• Injection g : [0,1] → (0,1): Define g(x) = x/2 + 1/4. Then g maps [0,1] into [1/4, 3/4] ⊂ (0,1), and g is clearly injective.
By Schroeder-Bernstein, a bijection exists → |(0,1)| = |[0,1]|. □
Example 5 — Countability by Listing
Is the set S = {1/n : n ∈ ℕ} countable?
Solution: Define f : ℕ → S by f(n) = 1/n. This is a bijection (injective: 1/m = 1/n ⟹ m = n; surjective: 1/n = f(n)). So S is countably infinite. In fact, S can be listed: 1, 1/2, 1/3, 1/4, … Note: S ≠ ℚ ∩ (0,1] even though both are countable.

Question 1 of 15
Which of the following sets is uncountable?
✓ D is correct. ℤ and ℚ are countably infinite. Algebraic numbers are also countable (countable union of roots of integer polynomials). ℝ∖ℚ is uncountable: if it were countable, then ℝ = ℚ ∪ (ℝ∖ℚ) would be a union of two countable sets → countable. But ℝ is uncountable — contradiction.
Question 2 of 15
If |A| = n, the number of proper subsets of A is:
✓ B is correct. Total subsets of A = 2ⁿ (including A itself). Proper subsets exclude A itself, so the count is 2ⁿ − 1. Note: Some textbooks also exclude ∅, giving 2ⁿ − 2, but the standard definition includes ∅ as a proper subset of any non-empty set.
Question 3 of 15
The set ℚ of rational numbers is countable. Which of the following best explains this?
✓ C is correct. A set is countably infinite iff there exists a bijection with ℕ. For ℚ: write rationals as p/q and enumerate by diagonalizing the grid ℤ × ℕ. This explicit listing gives a bijection. A is wrong (ℚ is infinite). B is wrong (ℝ is uncountable). D is false since |ℚ| = ℵ₀ < 𝔠 = |ℝ|.
Question 4 of 15
Let A = {1, 2} and B = {a, b, c}. What is |𝒫(A × B)|?
✓ C is correct. |A × B| = |A| · |B| = 2 × 3 = 6. The power set of a set with 6 elements has 2⁶ = 64 elements. So |𝒫(A × B)| = 2^6 = 64.
Question 5 of 15
Which of the following is TRUE about De Morgan's law for an indexed family {Aₙ}?
✓ A is correct. De Morgan's law: (⋃ Aₙ)ᶜ = ⋂ Aₙᶜ (complement of union = intersection of complements). Similarly, (⋂ Aₙ)ᶜ = ⋃ Aₙᶜ. These hold for arbitrary (even uncountable) indexed families.
Question 6 of 15
Consider the open intervals (0, 1/n) for n = 1, 2, 3, … What is ⋂_{n=1}^∞ (0, 1/n)?
✓ B is correct. ⋂ (0, 1/n) = ∅. For any x > 0, by the Archimedean property there exists n such that 1/n < x, so x ∉ (0, 1/n). And 0 ∉ (0, 1/n) since the interval is open. Hence no element belongs to all intervals, so the intersection is empty. This is a classic example where an infinite intersection of non-empty sets can be empty.
Question 7 of 15
The cardinality of 𝒫(ℕ) (power set of natural numbers) equals:
✓ C is correct. |𝒫(ℕ)| = 2^|ℕ| = 2^ℵ₀ = 𝔠 (the cardinality of the continuum). This equals |ℝ|. Note: ℵ₁ is the next cardinal after ℵ₀ (under AC), and whether 𝔠 = ℵ₁ is the Continuum Hypothesis — independent of ZFC.
Question 8 of 15
A function f : A → B is said to be a bijection if it is:
✓ C is correct. A bijection (one-to-one correspondence) must be both injective (one-to-one: distinct inputs give distinct outputs) and surjective (onto: every element of B is an output). Bijections are used to define equal cardinality between sets.
Question 9 of 15
Which of the following statements is FALSE? (CSIR NET 2023 style)
✓ C is FALSE (hence the correct answer). A is TRUE: infinite subsets of countable sets are countable. B is TRUE: bijection f: ℕ→ℤ by 0,1,−1,2,−2,… C is FALSE: ℝ is uncountable but ℚ ⊂ ℝ is countable — a subset of an uncountable set need not be uncountable! D is TRUE: finite subsets of ℕ can be enumerated (they form a countable collection).
Question 10 of 15
The symmetric difference A △ B equals:
✓ A is correct. A △ B = (A ∖ B) ∪ (B ∖ A) = elements in exactly one of A or B = (A ∪ B) ∖ (A ∩ B). This operation is commutative and associative, and (A △ B) △ C = A △ (B △ C).
Question 11 of 15
Cantor's theorem states that for any set A:
✓ C is correct. Cantor's theorem: |A| < |𝒫(A)| for every set A. This holds even for infinite sets: |ℕ| < |𝒫(ℕ)| (ℵ₀ < 2^ℵ₀). Consequence: there is no "largest" infinite set — you can always take the power set to get a bigger one.
Question 12 of 15
If A is a countably infinite set and B is a finite set, what is the cardinality of A ∪ B?
✓ A is correct. A ∪ B is a finite-or-countable union (A countable, B finite), so A ∪ B is countable. Since A ⊆ A ∪ B, we have |A ∪ B| ≥ ℵ₀. And A ∪ B ⊆ A ∪ B is countable, so |A ∪ B| = ℵ₀. Note: "ℵ₀ + n = ℵ₀" for any finite n — adding finitely many elements to an infinite countable set doesn't change its cardinality.
Question 13 of 15
The set S = {x ∈ ℝ : x is a root of a polynomial with integer coefficients} (algebraic numbers) is:
✓ B is correct. Polynomials with integer coefficients are countable (bounded by their degree and coefficients). Each polynomial of degree n has at most n real roots. A countable union of finite sets is countable. Hence algebraic numbers form a countable set. This implies most real numbers are transcendental (non-algebraic) — the transcendentals are uncountable!
Question 14 of 15
The Schroeder-Bernstein theorem is used to conclude |A| = |B| from:
✓ C is correct. Schroeder-Bernstein: if f : A → B injective and g : B → A injective, then |A| = |B|. It is powerful because finding explicit bijections for infinite sets is often hard (e.g., (0,1) vs [0,1]), but injections in both directions are easy to construct. Note: D also gives |A| = |B| by similar logic but is not the standard statement.
Question 15 of 15
Let f : ℕ → ℕ be defined by f(n) = n + 1. Which statement is TRUE?
✓ A is correct. Injective: f(m) = f(n) ⟹ m+1 = n+1 ⟹ m = n ✓. Not surjective: 1 ∈ ℕ has no preimage since f(n) = n+1 ≥ 2 for all n ∈ ℕ (taking ℕ = {1,2,3,…}). This shows an infinite set can have injections from itself to a proper subset — impossible for finite sets (pigeonhole principle). This is one characterization of infinite sets (Dedekind's definition).

📌 Summary & CSIR NET Tips

Must-Know Facts

  • ℕ, ℤ, ℚ, algebraic numbers → countable (ℵ₀)
  • ℝ, ℂ, ℝⁿ, 𝒫(ℕ), (a,b) → uncountable (𝔠 = 2^ℵ₀)
  • Countable ∪ Countable = Countable
  • |𝒫(A)| = 2^|A| always
  • Cantor: |A| < |𝒫(A)| for all A

Key Theorems

  • Cantor's Diagonal Argument → ℝ uncountable
  • Schroeder-Bernstein → injections both ways ⟹ bijection
  • Countable union of countable sets = countable
  • Infinite subset of countable = countable
  • Any proper subset of infinite set can have same cardinality

CSIR NET Common Q-Types

  • Identify countable/uncountable sets (2–3 Qs)
  • Power set cardinality computation
  • True/False about cardinality properties
  • De Morgan's law application
  • Bijection construction for standard sets

Common Mistakes to Avoid

  • ∅ ≠ {∅} — empty set vs. set containing empty set
  • Proper subsets: 2ⁿ − 1 (not 2ⁿ)
  • Subset of uncountable set need NOT be uncountable
  • (0,1) and ℝ have the SAME cardinality
  • ℵ₀ + ℵ₀ = ℵ₀ (cardinal arithmetic)
🏆
CSIR NET Exam Strategy: In Part B (2-mark questions), set theory questions are usually direct — know the countability table by heart. In Part C (4.75-mark questions), questions require constructing bijections or applying Cantor/Schroeder-Bernstein. Always verify both injectivity and surjectivity when claiming a bijection.
