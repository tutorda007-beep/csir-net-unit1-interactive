Cauchy's Theorem & Contour Integrals

Cauchy-Goursat · Integral Formula · Morera · Liouville · Maximum Modulus · Rouché

📐 Interactive: Contour Integration Visualizer

Cauchy's theorem: if f is analytic inside and on a simple closed contour C, then ∮_C f(z) dz = 0.

Core Theorems

Cauchy-Goursat Theorem Basic

If f is analytic inside and on a simple closed contour C, then: ∮_C f(z) dz = 0
  • C must be a simple closed curve (Jordan curve)
  • f must be analytic on C AND in the interior
  • No singularity inside or on C
  • Converse: Morera's theorem (zero integrals → analytic)

Cauchy Integral Formula Basic

If f analytic, z₀ inside C: f(z₀) = (1/2πi) ∮_C f(z)/(z−z₀) dz Generalization (nth derivative): f⁽ⁿ⁾(z₀) = n!/(2πi) ∮_C f(z)/(z−z₀)^(n+1) dz

Liouville's Theorem Intermediate

Every bounded entire function is constant.

Proof outline: f entire and |f(z)|≤M for all z. Then by CIF, |f'(z₀)| ≤ M/R → 0 as R→∞. So f'≡0, hence f is constant.

Application: Proof of Fundamental Theorem of Algebra. If p(z) has no root, then 1/p(z) is bounded entire → constant. Contradiction.

Maximum Modulus Principle Intermediate

If f analytic and non-constant in domain D, then |f(z)| attains its maximum only on the boundary ∂D.
  • |f| cannot have a maximum in the INTERIOR
  • Minimum Modulus: if f≠0, minimum also on boundary
  • Consequence: analytic functions are "open maps"
  • Schwarz Lemma: special case for the unit disk
Rouché's Theorem & Applications

Rouché's Theorem Advanced

If f, g analytic on and inside C, and |f(z)| > |g(z)| on C, then: f and f+g have the SAME number of zeros inside C (counted with multiplicity).

Strategy: to count zeros of h(z), write h = f + g where f dominates on boundary. Then zeros of h = zeros of f (which is usually easy).

Example: Count zeros of z⁵ + 3z + 1 inside |z|=2. Take f=z⁵, g=3z+1. On |z|=2: |f|=32, |g|≤7. So h has 5 zeros inside |z|=2.

Liouville's TheoremMedium
If f is an entire function with |f(z)| ≤ |z|² for all z ∈ ℂ, then f must be of the form:
Af(z) = az + b
Bf(z) = az²
Cf(z) = a sin z
Df(z) = ae^z
✓ Correct Answer: (B) — f(z) = az²
f is entire ⟹ has Taylor series: f(z) = a₀+a₁z+a₂z²+⋯. From CIF: |aₙ| = |f⁽ⁿ⁾(0)/n!| ≤ M_R/Rⁿ where |f(z)|≤R² gives M_R=R². So |aₙ| ≤ R²/Rⁿ = R^(2−n) → 0 for n>2. Thus a₃=a₄=⋯=0. Also a₀=f(0): |f(0)|≤0 → a₀=0. Similarly a₁=0. So f(z) = az².

Morera's TheoremAdvanced
Morera's theorem states: If f is continuous in D and ∮_C f(z) dz = 0 for every closed triangle C in D, then:
Af is bounded in D
Bf is analytic in D
Cf is constant in D
Df is zero in D
✓ Correct Answer: (B) — f is analytic in D
Morera's theorem is the converse of Cauchy-Goursat (in some sense). Zero integrals over triangles → f has a primitive F in D (F'=f) → F is analytic → f = F' is analytic. This is often used to prove analyticity by showing integral conditions rather than directly computing derivatives.

Laurent Series & Residue Theorem

Laurent Expansion · Isolated Singularities · Residues · Residue Theorem · Real Integrals

📊 Interactive: Laurent Series & Singularity Type

Laurent series: f(z) = Σ aₙ(z−z₀)ⁿ where sum runs from n=−∞ to ∞. The principal part (n<0 terms) determines singularity type.

Laurent Series Theory

Laurent Expansion Basic

f(z) = Σ_{n=−∞}^{∞} aₙ(z−z₀)ⁿ = ... + a₋₂/(z−z₀)² + a₋₁/(z−z₀) + a₀ + a₁(z−z₀) + ...

Principal part: terms with n < 0 (negative powers)

  • Coefficients: aₙ = (1/2πi)∮ f(z)/(z−z₀)^(n+1) dz
  • Valid in annular region R₁ < |z−z₀| < R₂
  • R₁ = 0 possible (punctured disk)
  • a₋₁ is the residue at z₀

Classifying Singularities Basic

From the principal part of the Laurent series at z₀:

  • Removable: no principal part (a₋ₙ=0 for all n≥1)
  • Pole of order m: principal part has finitely many terms; a₋ₘ≠0, a₋ₙ=0 for n>m
  • Essential: principal part has infinitely many terms

Order of pole m: lim (z−z₀)ᵐf(z) = L ≠ 0, ∞ at z₀

Computing Residues

Residue Formulas Intermediate

Simple pole at z₀: Res[f,z₀] = lim_{z→z₀} (z−z₀)·f(z) If f = p/q, q(z₀)=0, q'(z₀)≠0: Res[f,z₀] = p(z₀)/q'(z₀) Pole of order m at z₀: Res[f,z₀] = lim_{z→z₀} 1/(m−1)! · d^(m−1)/dz^(m−1) [(z−z₀)ᵐ f(z)]

Residue Theorem Intermediate

If f analytic inside/on C except at isolated singularities z₁,…,zₙ inside C: ∮_C f(z) dz = 2πi · Σ Res[f, zₖ]

This is the master tool for complex integration — all contour integrals reduce to computing residues!

Generalises Cauchy's theorem (no singularities → all residues = 0 → ∮=0).

Conformal Mappings & Möbius Transformations

Conformal Maps · Möbius/Bilinear · Cross-Ratio · Reflection Principle · Key Maps

📐 Interactive: How Analytic Functions Map Regions

A conformal map preserves angles locally. Any analytic function with f'(z)≠0 is conformal at z.

Conformal Mappings

What is a Conformal Map? Basic

A map f : D → ℂ is conformal at z₀ if it preserves angles between curves through z₀ (both magnitude AND direction/orientation).

f analytic at z₀ and f'(z₀) ≠ 0 ⟺ f is conformal at z₀
  • At z₀: magnification = |f'(z₀)|, rotation = arg(f'(z₀))
  • Critical points (f'=0): angle doubling/halving — not conformal
  • w=z² doubles angles at z=0: two lines at 45° map to lines at 90°
  • Riemann Mapping Theorem: every simply connected domain ≠ ℂ maps conformally to unit disk

Möbius Transformation Basic

w = (az + b) / (cz + d), ad − bc ≠ 0

Also called: linear fractional transformation / bilinear transformation.

  • Maps circles and lines to circles and lines
  • Fixed points: az+b=z(cz+d) → cz²+(d−a)z−b=0 (at most 2)
  • Determined by 3 point correspondences
  • Group under composition (bijection on ℂ∪{∞} = Riemann sphere)

Cross-Ratio Intermediate

(z, z₁; z₂, z₃) = [(z−z₁)(z₂−z₃)] / [(z−z₃)(z₂−z₁)]

Key property: Möbius transformations preserve the cross-ratio.

To find Möbius map sending z₁→0, z₂→1, z₃→∞:

w = (z−z₁)(z₂−z₃) / [(z−z₃)(z₂−z₁)]

Any 3 distinct points can be mapped to any 3 distinct points by a unique Möbius map.

Standard Maps Advanced

Upper half-plane → unit disk: w = (z − i) / (z + i) Unit disk → upper half-plane: z = i(1 − w) / (1 + w) Strip 0<Im z<π → upper half-plane: w = eᶻ Half-strip → unit disk: compose above

The map w=(z−i)/(z+i) sends: ℝ→|w|=1 (unit circle), i→0, 0→−1, ∞→1.

The map f(z) = z³ is conformal everywhere EXCEPT at:

Bz = i

✓ Correct Answer: (C) — z = 0

Riemann MappingAdvanced
The Riemann Mapping Theorem guarantees that every simply connected proper subdomain of ℂ:
AIs conformally equivalent to ℂ
BIs conformally equivalent to the upper half-plane only
CIs conformally equivalent to the open unit disk
DIs conformally equivalent to a rectangle
✓ Correct Answer: (C) — Conformally equivalent to the open unit disk
The Riemann Mapping Theorem: if D ⊊ ℂ is simply connected and non-empty, then there exists a bijective conformal map f: D → 𝔻 (unit disk). This is one of the deepest results in complex analysis. Note: ℂ itself is excluded (Liouville's theorem shows there's no bounded bijective analytic map from ℂ to 𝔻). The upper half-plane also works since it's conformally equivalent to 𝔻.

📌 CSIR Key Tips — Complex Analysis II

  • CIF: ∮ f(z)/(z−z₀)ⁿ⁺¹ dz = 2πi/n! · f⁽ⁿ⁾(z₀). Always check if singularity is INSIDE the contour first.
  • Residue theorem: ∮ f = 2πi · Σ Res. For simple pole p(z)/q(z): Res = p(z₀)/q'(z₀).
  • Laurent series: removable (no principal part), pole order m (m negative terms), essential (infinite negative terms).
  • Rouché: on C, |f|>|g| → zeros of f+g = zeros of f inside C.
  • Liouville: bounded + entire → constant. Key proof: FTA.
  • Cayley map: w=(z−i)/(z+i) sends upper half-plane to unit disk, ℝ to unit circle.
  • Conformal: analytic + f'≠0 ↔ conformal. f'=0 at critical points (angles multiply).
