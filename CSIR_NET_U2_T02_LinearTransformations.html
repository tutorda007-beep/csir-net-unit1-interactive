Introduction

A linear transformation is a map between vector spaces that preserves the two operations (addition and scalar multiplication). These are the structure-preserving maps of linear algebra, and they can always be represented by matrices once bases are chosen.

Key Concepts

  • Kernel (null space) and image (range)
  • Rank-Nullity theorem: fundamental constraint
  • Matrix representation depends on choice of basis
  • Change of basis → similar matrices
  • Invertible ↔ bijective ↔ isomorphism

CSIR NET Focus

  • Rank-Nullity calculations (very common)
  • Finding kernel and image from matrix
  • Determining invertibility from rank
  • Computing matrix of T in a given basis

Definition and Examples

Definition — Linear Transformation

T: V → W is a linear transformation (linear map) if for all u,v ∈ V and α ∈ 𝔽:

  1. Additivity: T(u + v) = T(u) + T(v)
  2. Homogeneity: T(αv) = αT(v)

Equivalently: T(αu + βv) = αT(u) + βT(v) for all α,β ∈ 𝔽.

Basic Consequences
  • T(0) = 0 (always maps zero to zero)
  • T(−v) = −T(v)
  • T(Σ αᵢvᵢ) = Σ αᵢ T(vᵢ) (linearity extends to all finite combinations)
  • T is completely determined by its values on a basis

Construction: T from Basis Values

If B = {b₁,...,bₙ} is a basis for V and w₁,...,wₙ ∈ W are arbitrary, there is a unique linear map T: V→W with T(bᵢ) = wᵢ. This is because every v = Σαᵢbᵢ forces T(v) = ΣαᵢT(bᵢ) = Σαᵢwᵢ.

Kernel and Image

Definitions

Kernel (null space): ker(T) = T⁻¹({0}) = {v ∈ V : T(v) = 0}

Image (range): im(T) = T(V) = {T(v) : v ∈ V} ⊆ W

  • ker(T) is a subspace of V
  • im(T) is a subspace of W

Nullity and Rank

Definitions

Nullity: nullity(T) = dim(ker T)

Rank: rank(T) = dim(im T)

T is Injective (one-to-one) ⟺

  • ker(T) = {0}
  • nullity(T) = 0
  • T maps linearly independent sets to linearly independent sets
  • T(u) = T(v) ⟹ u = v

T is Surjective (onto) ⟺

  • im(T) = W
  • rank(T) = dim W
  • Every w ∈ W has a preimage
  • T maps some spanning set of V to a spanning set of W

Rank-Nullity Theorem

Rank-Nullity Theorem (Dimension Theorem)

Let T: V → W be a linear transformation with V finite-dimensional. Then:

dim(ker T) + dim(im T) = dim V
nullity(T) + rank(T) = dim V
Proof Sketch

Let {u₁,...,uₖ} be a basis for ker(T) (nullity = k). Extend to a basis {u₁,...,uₖ,v₁,...,vᵣ} of V (so dim V = k+r). Claim: {T(v₁),...,T(vᵣ)} is a basis for im(T).

  • Spanning: Any T(v) = T(Σαᵢuᵢ + Σβⱼvⱼ) = Σβⱼ T(vⱼ) ∈ span{T(v₁),...,T(vᵣ)}
  • Independence: If Σβⱼ T(vⱼ) = 0, then T(Σβⱼvⱼ) = 0, so Σβⱼvⱼ ∈ ker(T) ⊆ span{u₁,...,uₖ}. But {u₁,...,uₖ,v₁,...,vᵣ} is independent, so all βⱼ = 0.

Thus rank(T) = r = dim V − k = dim V − nullity(T). ∎

Applications

ScenarioConsequence of RNT
T: V→W, dim V < dim WT cannot be surjective (rank ≤ dim V < dim W)
T: V→W, dim V > dim WT cannot be injective (nullity = dim V − rank ≥ dim V − dim W > 0)
T: V→V (same dimensions), injectiveT is also surjective (bijective)
dim V = dim W, T surjectiveT is also injective (bijective)
Key Insight

For linear maps between spaces of equal dimension: injective ⟺ surjective ⟺ bijective. This fails for non-linear maps and infinite-dimensional spaces (e.g., the shift operator on ℓ² is injective but not surjective).

Example

T: ℝ⁵ → ℝ³, rank(T) = 2. Then nullity(T) = 5 − 2 = 3. The kernel is a 3-dimensional subspace of ℝ⁵ and the image is a 2-dimensional subspace of ℝ³.

Matrix Representation

Matrix of a Linear Transformation

Let T: V → W, B = {b₁,...,bₙ} basis for V, C = {c₁,...,cₘ} basis for W. The matrix of T with respect to B and C is the m×n matrix [T]_{C,B} whose j-th column is [T(bⱼ)]_C (the coordinate vector of T(bⱼ) in basis C).

Then: [T(v)]_C = [T]_{C,B} · [v]_B

Computing the Matrix

Algorithm
  1. Apply T to each basis vector bⱼ of V to get T(bⱼ) ∈ W
  2. Write each T(bⱼ) as a linear combination of the C-basis vectors
  3. The coefficients form the j-th column of [T]_{C,B}
Example: Differentiation

T = d/dx: P₃(ℝ) → P₂(ℝ), B = {1,x,x²,x³}, C = {1,x,x²}.

T(1) = 0 = 0·1+0·x+0·x², T(x) = 1 = 1·1+0·x+0·x², T(x²) = 2x, T(x³) = 3x².

[T]_{C,B} = [[0,1,0,0],[0,0,2,0],[0,0,0,3]]

Change of Basis and Similar Matrices

Similar Matrices

If T: V→V and B, B' are two bases for V, then [T]_{B'} = P⁻¹[T]_B P where P is the change-of-basis matrix from B to B'. Matrices A and B are similar if B = P⁻¹AP for some invertible P; they represent the same linear operator in different bases.

Invariants Under Similarity

Similar matrices have the same: rank, nullity, determinant, trace, characteristic polynomial, eigenvalues, and minimal polynomial. These are intrinsic properties of the linear operator.

Composition of Linear Maps

Matrix Multiplication = Composition

If T: V→W and S: W→U with matrices [T]_{C,B} = A and [S]_{D,C} = B, then [S∘T]_{D,B} = B·A (matrix product). This is why matrix multiplication is defined as it is!

Invertibility and Isomorphisms

Definitions

T: V→W is an isomorphism if T is bijective (injective + surjective). Then T⁻¹: W→V is also an isomorphism.

V and W are isomorphic (V ≅ W) if there exists an isomorphism T: V→W.

Conditions for Invertibility

T: V→W is an isomorphism iff ALL of the following (in finite dimensions, any one implies the rest):

  • T is injective (ker T = {0})
  • T is surjective (im T = W)
  • rank(T) = dim V = dim W
  • [T]_{C,B} is an invertible matrix
  • T maps every basis of V to a basis of W
Classification of Vector Spaces

Over the same field 𝔽, any two n-dimensional vector spaces are isomorphic. The coordinate map v ↦ [v]_B gives an explicit isomorphism V ≅ 𝔽ⁿ.

Projections

Projection

T: V→V is a projection if T² = T (T is idempotent). Then:

  • V = im(T) ⊕ ker(T)
  • T acts as identity on im(T) and zero on ker(T)
  • I − T is also a projection (onto ker T)

Dual Transformations

Transpose (Dual) Map

Given T: V→W, the transpose (dual map) T*: W*→V* is defined by (T*f)(v) = f(T(v)) for f ∈ W*, v ∈ V.

Properties of T*
  • rank(T*) = rank(T)
  • ker(T*) = (im T)⁰ (annihilator of im T)
  • im(T*) = (ker T)⁰ (annihilator of ker T)
  • If [T]_{C,B} = A, then [T*]_{B*,C*} = Aᵀ (transpose matrix)

Annihilators

Annihilator

The annihilator of S ⊆ V is S⁰ = {f ∈ V* : f(v)=0 for all v∈S}. This is a subspace of V*.

dim(S⁰) = dim V − dim(span S).
