Introduction

A metric space is a set equipped with a notion of distance. The abstract framework captures all the essential properties of distance in ℝⁿ, function spaces, and beyond. Metric spaces are the foundation of functional analysis, topology, and analysis on general spaces.

Metric Space Axioms

Metric Space

A metric space (X, d) is a set X with a function d: X×X → [0,∞) satisfying:

  1. Non-negativity: d(x,y) ≥ 0, and d(x,y)=0 ⟺ x=y
  2. Symmetry: d(x,y) = d(y,x)
  3. Triangle inequality: d(x,z) ≤ d(x,y) + d(y,z)

Equivalent Metrics

Two metrics d₁, d₂ on X are equivalent if they define the same open sets (same topology). On ℝⁿ, all ℓᵖ metrics are equivalent. Equivalent metrics have the same convergent sequences and continuous functions.

Open & Closed Sets

Open Ball

B(x, r) = {y ∈ X : d(x,y) < r} — the open ball of radius r centered at x.

Open Set

U ⊆ X is open if for every x ∈ U, there exists r > 0 such that B(x,r) ⊆ U.

Properties: ∅ and X are open; arbitrary unions of open sets are open; finite intersections of open sets are open.

Closed Set

F ⊆ X is closed if X\F is open. Equivalently: F contains all its limit points.

Properties: ∅ and X are closed; arbitrary intersections of closed sets are closed; finite unions of closed sets are closed.

Convergence & Limits

Convergence

A sequence (xₙ) in X converges to x ∈ X (written xₙ→x) if:

∀ε>0, ∃N∈ℕ: n≥N ⟹ d(xₙ,x) < ε

In a metric space, limits are unique. xₙ→x iff d(xₙ,x)→0 in ℝ.

Cauchy Sequence

(xₙ) is Cauchy if:

∀ε>0, ∃N: m,n≥N ⟹ d(xₙ,xₘ) < ε

Every convergent sequence is Cauchy. The converse holds iff X is complete.

Sequential Characterization

In a metric space:

  • x ∈ cl(A) ⟺ some sequence in A converges to x.
  • A is closed ⟺ for every sequence in A that converges, its limit is in A.
  • A is open ⟺ every sequence converging to a point of A is eventually in A.

Continuity

Continuous Map (ε-δ)

f: (X,dₓ) → (Y,d_Y) is continuous at x₀ if:

∀ε>0, ∃δ>0: dₓ(x,x₀)<δ ⟹ d_Y(f(x),f(x₀))<ε

f is continuous on X if continuous at every point.

Equivalent Characterizations

f: X→Y is continuous iff any of these hold:

  • f⁻¹(V) is open in X for every open V ⊆ Y.
  • f⁻¹(F) is closed in X for every closed F ⊆ Y.
  • xₙ→x ⟹ f(xₙ)→f(x) (sequential continuity).
  • f(cl(A)) ⊆ cl(f(A)) for all A ⊆ X.
Uniform Continuity

f: X→Y is uniformly continuous if:

∀ε>0, ∃δ>0: dₓ(x,y)<δ ⟹ d_Y(f(x),f(y))<ε (δ independent of x,y)

Every continuous function on a compact metric space is uniformly continuous (Heine-Cantor theorem).

Standard Metrics & Comparisons

ℓᵖ Norms on ℝⁿ
‖x‖ₚ = (Σ|xᵢ|ᵖ)^(1/p) (1≤p<∞), ‖x‖∞ = max|xᵢ|

All ℓᵖ norms on ℝⁿ are equivalent. For 1≤p≤q≤∞: ‖x‖_q ≤ ‖x‖_p ≤ n^(1/p−1/q)‖x‖_q.

Function Space Metrics
  • C[a,b] with sup metric ‖f−g‖∞: complete.
  • L²[a,b] with ‖f−g‖₂: complete (Hilbert space).
  • Lᵖ[a,b] with ‖f−g‖ₚ: complete (Banach space).
Subspace Metric

If A ⊆ X, then (A, d|_A) is a metric space. "Open in A" ≠ "open in X": e.g., [0,1) is open in [0,2] but not in ℝ.
