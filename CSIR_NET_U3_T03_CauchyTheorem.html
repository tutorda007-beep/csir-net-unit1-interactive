Introduction

Cauchy's theorem is the fundamental result of complex analysis: the integral of a holomorphic function around a closed curve is zero. From this flows the Cauchy Integral Formula, which says that the values of an analytic function inside a curve are completely determined by its values on the curve — an extraordinary rigidity.

Contour Integrals

Complex Line Integral

Let γ: [a,b] → ℂ be a smooth curve (contour) and f continuous on γ. The contour integral is:

∫_γ f(z) dz = ∫_a^b f(γ(t)) · γ'(t) dt

Properties: ∫_γ (αf+βg) = α∫_γ f + β∫_γ g. ∫_{−γ} f = −∫_γ f. |∫_γ f dz| ≤ max|f|·length(γ) (ML-inequality).

ML Inequality (Estimation Lemma)
|∫_γ f(z) dz| ≤ M · L

where M = max_{z ∈ γ} |f(z)| and L = length(γ). Essential for showing integrals → 0 over large/small arcs.

Key Example

∮_{|z|=r} zⁿ dz: parameterize γ(t) = re^(it), t ∈ [0,2π]. Then ∫ = ∫_0^{2π} rⁿe^(int)·ire^(it)dt = irⁿ⁺¹∫_0^{2π} e^(i(n+1)t)dt. This equals 0 if n≠−1 and 2πi if n=−1.

Cauchy's Theorem

Cauchy's Theorem (Goursat version)

If f is holomorphic on a simply connected domain Ω, then for every closed curve γ in Ω:

∮_γ f(z) dz = 0
Cauchy's Theorem (General form)

If f is holomorphic on an open set containing a simple closed curve γ and its interior, then ∮_γ f(z) dz = 0.

Winding Number

The winding number (index) of a closed curve γ about a point z₀ ∉ γ is:

n(γ, z₀) = (1/2πi) ∮_γ dz/(z−z₀) ∈ ℤ

It counts how many times γ winds around z₀ counterclockwise. For a positively oriented simple closed curve: n(γ,z₀) = 1 for z₀ inside, 0 for z₀ outside.

Cauchy Integral Formula

Cauchy Integral Formula

If f is holomorphic on a domain containing a simple closed curve γ and its interior, and z₀ is inside γ, then:

f(z₀) = (1/2πi) ∮_γ f(z)/(z−z₀) dz

The value of f at any interior point z₀ is completely determined by f on the boundary γ!

Generalized Cauchy Integral Formula (Higher Derivatives)
f^(n)(z₀) = (n!/(2πi)) ∮_γ f(z)/(z−z₀)^(n+1) dz

Remarkable: every holomorphic function is infinitely differentiable (C^∞)! And each derivative is again holomorphic.

Example: Compute ∮_{|z|=2} e^z/(z−1) dz

f(z) = eᶻ is entire, z₀=1 is inside |z|=2. By CIF: ∮ eᶻ/(z−1)dz = 2πi·f(1) = 2πi·e.

Consequences of CIF

Power Series Expansion

Every holomorphic function has a convergent power series (Taylor series) at each point: a holomorphic function IS a power series locally. The radius of convergence extends to the nearest singularity.

Mean Value Property

If f is holomorphic on D(z₀,r), then:

f(z₀) = (1/2π) ∫_0^{2π} f(z₀ + re^(iθ)) dθ

The value at the center equals the average over any circle in the domain of holomorphicity.

Maximum Modulus Principle (Preview)

If f is holomorphic and non-constant on a domain Ω, then |f(z)| has no maximum in Ω. The maximum of |f| on a closed bounded domain is attained on the boundary.

Identity Theorem

If two holomorphic functions f, g on a connected domain Ω agree on a set with a limit point in Ω (e.g., a sequence converging to a point in Ω), then f ≡ g on all of Ω.

Liouville's Theorem & Morera's Theorem

Liouville's Theorem

Every bounded entire function is constant.

Proof: If |f(z)|≤M for all z∈ℂ, then by CIF: |f'(z₀)| ≤ M/R for any R>0. Letting R→∞: f'(z₀)=0, so f is constant.

Consequence: Fundamental Theorem of Algebra — every non-constant polynomial p(z) has a root in ℂ (if it had no root, 1/p(z) would be bounded and entire, hence constant, contradiction).

Morera's Theorem (Converse of Cauchy)

If f is continuous on a domain Ω and ∮_γ f(z) dz = 0 for every closed triangle γ in Ω, then f is holomorphic on Ω.

Cauchy's Inequality
|f^(n)(z₀)| ≤ n! · M_r / rⁿ

where M_r = max_{|z−z₀|=r} |f(z)|. Follows from the generalized CIF and ML-inequality.
