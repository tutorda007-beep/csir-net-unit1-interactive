Basic Concepts — Sequences

What is a Sequence? Basic

A sequence is a function a : ℕ → ℝ, written {aₙ}. Each aₙ is a term.

  • {1/n} = 1, 1/2, 1/3, 1/4, … → 0
  • {(−1)ⁿ} = −1, 1, −1, 1, … (diverges)
  • {n²} = 1, 4, 9, 16, … → ∞ (diverges)
lim aₙ = L ⟺ ∀ε>0, ∃N: n>N ⟹ |aₙ−L|<ε

Bounded & Monotone Basic

Bounded: ∃M>0: |aₙ| ≤ M for all n. Every convergent sequence is bounded. Converse is FALSE.

  • Increasing: a₁ ≤ a₂ ≤ a₃ ≤ …
  • Decreasing: a₁ ≥ a₂ ≥ a₃ ≥ …
Monotone Convergence Thm: bounded monotone ⟹ convergent

Series & Convergence Tests Intermediate

  • Divergence Test: If aₙ ↛ 0, then Σaₙ diverges
  • Comparison: 0 ≤ aₙ ≤ bₙ; Σbₙ conv ⟹ Σaₙ conv
  • Ratio Test: L = lim|aₙ₊₁/aₙ|. L<1 conv; L>1 div; L=1 inconclusive
  • Root Test: L = limsup|aₙ|^(1/n). L<1 conv; L>1 div
  • Alternating (Leibniz): bₙ ↘ 0 ⟹ Σ(−1)ⁿbₙ converges

Absolute vs Conditional Intermediate

Absolutely convergent: Σ|aₙ| converges ⟹ Σaₙ converges

Conditionally convergent: Σaₙ converges but Σ|aₙ| diverges

Example: Σ(−1)ⁿ⁺¹/n = ln 2 (conditional)

Riemann Rearrangement: A conditionally convergent series can be rearranged to sum to ANY real number!

Key Theorems

Bolzano-Weierstrass Theorem Advanced

Every bounded sequence in ℝ has a convergent subsequence.

Proof idea: repeatedly bisect intervals. If {aₙ} ⊆ [a,b], split into two halves — one contains infinitely many terms. Repeat to get a nested sequence of intervals → one limit point.

Equivalent to: closed bounded sets in ℝⁿ are compact (Heine-Borel).

Cauchy Criterion Advanced

{aₙ} converges ⟺ it is Cauchy: ∀ε>0, ∃N: m,n>N ⟹ |aₙ−aₘ|<ε

Works in complete metric spaces (like ℝ). Fails in ℚ (incomplete). Key advantage: no need to know the limit L in advance.

Power Series & Radius of Convergence Advanced

Σ aₙxⁿ  |  R = 1 / limsup |aₙ|^(1/n) (Cauchy-Hadamard)
  • Converges absolutely for |x| < R
  • Diverges for |x| > R
  • At |x| = R: check individually
  • Σn!xⁿ: R = 0 (converges only at x = 0)
  • Σxⁿ/n!: R = ∞ (converges everywhere)

Cauchy CriterionMedium
A sequence {aₙ} in ℝ converges if and only if it is:
ABounded
BA Cauchy sequence
CMonotone
DMonotone and bounded
✓ Correct Answer: (B) — A Cauchy sequence
In complete metric spaces like ℝ: convergent ⟺ Cauchy. This equivalence fails in ℚ (incomplete). Note: bounded alone does not imply convergence ({(−1)ⁿ} is bounded, diverges). Monotone+bounded (D) is sufficient but not necessary.

📌 Key Tips — Sequences & Series

  • Divergence test: If aₙ ↛ 0, Σaₙ definitely diverges. But aₙ → 0 is NOT enough (harmonic series!).
  • Memorize: Σ1/n² = π²/6 and Σ(−1)ⁿ⁺¹/n = ln 2 — they appear every year in CSIR.
  • Ratio test fails (L=1) for p-series — use integral/comparison test instead.
  • Riemann rearrangement: conditionally convergent → can rearrange to any sum. Absolutely convergent → rearrangement doesn't change sum.

Continuity & Differentiability

ε-δ Definition · Discontinuities · IVT · EVT · MVT · Uniform Continuity · Weierstrass Function

📊 Interactive: ε-δ Continuity Visualizer

For f(x)=x², continuity at a=1: as ε shrinks, δ also shrinks — but always exists. For discontinuous functions, δ fails to exist for small ε.

Basic Concepts

ε-δ Definition Basic

f is continuous at a if:

∀ε>0, ∃δ>0: |x−a|<δ ⟹ |f(x)−f(a)|<ε

Equivalently: (1) f(a) defined, (2) lim f(x) as x→a exists, (3) lim = f(a).

Types of Discontinuity Basic

  • Removable: lim exists but ≠ f(a). Fix: redefine f(a).
  • Jump: left-limit ≠ right-limit (e.g., sign function at 0).
  • Essential: lim doesn't exist (e.g., sin(1/x) at x=0).

Monotone functions have only jump discontinuities (at most countably many).

Uniform Continuity Intermediate

∀ε>0, ∃δ>0 (same δ everywhere!): |x−y|<δ ⟹ |f(x)−f(y)|<ε
  • Continuous on closed bounded interval ⟹ uniformly continuous (Heine-Cantor)
  • Lipschitz ⟹ uniformly continuous (not vice versa)
  • sin(x) is uniformly continuous on ℝ (Lipschitz with K=1)
  • x² is NOT uniformly continuous on ℝ

Differentiability Intermediate

f'(a) = lim_{h→0} [f(a+h)−f(a)] / h

Differentiable ⟹ Continuous (converse is FALSE!)

  • |x| is continuous at 0 but NOT differentiable
  • x^(2/3) is continuous, not differentiable at 0
  • Weierstrass function: continuous everywhere, differentiable nowhere
Key Theorems

Intermediate Value Theorem (IVT) Intermediate

If f:[a,b]→ℝ is continuous and c is strictly between f(a) and f(b), then ∃x₀ ∈ (a,b) with f(x₀) = c.

Applications:

  • Every odd-degree polynomial has a real root
  • Fixed point: f:[0,1]→[0,1] continuous ⟹ ∃c: f(c)=c (let g=f(x)−x, apply IVT)
  • Range of continuous function on [a,b] is an interval

Extreme Value Theorem (EVT) Intermediate

If f:[a,b]→ℝ is continuous on a closed bounded interval, then f is bounded and attains both its maximum and minimum.

Fails on open intervals: f(x)=1/x on (0,1) has no maximum. Fails if interval is unbounded: f(x)=x on ℝ has no max/min.

Mean Value Theorem — Lagrange (MVT) Advanced

If f is continuous on [a,b] and differentiable on (a,b), then ∃c ∈ (a,b) with:

f'(c) = [f(b) − f(a)] / (b − a)

Special case f(a)=f(b): Rolle's theorem gives f'(c)=0. Applications: f'≡0 ⟹ f=const; f'>0 ⟹ f strictly increasing.

Weierstrass Function Advanced

W(x) = Σ aⁿ cos(bⁿπx), 0<a<1, b odd, ab > 1+3π/2

This function is continuous everywhere (uniform limit of continuous functions) but differentiable nowhere on ℝ. A famous counterexample to the intuition that "continuous ⟹ differentiable almost everywhere."

Extreme Value TheoremMedium
If f:[a,b]→ℝ is continuous, then by the Extreme Value Theorem:
Af is differentiable on (a,b)
Bf attains its maximum and minimum values on [a,b]
Cf is monotone on [a,b]
Df has finitely many discontinuities
✓ Correct Answer: (B)
EVT: continuous on CLOSED BOUNDED [a,b] ⟹ f is bounded and attains both sup and inf. Requires: closed (not open) AND bounded. Fails for f(x)=1/x on (0,1) — unbounded. Fails for f(x)=x on ℝ — no max. Option A is NOT implied by continuity alone.
2
Mean Value TheoremMedium
By Lagrange's MVT, if f is continuous on [a,b] and differentiable on (a,b), then ∃c ∈ (a,b) with:
Af'(c) = [f(b)−f(a)] / (b−a)
Bf'(c) = f(b) − f(a)
Cf(c) = [f(a)+f(b)] / 2
Df'(c) = f(a) + f(b)
✓ Correct Answer: (A)
MVT equates instantaneous rate f'(c) with average rate [f(b)−f(a)]/(b−a). Geometrically: tangent at c is parallel to secant through (a,f(a)) and (b,f(b)). Proof: apply Rolle's theorem to g(x)=f(x) − (x−a)·[f(b)−f(a)]/(b−a).

The Weierstrass function (1872) is continuous everywhere (uniform limit of continuous functions) but differentiable nowhere. IVT Fixed PointAdvanced
Let f:[0,1]→[0,1] be continuous. By IVT, which must hold?
Af is monotone on [0,1]
Bf is differentiable at some point
Cf attains every value in [0,1] exactly once
D∃c ∈ [0,1] with f(c) = c (fixed point)
✓ Correct Answer: (D) — Fixed point exists
Let g(x) = f(x) − x. Then g(0) = f(0) ≥ 0 and g(1) = f(1)−1 ≤ 0 (since f maps into [0,1]). By IVT, ∃c ∈ [0,1] with g(c) = 0, i.e., f(c) = c. This is the 1D Brouwer Fixed Point Theorem. A CSIR classic!

📌 Key Tips — Continuity & Differentiability

  • Differentiable ⟹ Continuous, but NOT vice versa. Know examples: |x|, x^(1/3), Weierstrass.
  • Uniform continuity: continuous on CLOSED BOUNDED interval ⟹ uniformly continuous (Heine-Cantor). Open intervals: not necessarily.
  • IVT applications: fixed points, roots of polynomials, bisection method. Know the g(x)=f(x)−x trick!
  • MVT uses: f'=0 everywhere ⟹ f=const; prove monotonicity; estimate f(b)−f(a).

Riemann Integration

Partitions · Darboux Sums · Integrability Criterion · Fundamental Theorem · Lebesgue Criterion

📊 Interactive: Darboux Sums Visualizer

Visualizing f(x)=x² on [0,1]. Upper sum (red) ≥ integral ≥ Lower sum (blue). As n increases, both approach 1/3 (the actual integral).

Basic Concepts

Partitions & Darboux Sums Basic

A partition P of [a,b] is P = {x₀,x₁,…,xₙ} with a=x₀ < x₁ < … < xₙ=b.

U(f,P) = Σ Mᵢ·Δxᵢ  (Upper Darboux sum) L(f,P) = Σ mᵢ·Δxᵢ  (Lower Darboux sum)

Where Mᵢ = sup f, mᵢ = inf f on [xᵢ₋₁,xᵢ]. Always: L(f,P) ≤ U(f,P).

Riemann Integrability Intermediate

f is Riemann integrable on [a,b] if:

inf{U(f,P)} = sup{L(f,P)} = ∫ₐᵇ f dx

Equivalently: ∀ε>0, ∃P: U(f,P)−L(f,P) < ε.

Refinement: adding more partition points → L increases, U decreases.

Which Functions Are Integrable? Intermediate

  • Every continuous f on [a,b] is Riemann integrable ✅
  • Every monotone f on [a,b] is Riemann integrable ✅
  • Bounded f with finitely many discontinuities ✅
  • Dirichlet function: NOT Riemann integrable ❌
  • Cantor function: Riemann integrable ✅ (monotone, continuous)

Dirichlet Function Advanced

f(x) = 1 if x∈ℚ,   0 if x∉ℚ

For ANY partition: U(f,P) = 1 (rationals dense), L(f,P) = 0 (irrationals dense). So U−L = 1 always ⟹ NOT Riemann integrable.

But its Lebesgue integral = 0 (ℚ has measure 0).

Key Theorems

Lebesgue's Criterion for Riemann Integrability Advanced

A bounded function f:[a,b]→ℝ is Riemann integrable if and only if the set of discontinuities of f has Lebesgue measure zero.

(A set has measure zero if it can be covered by open intervals of arbitrarily small total length.)

This unifies: continuous ⟹ integrable; monotone ⟹ integrable (at most countably many jumps, measure 0); Dirichlet ⟹ NOT integrable (discontinuous everywhere).

Fundamental Theorem of Calculus — Part 1 Intermediate

If f is continuous on [a,b] and F(x) = ∫ₐˣ f(t) dt, then F is differentiable on (a,b) and:

F'(x) = f(x) for all x ∈ (a,b)

Differentiation UNDOES integration. F is called an antiderivative of f.

Fundamental Theorem of Calculus — Part 2 Intermediate

If f is Riemann integrable on [a,b] and G is any antiderivative of f (G'=f on (a,b)), then:

∫ₐᵇ f(x) dx = G(b) − G(a)

This is the evaluation formula. Note: requires f integrable AND G an antiderivative.

A bounded function f:[a,b]→ℝ is Riemann integrable if and only if:

Bf has finitely many discontinuities
CThe set of discontinuities of f has Lebesgue measure zero

Dirichlet FunctionAdvanced
The Dirichlet function f(x)=1 (x∈ℚ), f(x)=0 (x∉ℚ) on [0,1] is:
ARiemann integrable with integral = 1
BRiemann integrable with integral = 0
CRiemann integrable with integral = 1/2
DNOT Riemann integrable; Lebesgue integral = 0

FTC Part 1Medium
If f is continuous on [a,b] and F(x) = ∫ₐˣ f(t)dt, then:
AF'(x) = f(x) for all x ∈ (a,b)
BF(x) = f(x) for all x ∈ [a,b]
CF'(x) = f'(x) for all x ∈ (a,b)
DF is not differentiable
✓ Correct Answer: (A) — F'(x) = f(x)
FTC Part 1: The integral function F is differentiable when f is continuous, and F'(x) = f(x). Integration and differentiation are inverse operations. Common mistake: confusing F'=f (correct) with F=f (wrong). Option C confuses F' with f' — these need not be equal.
5
Darboux Sums RelationshipBasic
For bounded f on [a,b] and any partition P, the relationship between U(f,P) and L(f,P) is always:
AU(f,P) ≤ L(f,P)
BL(f,P) ≤ U(f,P)
CU(f,P) = L(f,P) always
DU(f,P) − L(f,P) = b−a always
✓ Correct Answer: (B) — L(f,P) ≤ U(f,P)
Since mᵢ = inf f ≤ Mᵢ = sup f on each subinterval, multiplying by Δxᵢ > 0 and summing: L(f,P) ≤ U(f,P) always. Equality holds iff f is constant on each subinterval. The function is Riemann integrable iff inf U(f,P) = sup L(f,P) over all partitions.

📌 Key Tips — Riemann Integration

  • Darboux = Riemann: both definitions of Riemann integral are equivalent for bounded functions.
  • FTC Part 1 vs Part 2: Part 1 says d/dx[∫ₐˣ f] = f(x); Part 2 says ∫ₐᵇ f = G(b)−G(a). Don't confuse them!
  • Lebesgue criterion is the most powerful: measure-zero discontinuity set ⟺ Riemann integrable.
  • Dirichlet function: NOT Riemann integrable (fails everywhere), but Lebesgue integral = 0. CSIR loves this!
