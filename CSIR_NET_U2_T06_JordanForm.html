Introduction

Not every matrix is diagonalizable — the Jordan Canonical Form (JCF) is the "best" upper triangular form achievable by similarity over an algebraically closed field (like ℂ). Every n×n complex matrix is similar to a unique JCF (up to ordering of blocks).

Generalized Eigenvectors

Generalized Eigenvectors

A vector v≠0 is a generalized eigenvector of rank k for eigenvalue λ if:

(A−λI)ᵏv = 0 but (A−λI)ᵏ⁻¹v ≠ 0

Ordinary eigenvectors have rank 1 (k=1). The generalized eigenspace for λ is:

Gλ = ker(A−λI)ⁿ (where n = dim V)

More precisely, Gλ = ker(A−λI)^m where m is the algebraic multiplicity of λ.

Theorem: Complete Decomposition

If the characteristic polynomial of A (over ℂ) splits as p_A(λ) = (λ−λ₁)^m₁ · ... · (λ−λₖ)^mₖ, then:

ℂⁿ = Gλ₁ ⊕ Gλ₂ ⊕ ... ⊕ Gλₖ

with dim Gλᵢ = mᵢ (the algebraic multiplicity). This is the primary decomposition.

Jordan Chains

Jordan Chain

A Jordan chain (or cycle of generalized eigenvectors) starting from vₖ:

vₖ, vₖ₋₁ = (A−λI)vₖ, vₖ₋₂ = (A−λI)²vₖ, ..., v₁ = (A−λI)^(k−1)vₖ

where v₁ is an eigenvector. These k vectors are linearly independent.

Jordan Blocks

Jordan Block Jₖ(λ)

The k×k Jordan block for eigenvalue λ is:

Jₖ(λ) = [ λ 1 0 ··· 0 ] [ 0 λ 1 ··· 0 ] [ 0 0 λ ··· 0 ] [ · · · ··· · ] [ 0 0 0 ··· λ ]

Diagonal entries: λ. Superdiagonal entries: 1. All other entries: 0.

Jₖ(λ) = λIₖ + Nₖ where Nₖ is the k×k nilpotent shift matrix (Nₖᵏ=0, Nₖᵏ⁻¹≠0).

Jordan Canonical Form

Jordan Decomposition Theorem

Every n×n matrix A over an algebraically closed field (e.g., ℂ) is similar to a Jordan matrix:

J = diag(J_{n₁}(λ₁), J_{n₂}(λ₂), ..., J_{nₖ}(λₖ))

This Jordan form is unique up to permutation of the Jordan blocks. There exists an invertible P such that A = PJP⁻¹.

Connection to Minimal Polynomial

Minimal Polynomial from JCF

The minimal polynomial m_A(x) of A is:

m_A(x) = (x−λ₁)^(s₁) · (x−λ₂)^(s₂) · ... · (x−λₖ)^(sₖ)

where sᵢ = size of the LARGEST Jordan block for eigenvalue λᵢ.

The characteristic polynomial p_A(x) = (x−λ₁)^(m₁) · ... · (x−λₖ)^(mₖ) where mᵢ = total size of all Jordan blocks for λᵢ (= algebraic multiplicity).

Computing the Jordan Canonical Form

Step-by-Step Algorithm
  1. Find eigenvalues (over ℂ): roots of det(A−λI)=0 with multiplicities mᵢ.
  2. For each eigenvalue λᵢ: Compute ranks of (A−λᵢI), (A−λᵢI)², ..., (A−λᵢI)^mᵢ.
  3. Count Jordan blocks: The number of Jordan blocks of size ≥ k for eigenvalue λ is rank(A−λI)^(k-1) − rank(A−λI)^k.
  4. Jordan blocks of size exactly k: (number of size ≥ k) − (number of size ≥ k+1).
  5. Assemble J from the blocks.
Key Formula

For eigenvalue λ, let dₖ = nullity(A−λI)ᵏ = dim ker(A−λI)ᵏ. Then:

number of Jordan blocks of size exactly k = (dₖ − dₖ₋₁) − (dₖ₊₁ − dₖ) = 2dₖ − dₖ₋₁ − dₖ₊₁
Example: 4×4 Matrix with λ=2 (algebraic mult 4)

Suppose: nullity(A−2I)=2, nullity(A−2I)²=3, nullity(A−2I)³=4.

Size ≥1: d₁−d₀ = 2−0 = 2 blocks. Size ≥2: d₂−d₁ = 3−2 = 1 block. Size ≥3: d₃−d₂ = 4−3 = 1 block. Size ≥4: d₄−d₃ = 4−4 = 0 blocks.

Blocks of size exactly 1: 2−1=1. Size exactly 2: 1−1=0. Size exactly 3: 1−0=1.

JCF for λ=2: one 3×3 block and one 1×1 block: J₃(2) ⊕ J₁(2).

Applications of Jordan Form

Matrix Functions

Function of a Jordan Block

For f analytic: f(Jₖ(λ)) is upper triangular with:

[f(Jₖ(λ))]ᵢⱼ = f^(j−i)(λ)/(j−i)! for j ≥ i = 0 for j < i

In particular: eᴬ = P·e^J·P⁻¹ where e^(Jₖ(λ)) = e^λ·[[1,1,1/2!,...,1/(k-1)!],[0,1,1,...,1/(k-2)!],...,[0,...,0,1]].

Nilpotent Matrices

Nilpotent = JCF has λ=0 only

A is nilpotent iff p_A(x)=xⁿ iff all eigenvalues are 0 iff Aᵏ=0 for some k. The JCF of a nilpotent matrix consists entirely of Jordan blocks J_{nᵢ}(0) (shift matrices). The minimal polynomial is x^s where s = size of the largest Jordan block.

Rational Canonical Form

Over Non-Algebraically Closed Fields

Over ℝ (or ℚ), the characteristic polynomial may not split. In this case we use the Rational Canonical Form (companion matrix blocks for invariant factors). Over ℂ, JCF = special case of RCF.
