Introduction

Normal subgroups are the "right" kind of subgroups that allow us to form quotient groups. Group homomorphisms are the structure-preserving maps between groups. Together they lead to the isomorphism theorems — the most fundamental tools for classifying groups.

Normal Subgroups

Normal Subgroup

N ◁ G (N is normal in G) if for all g ∈ G: gNg⁻¹ = N, i.e., gNg⁻¹ ⊆ N.

Equivalently: left cosets = right cosets: gN = Ng for all g ∈ G.

Criteria for Normality

  • Every subgroup of an abelian group is normal.
  • Any subgroup of index 2 is normal. (Because there are only two cosets: H and G\H.)
  • The kernel of any homomorphism is a normal subgroup.
  • The center Z(G) = {g ∈ G : gx=xg for all x} is always normal.
  • Intersection of normal subgroups is normal.
Examples
  • Aₙ ◁ Sₙ (index 2).
  • SL(n) ◁ GL(n) (kernel of det: GL→ℝ*).
  • nℤ ◁ ℤ (ℤ is abelian).
  • In S₃: {e,(123),(132)} ◁ S₃ but {e,(12)} is not normal.
Simple Group

A group G is simple if its only normal subgroups are {e} and G itself.

Examples: ℤₚ (p prime), A₅, A₆, A₇, ... Simple groups are the "atoms" of group theory.

Quotient Groups (Factor Groups)

Quotient Group G/N

If N ◁ G, the set of cosets G/N = {gN : g ∈ G} forms a group under:

(aN)(bN) = (ab)N

This is well-defined only when N is normal. The identity is N = eN, and (gN)⁻¹ = g⁻¹N.

|G/N| = [G:N] = |G|/|N|.

Key Examples
  • ℤ/nℤ ≅ ℤₙ (integers mod n).
  • ℝ/ℤ ≅ S¹ (circle group, angles mod 2π).
  • S₃/A₃ ≅ ℤ₂ (since |S₃/A₃|=2).
  • GL(n)/SL(n) ≅ ℝ* (multiplicative reals).
Natural Projection

The map π: G → G/N given by π(g) = gN is a surjective homomorphism with kernel N.

This is called the natural projection (or canonical homomorphism).

Group Homomorphisms

Homomorphism

A map φ: G → H is a homomorphism if φ(ab) = φ(a)φ(b) for all a,b ∈ G.

  • Monomorphism: injective homomorphism
  • Epimorphism: surjective homomorphism
  • Isomorphism: bijective homomorphism (G ≅ H)
  • Automorphism: isomorphism G → G
  • Endomorphism: homomorphism G → G

Kernel and Image

Kernel and Image

ker φ = {g ∈ G : φ(g) = eH} — always a normal subgroup of G.

im φ = φ(G) = {φ(g) : g ∈ G} — always a subgroup of H.

φ is injective ⟺ ker φ = {e}.

Basic Properties of Homomorphisms
  • φ(eG) = eH
  • φ(a⁻¹) = φ(a)⁻¹
  • φ(aⁿ) = φ(a)ⁿ
  • ord(φ(a)) divides ord(a)
  • The image of a normal subgroup under an epimorphism is normal.

Isomorphism Theorems

First Isomorphism Theorem

If φ: G → H is a homomorphism, then:

G/ker φ ≅ im φ

The quotient of G by the kernel is isomorphic to the image. The isomorphism is φ̄(gK) = φ(g) where K = ker φ.

Application

φ: ℤ → ℤₙ by φ(k) = k mod n. ker φ = nℤ, im φ = ℤₙ. First isomorphism theorem gives ℤ/nℤ ≅ ℤₙ.

Second Isomorphism Theorem

If H ≤ G and N ◁ G, then HN is a subgroup, H∩N ◁ H, and:

H/(H∩N) ≅ HN/N
Third Isomorphism Theorem

If N ◁ G and K ◁ G with N ≤ K, then K/N ◁ G/N and:

(G/N)/(K/N) ≅ G/K
Correspondence Theorem

If N ◁ G, there is a bijection between subgroups of G/N and subgroups of G containing N. Normal subgroups correspond to normal subgroups.
