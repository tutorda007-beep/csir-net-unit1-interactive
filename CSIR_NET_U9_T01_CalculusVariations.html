Introduction

Calculus of Variations (CoV) finds functions that extremize functionals (integrals of functions of a function and its derivative). Linear Programming (LP) optimizes a linear objective over a feasible region defined by linear constraints.

Functionals & Euler-Lagrange Equation

Functional
J[y] = ∫_a^b F(x, y(x), y′(x)) dx Boundary conditions: y(a)=y_a, y(b)=y_b Find y that extremizes J[y].
Euler-Lagrange Equation

Necessary condition for a local extremum: y satisfies

∂F/∂y − d/dx(∂F/∂y′) = 0

Derived by setting the first variation δJ=0 for all admissible variations δy with δy(a)=δy(b)=0.

Classic Example: Shortest Path

F=√(1+y′²). E-L equation: d/dx[y′/√(1+y′²)]=0 ⟹ y′=const ⟹ y=ax+b (straight line).

Special Cases & Applications

Beltrami Identity (F not explicitly depending on x)
F − y′·∂F/∂y′ = C (constant)

Used when ∂F/∂x=0 (F independent of x explicitly).

Hamiltonian Formulation
p = ∂F/∂y′ (conjugate momentum) H(x,y,p) = py′ − F(x,y,y′) Hamilton's equations: y′=∂H/∂p, p′=−∂H/∂y

Constrained Variational Problems

Isoperimetric Problem

Extremize J[y]=∫F dx subject to K[y]=∫G dx = const (integral constraint).

Method: use Lagrange multiplier λ — extremize ∫(F−λG)dx. Euler-Lagrange: ∂F/∂y−d/dx(∂F/∂y′)=λ(∂G/∂y−d/dx(∂G/∂y′)).

Classic: Dido's problem — enclose max area with fixed perimeter ⟹ circle.
Free Endpoint / Natural BCs

If y(b) is free: natural boundary condition ∂F/∂y′|_{x=b}=0.

Second Variation & Legendre Condition

For J to have a minimum: second variation δ²J≥0. Necessary: ∂²F/∂y′²≥0 (Legendre condition). Jacobi condition gives sufficiency for local minimum.

Linear Programming

Standard Form LP
minimize cᵀx subject to Ax = b, x ≥ 0

Feasible region is a convex polyhedron; optimal occurs at a vertex (basic feasible solution).

Simplex Method

Move from vertex to adjacent vertex improving objective. Terminates because: (1) finitely many vertices, (2) non-degeneracy. Tableau form; pivot on entering/leaving variable.

LP Duality
Primal: min cᵀx, Ax≥b, x≥0 Dual: max bᵀy, Aᵀy≤c, y≥0 Weak duality: cᵀx ≥ bᵀy (primal ≥ dual) Strong duality: optimal values are equal if feasible

Complementary slackness: at optimum, if xⱼ>0 then jth dual constraint is tight; if dual yᵢ>0 then ith primal constraint is tight.
