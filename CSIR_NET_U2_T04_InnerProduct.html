Introduction

An inner product adds geometry (angles and lengths) to a vector space. The Gram-Schmidt process constructs orthonormal bases from any basis, and the Spectral Theorem reveals that symmetric (and more generally normal) matrices have a beautiful orthogonal diagonalization.

Inner Products

Definition — Inner Product (Real)

A real inner product on V is ⟨·,·⟩: V×V → ℝ satisfying:

  1. Symmetry: ⟨u,v⟩ = ⟨v,u⟩
  2. Bilinearity: ⟨αu+βv,w⟩ = α⟨u,w⟩ + β⟨v,w⟩
  3. Positive definiteness: ⟨v,v⟩ ≥ 0, with ⟨v,v⟩ = 0 iff v = 0

Induced norm: ‖v‖ = √⟨v,v⟩. Distance: d(u,v) = ‖u−v‖.

Cauchy-Schwarz Inequality
|⟨u,v⟩| ≤ ‖u‖ · ‖v‖

Equality holds iff u and v are linearly dependent. Consequence: the angle θ between u and v is well-defined by cos θ = ⟨u,v⟩/(‖u‖‖v‖).

Parallelogram Law
‖u+v‖² + ‖u−v‖² = 2(‖u‖² + ‖v‖²)

A norm comes from an inner product iff the parallelogram law holds.

Orthogonality

Definitions

Orthogonal: u ⊥ v if ⟨u,v⟩ = 0. Generalized Pythagorean theorem: u⊥v ⟹ ‖u+v‖² = ‖u‖² + ‖v‖².

Orthogonal set: {v₁,...,vₖ} with ⟨vᵢ,vⱼ⟩ = 0 for i≠j.

Orthonormal set: orthogonal + unit vectors: ⟨vᵢ,vⱼ⟩ = δᵢⱼ (Kronecker delta).

Orthogonal complement: W⊥ = {v : ⟨v,w⟩=0 for all w ∈ W}.

Key Theorems
  • Any orthogonal set of non-zero vectors is linearly independent
  • If {e₁,...,eₙ} is orthonormal basis: v = Σ⟨v,eᵢ⟩eᵢ (Fourier expansion)
  • Parseval's identity: ‖v‖² = Σ|⟨v,eᵢ⟩|² for orthonormal basis {eᵢ}
  • V = W ⊕ W⊥ when W is a closed subspace (finite-dimensional V)
  • dim(W) + dim(W⊥) = dim(V)

Orthogonal Matrix

Orthogonal Matrix

Q is orthogonal if QᵀQ = QQᵀ = I, i.e., Q⁻¹ = Qᵀ. Columns (and rows) of Q form an orthonormal basis. det(Q) = ±1. Q preserves norms and inner products: ⟨Qu,Qv⟩ = ⟨u,v⟩.

Gram-Schmidt Orthogonalization

Gram-Schmidt Process

Given linearly independent vectors {v₁,...,vₙ}, produce orthonormal set {e₁,...,eₙ} with span{e₁,...,eₖ} = span{v₁,...,vₖ} for all k:

u₁ = v₁, e₁ = u₁/‖u₁‖ u₂ = v₂ − ⟨v₂,e₁⟩e₁, e₂ = u₂/‖u₂‖ u₃ = v₃ − ⟨v₃,e₁⟩e₁ − ⟨v₃,e₂⟩e₂, e₃ = u₃/‖u₃‖ uₖ = vₖ − Σᵢ₌₁ᵏ⁻¹ ⟨vₖ,eᵢ⟩eᵢ, eₖ = uₖ/‖uₖ‖
QR Decomposition

Gram-Schmidt applied to the columns of an m×n matrix A (rank n) produces: A = QR, where Q has orthonormal columns and R is upper triangular with positive diagonal. Useful for solving least-squares problems and numerical linear algebra.

Example: ℝ³

v₁=(1,1,0), v₂=(1,0,1), v₃=(0,1,1).

u₁=(1,1,0), e₁=(1,1,0)/√2.

⟨v₂,e₁⟩=(1·1+0·1+1·0)/√2=1/√2. u₂=v₂−(1/√2)e₁=(1,0,1)−(1/2,1/2,0)=(1/2,−1/2,1). e₂=u₂/‖u₂‖.

Continue for u₃, e₃ to get a full orthonormal basis.

Orthogonal Complement via Gram-Schmidt

To find W⊥ (orthogonal complement of a subspace W ⊆ ℝⁿ): extend a basis of W to a basis of ℝⁿ, apply Gram-Schmidt, then the extra orthonormal vectors span W⊥.

Orthogonal Projections

Orthogonal Projection onto Subspace W

The orthogonal projection of v onto W (subspace with orthonormal basis {e₁,...,eₖ}) is:

proj_W(v) = ⟨v,e₁⟩e₁ + ⟨v,e₂⟩e₂ + ... + ⟨v,eₖ⟩eₖ

The error v − proj_W(v) ⊥ W and ‖v − proj_W(v)‖ is minimized (best approximation).

Best Approximation Theorem

The orthogonal projection proj_W(v) is the closest point to v in W:

‖v − proj_W(v)‖ ≤ ‖v − w‖ for all w ∈ W

with equality iff w = proj_W(v).

Projection Matrix

Formula

If W = col(A) (column space of A, A with linearly independent columns), the projection matrix onto W is:

P = A(AᵀA)⁻¹Aᵀ

Properties: P² = P (idempotent), Pᵀ = P (symmetric), P is the orthogonal projection onto col(A).

Least Squares

Least Squares Solution

If Ax=b has no exact solution, the least-squares solution minimizes ‖Ax−b‖². Solution: x* = (AᵀA)⁻¹Aᵀb (when AᵀA invertible). The residual b−Ax* ⊥ col(A).

Spectral Theorem

Spectral Theorem (Real Symmetric Matrices)

Every real symmetric matrix A (Aᵀ=A) is orthogonally diagonalizable: there exists an orthogonal matrix Q and diagonal D such that:

A = QDQᵀ (A = QDQ⁻¹ with Q⁻¹=Qᵀ)

Equivalently: ℝⁿ has an orthonormal basis of eigenvectors of A. All eigenvalues are real.

Key Properties of Symmetric Matrices
  • All eigenvalues are real
  • Eigenvectors for distinct eigenvalues are orthogonal
  • Always diagonalizable (GM = AM for every eigenvalue)
  • The orthonormal eigenvectors form columns of Q

Normal Matrices

Definition

A (complex) matrix A is normal if AA* = A*A where A* = Āᵀ (conjugate transpose). Normal matrices include: symmetric (A=Aᵀ), skew-symmetric, orthogonal, Hermitian (A=A*), unitary (A*A=I).

Spectral Theorem (Normal Matrices)

A (complex) matrix A is normal iff there exists a unitary matrix U (U*U=I) and diagonal D such that A = UDU*. Equivalently, ℂⁿ has an orthonormal basis of eigenvectors of A.

Positive Definite Matrices

Definitions

A symmetric matrix A is:

  • Positive definite (PD): xᵀAx > 0 for all x ≠ 0 ⟺ all eigenvalues > 0
  • Positive semidefinite (PSD): xᵀAx ≥ 0 for all x ⟺ all eigenvalues ≥ 0
  • Negative definite: xᵀAx < 0 for all x ≠ 0 ⟺ all eigenvalues < 0
  • Indefinite: has both positive and negative eigenvalues

A is PD ⟺ A = BᵀB for some invertible B (Cholesky) ⟺ all leading principal minors > 0 (Sylvester's criterion).

Adjoint Operators

Adjoint of a Linear Operator

Given T: V→V on inner product space V, the adjoint T*: V→V is defined by ⟨Tv,w⟩ = ⟨v,T*w⟩ for all v,w ∈ V.

For a matrix A with standard inner product: T* corresponds to Aᵀ (real) or A* = Āᵀ (complex).

Property of TMatrix conditionEigenvalue property
Self-adjoint (T*=T)A = Aᵀ (real symmetric)All real eigenvalues
Skew-adjoint (T*=−T)A = −Aᵀ (skew-symmetric)Purely imaginary eigenvalues
Unitary (T*=T⁻¹)Aᵀ=A⁻¹ (orthogonal)All eigenvalues on unit circle
