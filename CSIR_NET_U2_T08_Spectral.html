Introduction

Spectral theory extends far beyond eigenvalues of square matrices. The Singular Value Decomposition (SVD) applies to every matrix (rectangular or square, real or complex) and underlies modern data science. Perron-Frobenius governs non-negative matrices, and matrix functions (exponential, logarithm) are key to differential equations.

Singular Value Decomposition (SVD)

SVD Theorem

Every real m×n matrix A can be written as:

A = UΣVᵀ

where U is m×m orthogonal, V is n×n orthogonal, and Σ is m×n diagonal with non-negative singular values σ₁ ≥ σ₂ ≥ ... ≥ σᵣ > 0 = σᵣ₊₁ = ... (r = rank A) on the diagonal.

Computing SVD
  • Singular values: σᵢ = √(λᵢ(AᵀA)) where λᵢ are eigenvalues of AᵀA (≥0).
  • Right singular vectors: columns of V = eigenvectors of AᵀA.
  • Left singular vectors: columns of U = eigenvectors of AAᵀ.
  • Relation: Avᵢ = σᵢuᵢ, Aᵀuᵢ = σᵢvᵢ.

Low-Rank Approximation (Eckart-Young Theorem)

The best rank-k approximation to A (in spectral or Frobenius norm) is:

Aₖ = U diag(σ₁,...,σₖ,0,...,0) Vᵀ = Σᵢ₌₁ᵏ σᵢ uᵢvᵢᵀ

with error ‖A−Aₖ‖₂ = σₖ₊₁ and ‖A−Aₖ‖_F = √(σₖ₊₁²+...+σᵣ²).

Matrix Functions

Matrix Exponential

Definition
eᴬ = I + A + A²/2! + A³/3! + ... = Σₙ₌₀^∞ Aⁿ/n!

This series always converges for every matrix A. Key properties:

  • e^(A+B) = eᴬeᴮ if AB = BA (commuting matrices)
  • (eᴬ)⁻¹ = e^(−A)
  • det(eᴬ) = e^(tr A)
  • If A = PDP⁻¹ (diagonalizable): eᴬ = Pe^D P⁻¹
  • Solution to x'(t)=Ax, x(0)=x₀ is x(t) = e^(tA)·x₀
Computing e^A via Diagonalization

If A = PDP⁻¹ with D = diag(λ₁,...,λₙ), then eᴬ = P·diag(e^λ₁,...,e^λₙ)·P⁻¹.

For Jordan form A = PJP⁻¹: e^A = P·e^J·P⁻¹ where e^(Jₖ(λ)) is computed using the formula from T06.

Cayley-Hamilton Method

Computing f(A) via Cayley-Hamilton

For any analytic function f, f(A) can be written as a polynomial of degree ≤ n−1 in A (by reducing using p_A(A)=0). Specifically, write f(λ) ≡ q(λ) mod p_A(λ) and then f(A) = q(A).

Non-Negative Matrices

Definitions

A matrix A is non-negative (A≥0) if all entries aᵢⱼ≥0. It is positive (A>0) if all aᵢⱼ>0. A is stochastic if A≥0 and each row sums to 1.

Perron-Frobenius Theorem

Positive matrices (A>0): There exists a unique positive eigenvalue ρ(A) (the Perron root or spectral radius) such that:

  • ρ(A) is a simple eigenvalue (algebraic multiplicity 1)
  • |λ| < ρ(A) for all other eigenvalues λ
  • The eigenvector for ρ(A) has all positive entries (Perron vector)

Non-negative irreducible matrices: Same conclusions hold for irreducible A≥0 (no permutation gives block triangular form).

Gershgorin Circle Theorem

Gershgorin Discs

Every eigenvalue of A lies in at least one Gershgorin disc:

Dᵢ = { z ∈ ℂ : |z − aᵢᵢ| ≤ Rᵢ } where Rᵢ = Σⱼ≠ᵢ |aᵢⱼ|

If k discs form a connected component isolated from the rest, exactly k eigenvalues lie in their union.

Companion Matrix & Rational Canonical Form

Companion Matrix

The companion matrix of monic polynomial p(x) = xⁿ + aₙ₋₁xⁿ⁻¹ + ... + a₁x + a₀ is:

C(p) = [ 0 0 ··· 0 −a₀ ] [ 1 0 ··· 0 −a₁ ] [ 0 1 ··· 0 −a₂ ] [ · · ··· · · ] [ 0 0 ··· 1 −aₙ₋₁ ]

The characteristic polynomial of C(p) equals p(x). So any polynomial is the characteristic polynomial of some matrix.

Rational Canonical Form (RCF)

Every matrix A is similar (over any field F) to a block-diagonal matrix of companion matrices:

A ∼ diag(C(f₁), C(f₂), ..., C(fₖ)) with f₁|f₂|...|fₖ

The invariant factors f₁,...,fₖ satisfy fᵢ|fᵢ₊₁. The largest invariant factor fₖ = minimal polynomial of A. The product f₁·f₂·...·fₖ = characteristic polynomial of A.

RCF vs JCF

RCF works over any field (no need for algebraically closed). JCF is a special case of RCF over ℂ where each invariant factor is a power of a linear factor (x−λ)^k.
