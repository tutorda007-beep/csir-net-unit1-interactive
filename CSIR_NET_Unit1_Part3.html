Linear Algebra

Vector Spaces · Basis & Dimension · Eigenvalues · Cayley-Hamilton · Rank-Nullity · Inner Product Spaces

Foundations

Vector Spaces Basic

A set V over field F with + and · satisfying 8 axioms (closure, associativity, identity, inverse, distributivity).

  • ℝⁿ, ℂⁿ, Mₙₓₙ (matrices), Pₙ (polynomials), C[a,b] (functions) are all vector spaces.
  • Subspace: non-empty subset closed under + and scalar ·
  • Intersection of subspaces = subspace; union need NOT be

Basis & Dimension Basic

A set B = {v₁,…,vₙ} is a basis if: (1) B is linearly independent; (2) B spans V.

dim(V) = number of elements in any basis
  • Every basis of V has the same cardinality
  • dim(ℝⁿ) = n; dim(Mₙₓₙ) = n²; dim(Pₙ) = n+1
  • dim(U+W) = dim(U)+dim(W) − dim(U∩W)

Rank-Nullity Theorem Intermediate

rank(T) + nullity(T) = dim(domain)

For T:ℝⁿ→ℝᵐ (m×n matrix A): rank(A) + null(A) = n.

  • rank(A) = dim(column space) = dim(row space)
  • nullity(A) = dim(null space) = dim of solution space of Ax=0
  • rank(A) = rank(Aᵀ) always
  • rank(AB) ≤ min(rank A, rank B)

Eigenvalues & Eigenvectors Intermediate

Av = λv, v ≠ 0 ⟺ det(A−λI) = 0
  • Characteristic polynomial: p(λ) = det(A−λI)
  • Eigenvalues = roots of p(λ)
  • Eigenvectors form eigenspaces (subspaces!)
  • Distinct eigenvalues ⟹ eigenvectors are linearly independent
  • Trace = Σλᵢ, Det = Πλᵢ
Key Theorems

Cayley-Hamilton Theorem Advanced

Every square matrix satisfies its own characteristic equation. If p(λ) = det(A−λI), then p(A) = 0.

Example: A = [[1,2],[2,1]], p(λ) = λ²−2λ−3 ⟹ A²−2A−3I = 0

Use to: compute A⁻¹ (if det≠0), find high powers Aⁿ, check minimal polynomial.

Diagonalization Advanced

A is diagonalizable iff A has n linearly independent eigenvectors. Then A = PDP⁻¹ where D=diag(λ₁,…,λₙ), P=[v₁|…|vₙ].

  • Symmetric real matrices are always diagonalizable (Spectral Theorem)
  • n distinct eigenvalues ⟹ diagonalizable
  • Not all matrices are diagonalizable (e.g., Jordan blocks)
  • If diagonalizable: Aⁿ = PDⁿP⁻¹ (easy computation!)

Inner Product Spaces — Key Facts Advanced

  • Gram-Schmidt: convert any basis to orthonormal basis
  • Cauchy-Schwarz: |⟨u,v⟩| ≤ ‖u‖·‖v‖
  • Parseval's identity: ‖f‖² = Σ|⟨f,eₙ⟩|² for orthonormal {eₙ}
  • Normal matrices: AA* = A*A ⟹ unitarily diagonalizable
  • Hermitian A: eigenvalues are real; eigenvectors for distinct eigenvalues are orthogonal

Let T: ℝ⁵ → ℝ³ be a linear transformation with rank(T) = 2. Then nullity(T) is:
A1
B2
C3
D5
✓ Correct Answer: (C) — 3
By Rank-Nullity Theorem: rank(T) + nullity(T) = dim(domain) = 5. So nullity = 5 − 2 = 3. Note: rank(T) ≤ min(5,3) = 3 ✓. The null space is 3-dimensional (solution space of Ax=0 for 5-variable, 3-equation system).

Eigenvalue from Trace/DetMedium
For matrix A = [[3,1],[0,3]], the eigenvalues are:
A3 and 0
B3 and 3 (repeated)
C1 and 3
D2 and 4
✓ Correct Answer: (B) — λ=3 (repeated)
det(A−λI) = (3−λ)(3−λ)−0 = (3−λ)² = 0 ⟹ λ=3 (multiplicity 2). This is a defective matrix (Jordan block): eigenspace = span{[1,0]ᵀ} has dim 1, but algebraic multiplicity = 2. Not diagonalizable over ℝ. Check: trace=6=3+3 ✓, det=9=3×3 ✓.
3
Cayley-HamiltonAdvanced
If the characteristic polynomial of A is p(λ) = λ² − 5λ + 6, then by Cayley-Hamilton:
AA² = 5A − 6I
BA² = 5A + 6I
CA = 5I − 6A²
DA² = 6A − 5I
✓ Correct Answer: (A)
Cayley-Hamilton: p(A) = 0. So A² − 5A + 6I = 0, which gives A² = 5A − 6I. This lets you express all higher powers: A³ = A·A² = A(5A−6I) = 5A²−6A = 5(5A−6I)−6A = 19A−30I, etc. Also, A⁻¹ = (5I − A)/6 (if det ≠ 0 = 6).

DiagonalizabilityAdvanced
A real symmetric matrix:
AMay have complex eigenvalues
BIs always diagonalizable over ℝ with orthogonal eigenvectors
CHas determinant equal to 1
DHas rank equal to its order
✓ Correct Answer: (B)
Spectral Theorem for real symmetric matrices (A=Aᵀ): (1) All eigenvalues are REAL; (2) Eigenvectors for distinct eigenvalues are ORTHOGONAL; (3) A is ORTHOGONALLY DIAGONALIZABLE: A = PDP^T where P is orthogonal (P^T = P⁻¹). This is the heart of principal component analysis (PCA) and has immense CSIR exam weight.

Linear IndependenceMedium
The set {v₁, v₂, v₃} ⊆ ℝ³ is linearly dependent if:
ANone of the vectors is the zero vector
BThe determinant of the matrix with columns v₁,v₂,v₃ is zero
Cv₁·v₂ = 0
Dv₁ + v₂ + v₃ = 0 is the only relation
✓ Correct Answer: (B)
Three vectors in ℝ³ are linearly dependent ⟺ det[v₁|v₂|v₃] = 0 ⟺ they lie in the same plane (coplanar). A is wrong (non-zero vectors can still be dependent). C (orthogonal) doesn't imply dependence. D describes a specific dependent relation, but "only such relation" = {0} would be independent.

Dimension FormulaAdvanced
If U and W are subspaces of a vector space V with dim U = 4, dim W = 3, dim(U+W) = 6, then dim(U∩W) is:
A0
B1
C2
D3
✓ Correct Answer: (B) — 1
By dimension formula: dim(U+W) = dim(U) + dim(W) − dim(U∩W). So 6 = 4 + 3 − dim(U∩W), which gives dim(U∩W) = 1. The intersection is a 1-dimensional subspace (a line through the origin). This formula is the vector space analogue of |A∪B| = |A|+|B|−|A∩B|.

📌 Key Tips — Linear Algebra

  • Trace = sum of eigenvalues, Det = product. For 2×2: char poly = λ²−tr·λ+det.
  • Rank+Nullity = n always. Memorize this formula!
  • Cayley-Hamilton: p(A)=0 → use to compute powers of A and A⁻¹.
  • Symmetric ⟹ always diagonalizable over ℝ with real eigenvalues and orthogonal eigenvectors.
  • Distinct eigenvalues ⟹ LI eigenvectors (sufficient for diagonalizability).

Metric Spaces

Open/Closed Sets · Completeness · Compactness · Connectedness · Baire Category · Contraction Mapping

Foundations

Metric Space Definition Basic

A metric on X is d: X×X → ℝ satisfying:

  1. d(x,y) ≥ 0; d(x,y)=0 ⟺ x=y  (positivity)
  2. d(x,y) = d(y,x)  (symmetry)
  3. d(x,z) ≤ d(x,y) + d(y,z)  (triangle inequality)

Examples: Euclidean d₂, discrete, p-adic, sup metric on C[a,b].

Open & Closed Sets Basic

Open ball: B(x,r) = {y ∈ X : d(x,y) < r}

Open set G: ∀x∈G, ∃r>0: B(x,r) ⊆ G

Closed set F: F = X\G for some open G, OR: every limit point of F is in F.

  • ∅ and X are both open AND closed
  • Arbitrary unions of open sets = open
  • Finite intersections of open sets = open
  • In ℝ: open intervals (a,b) open; [a,b] closed; [a,b) = neither!

Completeness Intermediate

A metric space (X,d) is complete if every Cauchy sequence converges (to a point in X).

  • ℝ and ℂ are complete; ℚ is NOT
  • Every compact metric space is complete
  • C[a,b] with sup norm is complete (Banach space)
  • L²[a,b] (with L² norm) is complete (Hilbert space)
  • (0,1) is NOT complete: 1/n → 0 ∉ (0,1)

Compactness Intermediate

Compact: every open cover has a finite subcover.

In ℝⁿ (Heine-Borel): compact ⟺ closed AND bounded

  • Compact ⟹ complete AND totally bounded
  • Continuous image of compact set is compact
  • Continuous function on compact set is uniformly continuous
  • Compact ⟹ sequentially compact (BW in general metric spaces)
Key Theorems

Banach Contraction Mapping Theorem Advanced

Let (X,d) be a complete metric space. If T:X→X is a contraction (∃k<1: d(Tx,Ty) ≤ k·d(x,y) for all x,y), then:

  • T has a unique fixed point x* (T(x*)=x*)
  • The iterates Tⁿ(x₀) converge to x* for ANY starting x₀

Applications: IVP existence (Picard iteration), Newton's method convergence, computer graphics.

Baire Category Theorem Advanced

A complete metric space is of the second category (cannot be written as countable union of nowhere dense sets).

Equivalently: countable intersection of dense open sets is dense.

  • ℝ is a complete metric space ⟹ second category
  • Proves: continuous nowhere-differentiable functions are "generic"
  • Proves: ℝ cannot be covered by countably many nowhere-dense sets

Connectedness Intermediate

A metric space is connected if it cannot be written as disjoint union of two non-empty open sets.

  • In ℝ: connected ⟺ is an interval
  • Continuous image of connected set is connected (generalizes IVT)
  • Path-connected ⟹ connected (converse fails for topologist's sine curve)
  • ℝ\{0} is NOT connected; ℝ\ℚ is NOT connected

CompletenessMedium
Which of the following metric spaces is NOT complete?
Aℝ with the usual metric
BC[0,1] with the sup norm metric
Cℚ with the usual metric
Dℝ² with the Euclidean metric
✓ Correct Answer: (C) — ℚ
ℚ is NOT complete: the sequence 3, 3.1, 3.14, 3.141, 3.1415, … (decimal approximations of π) is Cauchy in ℚ but converges to π ∉ ℚ. ℝ, ℝ², and C[0,1] with sup norm are all complete (Banach spaces). Completeness of ℝ is equivalent to the Least Upper Bound Property!
2
Compactness in ℝMedium
By the Heine-Borel theorem, a subset K ⊆ ℝⁿ is compact if and only if:
AK is bounded
BK is closed
CK is closed and bounded
DK is connected
✓ Correct Answer: (C) — Closed and bounded
Heine-Borel Theorem (ℝⁿ): compact ⟺ closed AND bounded. This is specific to ℝⁿ — fails in general metric spaces! In infinite-dimensional Banach spaces, closed+bounded does NOT imply compact. Example: unit ball in l² is closed and bounded but NOT compact.
3
Contraction MappingAdvanced
The Banach Contraction Principle states: if T is a contraction on a complete metric space, then:
AT has at least one fixed point
BT has exactly one fixed point
CT has no fixed point
DT may or may not have a fixed point
✓ Correct Answer: (B) — Exactly one fixed point
Uniqueness: if T(x*)=x* and T(y*)=y*, then d(x*,y*)=d(Tx*,Ty*)≤k·d(x*,y*). Since k<1: d(x*,y*)=0 ⟹ x*=y*. Existence: start anywhere x₀, iterate Tⁿ(x₀) → x* by completeness (Cauchy: d(Tⁿx₀,Tᵐx₀) ≤ kᵐ/(1−k)·d(Tx₀,x₀) → 0).

Open SetsBasic
In ℝ with the usual metric, which of the following is an open set?
A[0, 1]
B[0, 1)
C(0, 1)
D{0, 1}
✓ Correct Answer: (C) — (0,1)
An open set: every point has a neighborhood entirely in the set. (0,1): for any x∈(0,1), take r=min(x,1−x)>0; B(x,r)=(x−r,x+r)⊆(0,1) ✓. [0,1]: point 0 has no such r (any ball reaches negative reals). [0,1): same problem at 0. {0,1}: discrete set, not open.

ConnectednessAdvanced
Which of the following subsets of ℝ is connected?
Aℚ (rational numbers)
Bℝ \ {0} (reals minus origin)
C[0,1] ∪ [2,3]
D[0,1] (closed interval)
✓ Correct Answer: (D) — [0,1]
In ℝ: connected ⟺ is an interval. [0,1] is an interval ✓. ℚ = ℚ∩(−∞,√2) ∪ ℚ∩(√2,∞) — disconnected. ℝ\{0} = (−∞,0)∪(0,∞) — two disjoint open sets, disconnected. [0,1]∪[2,3] = two disjoint closed sets — disconnected.

📌 Key Tips — Metric Spaces

  • Complete ≠ Compact: ℝ is complete but not compact. [0,1] is both.
  • Heine-Borel only in ℝⁿ: closed+bounded=compact only in finite-dimensional Euclidean space!
  • Cauchy + complete = convergent. In incomplete spaces, Cauchy sequences can fail to converge.
  • Connected in ℝ = interval. Any set that can be split by a "gap" is disconnected.
  • Banach contraction: needs (1) complete space, (2) k<1. Cannot have k=1.
