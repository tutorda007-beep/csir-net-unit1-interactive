Introduction

Probability theory provides a rigorous mathematical foundation for uncertainty. Starting from Kolmogorov's axioms, we build conditional probability, independence, random variables, and key inequalities that underpin statistics and stochastic processes — all heavily tested in CSIR NET.

Probability Axioms

Kolmogorov Axioms

A probability space (Ω, F, P) where Ω is sample space, F is σ-algebra, P: F→[0,1]:

1. P(A) ≥ 0 for all A ∈ F 2. P(Ω) = 1 3. Countable additivity: A₁,A₂,... disjoint ⟹ P(∪Aᵢ) = ΣP(Aᵢ)
Derived Properties
  • P(Aᶜ) = 1−P(A)
  • P(A∪B) = P(A)+P(B)−P(A∩B) (inclusion-exclusion)
  • A⊆B ⟹ P(A)≤P(B)
  • Boole's inequality: P(∪Aᵢ)≤ΣP(Aᵢ)

Conditional Probability & Bayes' Theorem

Conditional Probability
P(A|B) = P(A∩B)/P(B), P(B)>0

Interpretation: probability of A given B has occurred.

Independence

A and B are independent iff P(A∩B)=P(A)·P(B), equivalently P(A|B)=P(A).

Pairwise independence does NOT imply mutual independence.

Law of Total Probability

If B₁,…,Bₙ partition Ω (disjoint, union=Ω, all positive probability):

P(A) = Σᵢ P(A|Bᵢ)P(Bᵢ)
Bayes' Theorem
P(Bⱼ|A) = P(A|Bⱼ)P(Bⱼ) / Σᵢ P(A|Bᵢ)P(Bᵢ) Posterior ∝ Likelihood × Prior

Random Variables

Random Variable

A measurable function X: Ω→ℝ. CDF: F(x)=P(X≤x).

  • Discrete: PMF p(x)=P(X=x); Σp(x)=1
  • Continuous: PDF f(x)≥0; ∫f(x)dx=1; P(a≤X≤b)=∫_a^b f(x)dx
Joint, Marginal, Conditional Distributions

Joint CDF: F(x,y)=P(X≤x,Y≤y). Marginal: F_X(x)=lim_{y→∞}F(x,y).

f_X(x) = ∫_{-∞}^{∞} f(x,y) dy (marginal PDF) f_{Y|X}(y|x) = f(x,y)/f_X(x) (conditional PDF)

X,Y independent iff f(x,y)=f_X(x)f_Y(y) for all (x,y).

Expectation, Variance & Inequalities

Expectation
E[X] = Σ x·p(x) (discrete) E[X] = ∫ x·f(x)dx (continuous) E[g(X)] = ∫ g(x)f(x)dx (LOTUS)
Variance & Covariance
Var(X) = E[(X−μ)²] = E[X²]−(E[X])² Cov(X,Y) = E[(X−μ_X)(Y−μ_Y)] = E[XY]−E[X]E[Y] ρ(X,Y) = Cov(X,Y)/√(Var(X)·Var(Y)) ∈ [−1,1]
Markov's Inequality
X≥0, a>0: P(X≥a) ≤ E[X]/a
Chebyshev's Inequality
P(|X−μ|≥k·σ) ≤ 1/k² (k>0)

No assumption on distribution; works for any finite mean and variance.

MGF (Moment Generating Function)
M_X(t) = E[e^{tX}] (when finite near t=0) E[Xⁿ] = M_X^{(n)}(0) (nth derivative at t=0) If X,Y independent: M_{X+Y}(t)=M_X(t)·M_Y(t)
