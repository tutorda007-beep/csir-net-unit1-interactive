Introduction

Near a singularity, the Taylor series breaks down. The Laurent series extends the power series to include negative powers of (z−z₀), and the coefficient of (z−z₀)⁻¹ — the residue — is the key to computing complex integrals via the Residue Theorem.

Taylor Series

Taylor's Theorem (Complex)

If f is holomorphic on D(z₀,R), then f has a convergent power series (Taylor series):

f(z) = Σ_{n=0}^∞ aₙ(z−z₀)ⁿ, aₙ = f^(n)(z₀)/n! = (1/2πi)∮ f(z)/(z−z₀)^(n+1) dz

Radius of convergence extends to the nearest singularity of f.

Laurent Series

Laurent Theorem

If f is holomorphic on an annulus A = {z : r < |z−z₀| < R} (0 ≤ r < R ≤ ∞), then f has a unique Laurent expansion:

f(z) = Σ_{n=−∞}^{∞} cₙ(z−z₀)ⁿ = Σ_{n=0}^{∞} aₙ(z−z₀)ⁿ + Σ_{n=1}^{∞} bₙ/(z−z₀)ⁿ

where cₙ = (1/2πi) ∮ f(z)/(z−z₀)^(n+1) dz (any circle in the annulus).

Principal part: the negative-power terms Σ bₙ/(z−z₀)ⁿ. Analytic part: the positive-power terms.

Example: Laurent expansion of 1/(z(z−1)) at z₀=0 for 0<|z|<1

1/(z(z−1)) = −1/z · 1/(1−z) = −(1/z)·Σ_{n=0}^∞ zⁿ = −Σ_{n=0}^∞ zⁿ⁻¹ = −1/z − 1 − z − z² − ...

Principal part: −1/z. Residue at z=0: c₋₁ = −1.

Types of Isolated Singularities

Classification via Laurent Series

If f has an isolated singularity at z₀ with Laurent series Σcₙ(z−z₀)ⁿ (for 0<|z−z₀|<r):

  • Removable singularity: cₙ = 0 for all n < 0. f extends analytically to z₀.
  • Pole of order m: c₋ₘ ≠ 0 and cₙ = 0 for n < −m. The Laurent series has finitely many negative powers (lowest is (z−z₀)⁻ᵐ).
  • Essential singularity: Infinitely many non-zero cₙ with n < 0.

Casorati-Weierstrass Theorem

Near an essential singularity z₀, the image f(D\{z₀}) is dense in ℂ: for any w∈ℂ and ε>0, there are points z arbitrarily close to z₀ with f(z) arbitrarily close to w.

Great Picard Theorem (stronger): Near an essential singularity, f takes every complex value, with at most one exception, infinitely often.

Example: e^(1/z) near z=0

e^(1/z) = Σ_{n=0}^∞ 1/(n! zⁿ) = 1 + 1/z + 1/(2!z²) + ... Essential singularity at z=0 (infinitely many negative powers). The function takes all values except 0 near z=0.

Residues

Residue

The residue of f at z₀ is the coefficient c₋₁ in the Laurent expansion:

Res(f, z₀) = c₋₁ = (1/2πi) ∮_{|z−z₀|=ε} f(z) dz

Computing Residues at Poles

Type of poleResidue formula
Simple pole (order 1)Res(f,z₀) = lim_{z→z₀} (z−z₀)f(z)
Pole of order mRes(f,z₀) = (1/(m−1)!) · lim_{z→z₀} d^(m-1)/dz^(m-1)[(z−z₀)ᵐf(z)]
f=p/q, simple zero of q at z₀Res(f,z₀) = p(z₀)/q'(z₀)
Examples

Simple pole: Res(1/(z²+1), i) = lim_{z→i}(z−i)/(z²+1) = lim_{z→i}(z−i)/((z−i)(z+i)) = 1/(2i).

Order 2 pole: Res(1/z², 0) = coefficient of z⁻¹ in 1/z² = 0 (principal part is 1/z²; c₋₁=0).

Meromorphic Functions

Meromorphic Function

f is meromorphic on a domain Ω if it is holomorphic on Ω except for isolated poles (no essential singularities, no accumulation of poles).

Examples: rational functions p(z)/q(z), tan z, 1/sin z, Γ(z) (Gamma function).

Argument Principle (preview)

If f is meromorphic inside a simple closed curve γ (with no zeros or poles on γ), then:

(1/2πi)∮_γ f'(z)/f(z) dz = Z − P

where Z = number of zeros, P = number of poles (counted with multiplicity) inside γ. This equals the winding number of the image curve f(γ) about 0.
