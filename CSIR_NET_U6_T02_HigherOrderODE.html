Introduction

The second-order (and higher) linear ODE Ly = aₙy^(n) + … + a₁y' + a₀y = f(x) is the backbone of mathematical physics. The general solution is y = yₕ + y_p, where yₕ is the homogeneous solution and y_p is any particular solution.

Solution Structure

Superposition Principle

For L[y] = aₙy^(n)+…+a₀y = 0 (homogeneous, linear), the solution set forms a vector space of dimension n (the order).

For L[y]=f(x), general solution: y = yₕ + y_p where yₕ is general solution of homogeneous and y_p is any particular solution.

Wronskian

For y₁,…,yₙ solutions of the homogeneous equation:

W(y₁,…,yₙ) = det[yᵢ^(j−1)] (i=row, j=col)

They form a fundamental set iff W≠0 on the interval.

Abel's identity: W(x) = W(x₀)·exp(−∫_{x₀}^x aₙ₋₁(t)/aₙ(t) dt).

Reduction of Order

If y₁ is one known solution of Ly=0, set y₂=v(x)y₁ and substitute to reduce to a first-order ODE in v'.

v = ∫ e^(−∫P dx)/y₁² dx (for 2nd order y''+Py'+Qy=0)

Constant Coefficient ODEs

Characteristic Equation

For aₙy^(n)+…+a₁y'+a₀y = 0, substitute y=e^(rx) to get the characteristic polynomial:

aₙrⁿ + aₙ₋₁rⁿ⁻¹ + … + a₁r + a₀ = 0
Root TypeContribution to yₕ
Real distinct re^(rx)
Real repeated r (multiplicity m)e^(rx), xe^(rx), …, x^(m−1)e^(rx)
Complex r = α±βie^(αx)cos(βx), e^(αx)sin(βx)
Complex repeated (multiplicity m)xe^(αx)cos(βx), xe^(αx)sin(βx), …

Method of Undetermined Coefficients

For y_p when f(x) is polynomial, exponential, sin, cos, or their products:

f(x)Trial y_p
Pₙ(x) (degree n poly)Qₙ(x) (degree n poly)
e^(αx)Pₙ(x)e^(αx)Qₙ(x) (if α not a root; else ×x^m)
e^(αx)sin(βx) or cos(βx)e^(αx)(A sin βx + B cos βx)
Modification Rule

If the trial y_p is already a solution of the homogeneous equation (root of multiplicity m), multiply the trial by x^m.

Variation of Parameters

Method (2nd Order)

For y'' + P(x)y' + Q(x)y = R(x) with fundamental solutions y₁, y₂:

y_p = −y₁∫(y₂R/W)dx + y₂∫(y₁R/W)dx

where W = W(y₁,y₂) is the Wronskian.

Derivation Idea

Write y_p = u₁(x)y₁ + u₂(x)y₂. Impose u₁'y₁+u₂'y₂=0 and u₁'y₁'+u₂'y₂'=R. Solve by Cramer's rule: u₁'=−y₂R/W, u₂'=y₁R/W.

Advantage over Undetermined Coefficients

Variation of parameters works for ANY f(x) as long as f is continuous (not just special forms). But requires knowing the homogeneous solutions.

Cauchy-Euler Equation

Cauchy-Euler (Euler-Cauchy) ODE

x²y'' + bxy' + cy = 0 (or more generally xⁿy^(n)+…). Variable coefficients but special form.

Substitute x = e^t (i.e., t = ln x) to convert to constant-coefficient ODE in t.

Or try y = xʳ directly: r(r−1)+br+c = 0 (characteristic equation).

Roots of r(r−1)+br+c=0Solution y
Real distinct r₁≠r₂C₁xʳ¹ + C₂xʳ²
Repeated r₁=r₂=r(C₁+C₂ ln x)xʳ
Complex r = α±βixᵅ[C₁cos(β ln x)+C₂sin(β ln x)]
Example

x²y''−xy'+y=0. Try y=xʳ: r(r−1)−r+1=0 ⟹ r²−2r+1=0 ⟹ (r−1)²=0 ⟹ r=1 (double). Solution: y=(C₁+C₂ ln x)x.

Summary

General Solution

y=yₕ+y_p. yₕ spans n-dim vector space. Fundamental set iff W≠0.

Characteristic Eq.

Constant coeff: substitute y=e^(rx). Roots → solutions (×xᵐ for repeated roots).

Undetermined Coeff.

Trial y_p for poly, exp, trig f(x). Multiply by xᵐ if trial satisfies homogeneous eq.

Variation of Params.

y_p=−y₁∫(y₂R/W)dx+y₂∫(y₁R/W)dx. Works for any continuous R(x).

Cauchy-Euler

x²y''+bxy'+cy=0. Substitute y=xʳ. Roots give xʳ, xʳln x, xᵅcos(β ln x), etc.

Abel's Identity

W(x)=W(x₀)e^(−∫P dt). Either W=0 everywhere or W≠0 everywhere.
