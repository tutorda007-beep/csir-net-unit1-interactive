Combinatorics

Pigeonhole · Inclusion-Exclusion · Permutations · Combinations · Generating Functions · Partitions

📐 Interactive: Inclusion-Exclusion Principle Visualizer

The inclusion-exclusion principle counts elements in unions by adding sets, subtracting pairwise intersections, adding triple intersections, etc.

Core Counting Principles

Basic Counting Basic

Permutations: P(n,r) = n!/(n−r)! Combinations: C(n,r) = n!/(r!(n−r)!) Circular: (n−1)! arrangements in circle With repetition: nʳ functions from r-set to n-set
  • C(n,r) = C(n,n−r) (symmetry)
  • C(n,0)+C(n,1)+…+C(n,n) = 2ⁿ (binary subsets)
  • Σ C(n,r)(-1)ʳ = 0 (alternating sum)
  • Multinomial: n!/(n₁!n₂!…nₖ!) arrangements

Pigeonhole Principle Basic

Basic: If n+1 objects in n boxes, some box has ≥ 2 objects. Generalized: If m objects in n boxes, some box has ≥ ⌈m/n⌉ objects.
  • 5 points in unit square → 2 within distance √2/2
  • Among 13 people: 2 have same birth month
  • Among n+1 integers: 2 are congruent mod n
  • Among 101 integers 1–200: 2 differ by exactly 1

Inclusion-Exclusion Intermediate

|A∪B| = |A|+|B| − |A∩B| |A∪B∪C| = |A|+|B|+|C| −|A∩B|−|B∩C|−|A∩C| +|A∩B∩C| |Ā₁∩Ā₂∩…∩Āₙ| = Σ(-1)^k · S_k

S_k = sum of |intersections of k sets|

Derangements Intermediate

D(n) = n!·Σ_{k=0}^{n} (-1)^k/k! = n!(1 − 1 + 1/2! − 1/3! + … ± 1/n!) Recurrence: D(n) = (n−1)[D(n−1)+D(n−2)] D(1)=0, D(2)=1, D(3)=2, D(4)=9

A derangement is a permutation with no fixed point.

D(n)/n! → 1/e ≈ 0.368 as n→∞ (about 37% chance)

Generating Functions & Recurrences

Binomial & Multinomial Intermediate

Binomial: (x+y)ⁿ = Σ C(n,k) xᵏyⁿ⁻ᵏ Negative: (1−x)⁻ⁿ = Σ C(n+k−1,k) xᵏ Stars&Bars: distribute n identical into k boxes = C(n+k−1, k−1) ways

Pascal's triangle: C(n,k) = C(n−1,k−1) + C(n−1,k)

Vandermonde: Σ C(m,k)C(n,r−k) = C(m+n,r)

Catalan & Stirling Advanced

Catalan: Cₙ = C(2n,n)/(n+1) C₀=1,C₁=1,C₂=2,C₃=5,C₄=14 Counts: Dyck paths, triangulations, balanced parentheses Stirling 2nd kind: S(n,k) = partitions of n-set into k parts S(n,k) = k·S(n−1,k) + S(n−1,k−1)

Bell number: Bₙ = Σₖ S(n,k) = total set partitions

Number Theory

Divisibility · GCD/LCM · Primes · Congruences · Euler φ · CRT · Primitive Roots

🔢 Interactive: Modular Arithmetic & Euler's Theorem

Modular arithmetic: a ≡ b (mod n) means n | (a−b). The set ℤₙ = {0,1,...,n−1} forms a ring under +,× mod n.

Divisibility & Primes

GCD, LCM & Bézout Basic

gcd(a,b) = gcd(b, a mod b) [Euclidean algo] gcd(a,b)·lcm(a,b) = a·b Bézout: gcd(a,b)=d ↔ ∃x,y: ax+by=d
  • Euclid's algorithm: repeated division
  • gcd(0,n) = n for all n
  • If gcd(a,b)=1: a,b coprime (relatively prime)
  • gcd(aⁿ−1, aᵐ−1) = a^(gcd(n,m)) − 1

Euler's Totient Function Basic

φ(n) = n · Π_{p|n} (1 − 1/p) = #{k : 1≤k≤n, gcd(k,n)=1} φ(p) = p−1 (p prime) φ(pᵏ) = pᵏ − pᵏ⁻¹ = pᵏ⁻¹(p−1) φ(mn) = φ(m)φ(n) if gcd(m,n)=1

Values: φ(1)=1, φ(6)=2, φ(12)=4, φ(15)=8

Fermat & Euler Theorems Intermediate

Fermat's Little: aᵖ ≡ a (mod p), p prime If gcd(a,p)=1: aᵖ⁻¹ ≡ 1 (mod p) Euler's: If gcd(a,n)=1: a^φ(n) ≡ 1 (mod n)

Application: To find aⁿ mod m: reduce n mod φ(m) first.

Wilson's theorem: (p−1)! ≡ −1 (mod p) for prime p.

Chinese Remainder Theorem Intermediate

If gcd(mᵢ,mⱼ)=1 for all i≠j, then: x ≡ a₁ (mod m₁) x ≡ a₂ (mod m₂) x ≡ a₃ (mod m₃) has a UNIQUE solution mod m₁m₂m₃.

Method: M=m₁m₂…mₖ, Mᵢ=M/mᵢ, find yᵢ: Mᵢyᵢ≡1 (mod mᵢ). Solution: x ≡ Σ aᵢMᵢyᵢ (mod M).

Group Fundamentals

Group Axioms & Examples Basic

A group (G,·) satisfies: 1. Closure: a·b ∈ G 2. Associativity: (a·b)·c = a·(b·c) 3. Identity: ∃e: a·e = e·a = a 4. Inverse: ∀a, ∃a⁻¹: a·a⁻¹ = e

Examples: (ℤ,+), (ℚ\{0},×), (GLₙ(ℝ),×), (ℤₙ,+), Sₙ

Abelian: also satisfies a·b = b·a

Order: |G| (group), ord(g) = min{n≥1: gⁿ=e}

Lagrange's Theorem Basic

If H is a subgroup of finite group G: |H| divides |G| Number of cosets = |G|/|H| (index [G:H])
  • Order of any element divides |G|
  • Group of prime order p is cyclic ≅ ℤₚ
  • Every element of |G|=p has order 1 or p
  • Converse FALSE: Z₁₂ has no subgroup of order 8

Normal Subgroups & Quotient Intermediate

N ⊴ G (normal) iff gNg⁻¹ = N for all g∈G Equivalently: left cosets = right cosets Quotient group G/N: cosets as elements Operation: (aN)(bN) = (ab)N
  • Every subgroup of an abelian group is normal
  • Kernel of a homomorphism is always normal
  • G/N is abelian iff N⊇[G,G] (commutator subgroup)

Homomorphisms & Isomorphisms Intermediate

φ: G→H is homomorphism if: φ(ab) = φ(a)φ(b) for all a,b ∈ G First Isomorphism Thm: G/ker(φ) ≅ Im(φ) ker(φ) = {g∈G : φ(g)=eₕ} ⊴ G
  • φ injective iff ker(φ) = {e}
  • Cayley's theorem: every group ≅ subgroup of Sₙ
  • Isomorphism: bijective homomorphism
Sylow's Theorems

Sylow's Three Theorems Advanced

Let |G| = pⁿm where p∤m. A Sylow p-subgroup has order pⁿ.

Sylow I: Sylow p-subgroup EXISTS. Sylow II: All Sylow p-subgroups are conjugate. Sylow III: nₚ (number of Sylow p-subgroups) satisfies: (a) nₚ ≡ 1 (mod p) (b) nₚ | m (i.e., nₚ divides the "non-p part")

Application: For |G|=15=3·5: n₃ divides 5 and ≡1(mod 3) → n₃=1. n₅ divides 3 and ≡1(mod 5) → n₅=1. Unique Sylow subgroups → both normal → G ≅ ℤ₁₅.

A3
B8
C24

✓ Correct Answer: (C) — 24
By First Isomorphism Theorem: ℤ₂₄/ker(φ) ≅ Im(φ) ≤ ℤ₁₅. So |Im(φ)| divides |ℤ₁₅|=15 AND |Im(φ)| divides |ℤ₂₄|=24. divisors of gcd(15,24)=3: Im(φ) has order 1 or 3. |ker(φ)| = 24/|Im(φ)|. 📌 CSIR Key Tips — Algebra I

  • Derangements: D(n) = n!·Σ(-1)ᵏ/k!. Memorise D(3)=2, D(4)=9, D(5)=44.
  • Stars&Bars: Non-negative integer solutions x₁+…+xₖ=n is C(n+k−1,k−1).
  • Euler φ(n): n·Π(1−1/p). φ(pq)=(p−1)(q−1) for distinct primes p,q.
  • Sylow III: nₚ ≡ 1 (mod p) AND nₚ | (|G|/pⁿ). Unique Sylow → normal.
  • Order in ℤₙ: ord(a) = n/gcd(a,n). ℤₙ is cyclic; generators are elements with gcd=1.
  • Group of order p or prime power p²: always abelian. Order pq: often cyclic.
  • First Isomorphism Thm: G/ker(φ) ≅ Im(φ). |Im| divides gcd(|G|,|H|).
