Introduction

Connectedness captures the intuitive idea of a space "in one piece". There are multiple notions — topological connectedness (no separation), path-connectedness (any two points can be joined by a path), and local variants. These properties are preserved under continuous maps and are key in topology and analysis.

Connected Spaces

Connected Space

X is connected if it cannot be written as a disjoint union of two non-empty open sets U and V (a "separation").

Equivalently: X is connected iff the only subsets that are both open and closed (clopen) are ∅ and X.

Disconnected / Separation

X is disconnected if X = U ∪ V where U, V are non-empty, open, and disjoint. Such {U,V} is called a separation of X.

Connected Subsets of ℝ

A subset A ⊆ ℝ is connected iff A is an interval (including rays and ℝ itself).

Continuous Image

The continuous image of a connected space is connected. If f: X→Y is continuous and X is connected, then f(X) is connected.

Path Connectedness

Path Connected

X is path connected if for any x,y ∈ X, there exists a continuous γ: [0,1]→X with γ(0)=x and γ(1)=y.

Path connected ⟹ connected. The converse fails in general.

Topologist's Sine Curve

S = {(x, sin(1/x)) : x>0} ∪ {(0,y) : −1≤y≤1} is connected but NOT path connected. Any path from (0,0) to a point on the sine curve must pass through infinitely many oscillations.

Convex ⟹ Path Connected ⟹ Connected

In a normed space: convex ⟹ path connected (straight-line path γ(t)=(1−t)x+ty). Path connected ⟹ connected. Neither converse holds in general.

For Open Subsets of ℝⁿ

An open subset U ⊆ ℝⁿ is connected iff it is path connected.

Connected Components

Connected Component

The connected component of x ∈ X is the largest connected subset of X containing x. Components are always closed. They partition X into disjoint closed sets.

Totally Disconnected

X is totally disconnected if every connected component consists of a single point. Examples: ℚ, ℤ, Cantor set.

Locally Connected

X is locally connected if every point has a neighborhood basis of connected open sets. ℝⁿ is locally connected. ℚ is NOT locally connected.

Path Components

Similarly, path components partition X. For manifolds: connected ↔ path connected ↔ path component = whole space.

Intermediate Value Theorem

Intermediate Value Theorem (General Form)

If f: X→ℝ is continuous and X is connected, then f(X) is an interval. In particular: if f(a)

Classical form: f: [a,b]→ℝ continuous, f(a) and f(b) have opposite signs ⟹ ∃c∈(a,b) with f(c)=0.

Proof Sketch

f([a,b]) is a continuous image of a connected set [a,b], hence connected in ℝ, hence an interval. Since it contains f(a) and f(b), it contains all values between them.

Fixed Point (Brouwer 1D)

If f: [0,1]→[0,1] is continuous, then f has a fixed point (∃x: f(x)=x).

Proof: apply IVT to g(x)=f(x)−x: g(0)≥0 and g(1)≤0.

Applications
  • Root finding: Bisection method.
  • Existence proofs: ODE solutions, Nash equilibria.
  • Connected components: IVT shows ℝ\{point} is disconnected (⟹ ℝ is not homeomorphic to ℝⁿ for n≥2).

Summary

Connected

No separation into two open disjoint non-empty sets. Equivalently: only clopen sets are ∅ and X.

ℝ Subsets

Connected iff an interval. Path connected iff connected for open subsets of ℝⁿ.

Path Connected

Any two points joined by a path. Path connected ⟹ connected. Converse fails (topologist's sine curve).

Components

Maximal connected subsets. Always closed. Totally disconnected: each component = {point}.

IVT

f continuous, X connected ⟹ f(X) is connected (interval in ℝ). Generalizes classical IVT.

Brouwer 1D

f: [0,1]→[0,1] continuous ⟹ ∃ fixed point. Proof via IVT on g(x)=f(x)−x.
