Set Theory

Countability · Cardinality · Cantor Set · Power Sets · Operations · Schroder-Bernstein

📊 Interactive: Set Operations Visualizer

Basic Concepts

Sets & Notation Basic

A set is a well-defined collection of distinct objects. Notation: A = {1, 2, 3} or A = {x : property}.

  • x ∈ A — x is a member of A
  • A ⊆ B — A is a subset of B (every element of A is in B)
  • A ⊂ B — proper subset (A ⊆ B and A ≠ B)
  • ∅ — empty set (∅ ⊆ A for any A)
  • ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ ⊂ ℂ

Set Operations Basic

  • Union: A ∪ B = {x : x ∈ A or x ∈ B}
  • Intersection: A ∩ B = {x : x ∈ A and x ∈ B}
  • Complement: Aᶜ = {x ∈ U : x ∉ A}
  • Difference: A \ B = {x : x ∈ A, x ∉ B}
  • Symmetric diff: A △ B = (A\B) ∪ (B\A)
De Morgan: (A∪B)ᶜ = Aᶜ∩Bᶜ  |  (A∩B)ᶜ = Aᶜ∪Bᶜ

Power Set Basic

𝒫(A) = collection of ALL subsets of A (including ∅ and A itself).

If |A| = n → |𝒫(A)| = 2ⁿ

Example: A = {1,2}: 𝒫(A) = {∅, {1}, {2}, {1,2}} → |𝒫(A)| = 4 = 2²

Key: |𝒫(ℕ)| = cardinality of ℝ (uncountable).

Countable vs Uncountable Intermediate

A set is countable if it is finite OR has a bijection with ℕ.

  • Countable: ℕ, ℤ, ℚ, ℕ×ℕ, all finite subsets of ℕ
  • Uncountable: ℝ, (0,1), 𝒫(ℕ), ℝ\ℚ
  • Countable ∪ countable = countable
  • Countable ∩ (anything) = at most countable
Key Theorems

Cantor's Diagonal Argument Advanced

Theorem: ℝ is uncountable.

Proof idea: Assume (0,1) is countable — list all reals as x₁, x₂, x₃, … Define y by choosing the n-th decimal digit of y to differ from the n-th decimal digit of xₙ. Then y ∈ (0,1) but y ≠ xₙ for all n. Contradiction!

Cantor Set Properties Advanced

The middle-thirds Cantor set C is constructed by repeatedly removing the open middle third of each remaining interval from [0,1].

  • Closed & bounded in [0,1]
  • Nowhere dense — contains NO open interval
  • Uncountable — has cardinality of ℝ = 2^ℵ₀
  • Perfect — every point is a limit point
  • Lebesgue measure = 0 (a "large" yet "thin" set)

Schroder-Bernstein Theorem Advanced

If injections f:A→B and g:B→A both exist, then there is a bijection h:A→B, so |A| = |B|.

Used to prove: |(0,1)| = |ℝ| = |𝒫(ℕ)|

Countability of Sets Medium
Which of the following sets is UNCOUNTABLE?
AThe set of all integers ℤ
BThe set of all rational numbers ℚ
CThe set of all real numbers ℝ
DThe set of all algebraic numbers
✓ Correct Answer: (C) — The set of all real numbers ℝ
By Cantor's diagonal argument, ℝ is uncountable. ℤ and ℚ are countably infinite (bijection with ℕ). Algebraic numbers are countable (countable union of finite sets — roots of each integer polynomial). Only ℝ is uncountable among these.
2
Power Set Cardinality Basic
If a set A has n elements, the number of elements in its power set 𝒫(A) is:
An
B2n
C2ⁿ
Dn!
✓ Correct Answer: (C) — 2ⁿ
𝒫(A) contains ALL subsets of A including ∅ and A itself. For each element, independently include it (2 choices per element), so total = 2ⁿ. Example: |A|=3 → |𝒫(A)|=8. For n=0: 𝒫(∅)={∅}, so |𝒫(∅)|=1=2⁰. ✓
3
Cantor Set Advanced
The Cantor set C (middle-thirds construction) is:
ACountable and nowhere dense
BCountable and dense in [0,1]
CUncountable and nowhere dense
DUncountable and dense in [0,1]
✓ Correct Answer: (C) — Uncountable and nowhere dense
The Cantor set has cardinality of ℝ (uncountable) yet Lebesgue measure = 0. It is nowhere dense (contains no open interval) and its complement in [0,1] is open and dense. It is NOT dense in [0,1] (D is wrong). It is a perfect set (every point is a limit point).

Set Operations Basic
If A ⊂ B (A is a proper subset of B), then A ∪ B equals:
AA
BB
CA ∩ B
DA \ B
✓ Correct Answer: (B) — B
Since A ⊂ B, every element of A is already in B. So A ∪ B = B (no new elements added). Also A ∩ B = A when A ⊂ B. Standard identity: if A ⊆ B then A ∪ B = B and A ∩ B = A.
5
Countability of Subsets Advanced
Which statement about countable sets is CORRECT?
AThe union of countably many countable sets is uncountable
BThe Cartesian product ℕ × ℕ is uncountable
CThe set of all finite subsets of ℕ is countable
DThe set of all infinite subsets of ℕ is countable
✓ Correct Answer: (C) — Finite subsets of ℕ is countable
Finite subsets of ℕ: for each n, subsets of size n are countable; countable union of countable sets is countable. A is FALSE: countable union of countable = countable. B is FALSE: ℕ×ℕ ≅ ℕ (countable). D is FALSE: infinite subsets of ℕ are in bijection with 𝒫(ℕ) = uncountable.

📌 Key Tips — Set Theory

  • Countable ≠ Finite: ℤ and ℚ are infinite but countable.
  • Cantor set trap: uncountable yet measure zero — seemingly contradictory. Know it!
  • Power set: |𝒫(A)| = 2ⁿ. Also |𝒫(ℕ)| = |ℝ| = 2^ℵ₀.
  • De Morgan's laws appear in probability and topology too — memorize them.

Real Number System

Field Axioms · Order · Absolute Value · Supremum · Infimum · Completeness · Archimedean Property

📊 Interactive: Supremum & Infimum on Number Line

S = {1−1/n : n∈ℕ} = {0, 1/2, 2/3, 3/4, …} → sup=1 (not in S), inf=0 (not in S, min=0 exists).

Basic Concepts

Field & Order Axioms Basic

ℝ is an ordered field: satisfies commutativity, associativity, distributivity, identity and inverse elements for both + and ×, plus order (trichotomy).

  • If a < b and c > 0, then ac < bc
  • If a < b and c < 0, then ac > bc (flips!)
  • ℤ is not a field (no 1/2 ∈ ℤ)
  • ℚ is a field but not complete

Absolute Value Basic

|x| = x if x ≥ 0,   −x if x < 0
  • Triangle Inequality: |x + y| ≤ |x| + |y|
  • Reverse: | |x| − |y| | ≤ |x − y|
  • |xy| = |x||y|
  • |x| < r ⟺ −r < x < r (ε-δ proofs!)

Supremum & Infimum Intermediate

sup S = least upper bound (LUB) = smallest M with s ≤ M for all s ∈ S.

inf S = greatest lower bound (GLB) = largest m with m ≤ s for all s ∈ S.

  • sup S ∈ S ⟺ S has a maximum
  • inf S ∈ S ⟺ S has a minimum
  • inf(−S) = −sup(S)
sup{1 − 1/n : n∈ℕ} = 1  (but 1 ∉ S)

Bounded Sets Basic

S ⊆ ℝ is bounded above if ∃M ∈ ℝ: s ≤ M for all s ∈ S.

S is bounded below if ∃m ∈ ℝ: m ≤ s for all s ∈ S.

S is bounded if bounded both above and below.

  • (0,1) is bounded above by 1, below by 0
  • ℕ is bounded below by 1, but NOT above
  • ℤ is neither bounded above nor below
Key Theorems

Completeness Axiom (LUB Property) Advanced

Every non-empty subset S of ℝ that is bounded above has a supremum in ℝ.

S ≠ ∅, S bounded above ⟹ sup S ∈ ℝ exists

This DEFINES ℝ and distinguishes it from ℚ! In ℚ: S = {x ∈ ℚ : x² < 2} is bounded above, but sup S = √2 ∉ ℚ — the LUB does not exist in ℚ.

Archimedean Property Intermediate

For every real x > 0, there exists n ∈ ℕ such that n > x.

Equivalently: for every ε > 0, ∃n ∈ ℕ with 1/n < ε.

This means ℕ is unbounded above in ℝ. Follows from Completeness Axiom. Used in virtually every ε-δ proof.

Density Theorem Intermediate

Between any two distinct real numbers a < b, there exists BOTH a rational q ∈ ℚ AND an irrational r ∈ ℝ\ℚ with a < q < b and a < r < b.

In fact, infinitely many of each. So ℚ is dense in ℝ, and ℝ\ℚ is also dense in ℝ.

Nested Intervals Theorem Advanced

If [a₁,b₁] ⊇ [a₂,b₂] ⊇ [a₃,b₃] ⊇ … is a nested sequence of closed bounded intervals with (bₙ − aₙ) → 0, then ∩[aₙ,bₙ] contains exactly one point.

Fails for open intervals: ∩(0, 1/n) = ∅.

Archimedean Property Basic
The Archimedean property of ℝ states that for any real x > 0, there exists n ∈ ℕ such that:
An > x
Bn < x
Cn = x²
Dn · x < 1
✓ Correct Answer: (A) — n > x
The Archimedean property says ℕ is unbounded above: for every x > 0, ∃n ∈ ℕ with n > x. Equivalent: for every ε > 0, ∃n with 1/n < ε. This is used in virtually every epsilon-delta proof to bound denominators.
2
Supremum Medium
The supremum of S = {1 − 1/n : n ∈ ℕ} is:
A0
BDoes not exist
C1
D2
✓ Correct Answer: (C) — 1
S = {0, 1/2, 2/3, 3/4, …}. Each element < 1, so 1 is an upper bound. For any ε > 0, choose n > 1/ε; then 1 − 1/n > 1 − ε, so no number less than 1 is an upper bound. Hence sup S = 1. Note: 1 ∉ S, so the maximum does not exist.
3
Infimum Basic
The infimum of S = {1/n : n ∈ ℕ} is:
A1
B0
CDoes not exist
D−∞
✓ Correct Answer: (B) — 0
Since 1/n > 0 for all n, 0 is a lower bound. For any ε > 0, by Archimedean property ∃n with 1/n < ε, so no positive number is a lower bound. Hence inf S = 0. Note: 0 ∉ S — the minimum does not exist.
4
Completeness Axiom Medium
A non-empty subset S of ℝ has a supremum if and only if:
AS is finite
BS is bounded above
CS is closed
DS contains its least upper bound
✓ Correct Answer: (B) — S is bounded above
Completeness Axiom: every non-empty S ⊆ ℝ bounded above has a supremum in ℝ. The sup need NOT belong to S (e.g., sup{1/n} = 0 ∉ S). This property fails in ℚ — it is the DEFINING difference between ℝ and ℚ.
5
Density Theorem Medium
Between any two distinct real numbers a < b, there exists:
AOnly rational numbers
BOnly irrational numbers
CNo number at all
DBoth a rational and an irrational number
✓ Correct Answer: (D) — Both a rational and an irrational
Density Theorem: between any a < b exist infinitely many rationals AND infinitely many irrationals. For rationals: use Archimedean to find n with 1/n < b−a, then find m ∈ ℤ with m/n ∈ (a,b). For irrationals: a + (b−a)/√2 type construction works.

📌 Key Tips — Real Numbers

  • ℝ vs ℚ: The key difference is completeness (LUB). ℚ has all field and order axioms but lacks completeness.
  • sup ≠ max: sup S always exists if S is bounded above. max S exists only if sup S ∈ S.
  • Archimedean: "ℕ is unbounded" and "1/n → 0" — two equivalent powerful statements used in every ε-δ proof.
  • CSIR classic: Is √2 ∈ ℚ? No — it is the prototypical irrational number, showing ℚ is incomplete.
