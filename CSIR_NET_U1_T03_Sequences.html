🎯
CSIR NET Frequency: Sequences and series are the most heavily tested topic in Unit 1 — expect 4–6 questions per exam. Limsup/liminf, Cauchy sequences, and the comparison test are especially common in Part C (4.75-mark questions).

🔢 Sequences — Convergence

→
Concept 1
Definition of Convergence (ε-N)
Formal Definition
A sequence (aₙ) converges to L (written aₙ → L or lim aₙ = L) if:

∀ ε > 0, ∃ N ∈ ℕ such that n ≥ N ⟹ |aₙ − L| < ε

In words: the terms eventually get arbitrarily close to L.
🔑
How to use ε-N: Given ε > 0, find N (usually in terms of ε) such that for n ≥ N the inequality |aₙ − L| < ε holds. The critical skill is working BACKWARDS: start from |aₙ − L| < ε and solve for n in terms of ε to find N.

Properties of Convergent Sequences

  • Uniqueness: If aₙ → L and aₙ → M then L = M
  • Boundedness: Every convergent sequence is bounded
  • Squeeze theorem: If aₙ ≤ bₙ ≤ cₙ and aₙ → L and cₙ → L, then bₙ → L
  • Algebra of limits: lim(aₙ + bₙ) = lim aₙ + lim bₙ; lim(aₙ · bₙ) = (lim aₙ)(lim bₙ)
  • Order preservation: If aₙ → L and bₙ → M and aₙ ≤ bₙ, then L ≤ M

Monotone Convergence Theorem

Monotone Convergence Theorem (MCT)
Every bounded monotone sequence converges.
• Bounded increasing → converges to sup{aₙ}
• Bounded decreasing → converges to inf{aₙ}
This is equivalent to the Completeness Axiom. The classic example: define a₁ = 1, aₙ₊₁ = aₙ/2 + 1. Show it's bounded and increasing to find limit L = 2.

🔼🔽 limsup & liminf

∞
Concept 2
lim sup and lim inf
Definitions
lim sup aₙ = lim_{n→∞} sup{aₖ : k ≥ n} = inf_{n} sup{aₖ : k ≥ n}

lim inf aₙ = lim_{n→∞} inf{aₖ : k ≥ n} = sup_{n} inf{aₖ : k ≥ n}

Let Mₙ = sup{aₖ : k ≥ n} (tail supremum). Then Mₙ is decreasing, so lim sup = lim Mₙ.
Let mₙ = inf{aₖ : k ≥ n} (tail infimum). Then mₙ is increasing, so lim inf = lim mₙ.
Key Properties
1. lim inf aₙ ≤ lim sup aₙ always
2. aₙ → L ⟺ lim inf aₙ = lim sup aₙ = L
3. lim sup(aₙ + bₙ) ≤ lim sup aₙ + lim sup bₙ
4. If aₙ → L then lim sup aₙ = lim inf aₙ = L
5. lim sup(−aₙ) = −lim inf(aₙ)
🧠
Intuition: lim sup is the largest cluster point of the sequence; lim inf is the smallest. For (−1)ⁿ: lim sup = 1 (the sequence reaches +1 infinitely often), lim inf = −1 (reaches −1 infinitely often). Since they differ, the sequence diverges.

🔄 Cauchy Sequences

≈
Concept 3
Cauchy Sequences
Definition
A sequence (aₙ) is Cauchy if the terms become arbitrarily close to each other:

∀ ε > 0, ∃ N ∈ ℕ such that m, n ≥ N ⟹ |aₙ − aₘ| < ε

Key: no reference to a limit L — we compare terms with each other.
Completeness Theorem for ℝ
In ℝ: A sequence converges ⟺ it is a Cauchy sequence.
→ (Convergent ⟹ Cauchy): If aₙ → L, then |aₙ − aₘ| ≤ |aₙ − L| + |L − aₘ| < ε/2 + ε/2 = ε for n,m ≥ N.
← (Cauchy ⟹ Convergent): Use BWT to extract a convergent subsequence, then show the full sequence converges to the same limit. This direction requires completeness of ℝ — it FAILS in ℚ!
⚡
Cauchy criterion for series: ∑aₙ converges iff for every ε > 0, ∃N such that |aₙ₊₁ + aₙ₊₂ + … + aₘ| < ε for all m > n ≥ N. This is the basis for many convergence tests.

∑ Series — Convergence Tests

Σ
Concept 4
Series Definitions

A series ∑_{n=1}^∞ aₙ is the limit of partial sums: Sₙ = a₁ + a₂ + … + aₙ. The series converges iff (Sₙ) converges.

Necessary Condition for Convergence
If ∑aₙ converges, then aₙ → 0. The converse is FALSE: ∑1/n diverges even though 1/n → 0.
Absolute vs Conditional Convergence
Absolutely convergent: ∑|aₙ| converges
Conditionally convergent: ∑aₙ converges but ∑|aₙ| diverges
Fact: Absolute convergence ⟹ convergence (but not vice versa)

Convergence Tests — Quick Reference

Comparison Test
0 ≤ aₙ ≤ bₙ
∑bₙ conv. ⟹ ∑aₙ conv.
∑aₙ div. ⟹ ∑bₙ div.
Limit Comparison Test
L = lim(aₙ/bₙ) ∈ (0,∞)
∑aₙ and ∑bₙ both converge or both diverge
Ratio Test (D'Alembert)
L = lim |aₙ₊₁/aₙ|
L < 1 → conv. | L > 1 → div. | L = 1 → inconclusive
Root Test (Cauchy)
L = lim sup |aₙ|^(1/n)
L < 1 → conv. | L > 1 → div. | L = 1 → inconclusive
Integral Test
f positive, decreasing
∑f(n) converges ⟺ ∫₁^∞ f(x)dx converges
Alternating Series (Leibniz)
∑(−1)ⁿ bₙ, bₙ↘0
Converges if bₙ ↓ 0 (monotone decreasing to 0)
p-Series Test
∑1/nᵖ
Converges if p > 1, Diverges if p ≤ 1
Geometric Series
∑rⁿ
Converges to 1/(1−r) if |r| < 1, Diverges if |r| ≥ 1
⚠️
Test Hierarchy for CSIR NET: When L = 1 for ratio/root test, they're inconclusive — don't stop! Use p-series test or comparison. Root test is STRONGER than ratio test (ratio test inconclusive ⟹ root test also inconclusive, but NOT vice versa).

Riemann Rearrangement Theorem
If ∑aₙ is conditionally convergent (converges but not absolutely), then for any real number r (or ±∞), there exists a rearrangement of the series that sums to r.
This is why order matters for non-absolutely convergent series! For absolutely convergent series, any rearrangement gives the same sum (Dirichlet's theorem). This theorem is frequently tested in CSIR NET.

📜 Key Theorems

Theorem 1 — Bolzano-Weierstrass for Sequences
Every bounded sequence in ℝ has a convergent subsequence.
Proved using the Nested Interval Theorem: repeatedly bisect intervals containing infinitely many terms. The subsequence converges to the "trapped" limit point. (Full proof in T04.)
Theorem 2 — Subsequential Limits and limsup
lim sup aₙ = sup{L : L is a subsequential limit of (aₙ)}.
Similarly, lim inf aₙ = inf{L : L is a subsequential limit of (aₙ)}.
The limsup is the largest limit of any convergent subsequence. For (−1)ⁿ: subsequential limits are {−1, 1}, so limsup = 1, liminf = −1.
Theorem 3 — Cauchy Condensation Test
For decreasing aₙ ≥ 0: ∑aₙ converges iff ∑2ⁿa_{2ⁿ} converges.
Application: ∑1/nᵖ converges iff ∑2ⁿ/(2ⁿᵖ) = ∑2^{n(1−p)} converges iff p > 1. This gives the p-series test.
Theorem 4 — Abel's Test
If ∑aₙ converges and (bₙ) is bounded and monotone, then ∑aₙbₙ converges.
Used when neither ratio nor root test applies — powerful for series like ∑(sin n)/n² or ∑aₙ·rⁿ.
Theorem 5 — Dirichlet's Test
If partial sums of ∑aₙ are bounded and bₙ → 0 monotonically, then ∑aₙbₙ converges.
Application: ∑(sin nθ)/n converges for θ ≠ 2kπ. The partial sums of ∑sin nθ are bounded, and 1/n → 0 monotonically.

✏️ Worked Examples

Example 1 — ε-N Proof
Prove from definition that lim_{n→∞} (3n+1)/(2n−1) = 3/2.
Setup: Let L = 3/2. We need |aₙ − 3/2| < ε.
Computation: |(3n+1)/(2n−1) − 3/2| = |(2(3n+1) − 3(2n−1))/(2(2n−1))| = |6n+2−6n+3|/(2|2n−1|) = 5/(2(2n−1)) for n ≥ 1.
Find N: We need 5/(2(2n−1)) < ε ⟺ 2n−1 > 5/(2ε) ⟺ n > 5/(4ε) + 1/2.
Choose N = ⌈5/(4ε) + 1⌉. Then for n ≥ N: |(3n+1)/(2n−1) − 3/2| = 5/(2(2n−1)) < ε. □
Example 2 — Computing limsup and liminf
Find lim sup and lim inf for aₙ = sin(nπ/2).
Values: a₁ = 1, a₂ = 0, a₃ = −1, a₄ = 0, a₅ = 1, … (period 4)
Tail supremum: Mₙ = sup{sin(kπ/2) : k ≥ n}. Since the sequence repeats, every tail contains the value 1. So Mₙ = 1 for all n → lim sup = 1.
Tail infimum: Similarly mₙ = −1 for all n → lim inf = −1.
Since lim sup ≠ lim inf, the sequence diverges. Subsequential limits: {−1, 0, 1}.
Example 3 — Ratio Test
Test ∑_{n=1}^∞ n²·3ⁿ/n! for convergence.
Ratio test: |a_{n+1}/aₙ| = [(n+1)²·3^{n+1}/(n+1)!] / [n²·3ⁿ/n!]
= (n+1)²·3 / [(n+1)·n²] = 3(n+1)/n²
As n → ∞: 3(n+1)/n² → 0 < 1.
Conclusion: By the ratio test, the series converges absolutely.
Example 4 — Alternating Series
Does ∑_{n=1}^∞ (−1)^{n+1}/√n converge absolutely or conditionally?
Absolute convergence: ∑1/√n = ∑1/n^{1/2} is a p-series with p = 1/2 ≤ 1 → diverges. Not absolutely convergent.
Conditional convergence: Apply Leibniz test: bₙ = 1/√n. Is it decreasing? Yes: 1/√(n+1) < 1/√n. Does bₙ → 0? Yes: 1/√n → 0. By the Alternating Series Test, the series converges conditionally.
Example 5 — Cauchy Sequence
Show that aₙ = 1 + 1/2 + 1/4 + … + 1/2^n is Cauchy.
Observation: This is a partial sum of the geometric series ∑(1/2)ⁿ with r = 1/2, which converges. So aₙ → L = 2. Since it converges in ℝ, it must be Cauchy.
Direct proof: For m > n, |aₘ − aₙ| = |1/2^{n+1} + … + 1/2^m| ≤ ∑_{k=n+1}^∞ 1/2^k = 1/2^n → 0. For any ε > 0, choose N with 1/2^N < ε. Then for m > n ≥ N: |aₘ − aₙ| < ε. □

Question 2 of 15
The series ∑_{n=1}^∞ 1/nᵖ converges if and only if:
✓ B is correct. The p-series ∑1/nᵖ converges iff p > 1. At p = 1 it's the harmonic series (diverges). At p = 2, sum = π²/6. At p = 1/2, it diverges. Proved via the integral test: ∫₁^∞ x^{-p}dx converges iff p > 1.
Question 3 of 15
For the sequence aₙ = (−1)ⁿ, which is TRUE?
✓ C is correct. aₙ alternates between +1 (even n) and −1 (odd n). Tail supremum = 1 for all n (each tail contains the value +1), so lim sup = 1. Tail infimum = −1 for all n, so lim inf = −1. Since lim sup ≠ lim inf, the sequence diverges.
Question 4 of 15
In ℝ, a sequence converges if and only if it is:
✓ C is correct. In ℝ, convergence ⟺ Cauchy (this is what "completeness" means). A is wrong: (−1)ⁿ is bounded but diverges. B is wrong: an increasing sequence might diverge to +∞. D is correct only in one direction: monotone + bounded ⟹ convergent (MCT). But convergent doesn't imply monotone. Cauchy is the EXACT characterization.
Question 5 of 15
Applying the ratio test to ∑n/2ⁿ gives:
✓ A is correct. |a_{n+1}/aₙ| = [(n+1)/2^{n+1}] / [n/2ⁿ] = (n+1)/(2n) → 1/2 as n→∞. Since L = 1/2 < 1, the series converges (absolutely). Sum = ∑n/2ⁿ = 2.
Question 6 of 15
Which of the following series is conditionally convergent but NOT absolutely convergent?
✓ B is correct. ∑(−1)ⁿ/n converges by Leibniz test (1/n → 0 monotonically), but ∑1/n diverges (harmonic series) → conditionally convergent. A: ∑1/n² converges absolutely (p=2>1). C: ∑(−1)ⁿ/n² also absolutely convergent (∑1/n² converges). D: ∑1/n diverges entirely.
Question 7 of 15
The necessary condition for ∑aₙ to converge is:
✓ A is correct. If ∑aₙ converges, then aₙ → 0 (necessary condition). The converse fails: ∑1/n diverges even though 1/n → 0. C is SUFFICIENT (absolute convergence implies convergence) but not necessary. B is sufficient for alternating series but not necessary in general. D is neither necessary nor sufficient.
Question 8 of 15
Let aₙ = (1 + 1/n)ⁿ. What is lim_{n→∞} aₙ?
✓ C is correct. By definition, e = lim_{n→∞}(1 + 1/n)ⁿ ≈ 2.71828. This is one of the fundamental limits in analysis. Verify: (1+1/n)ⁿ is increasing and bounded above by 3, so by MCT it converges. The limit is defined to be e.
Question 9 of 15
By the Riemann Rearrangement Theorem, if ∑aₙ converges conditionally, its terms can be rearranged to:
✓ C is correct. Riemann's theorem: if ∑aₙ converges conditionally, for ANY target T ∈ ℝ ∪ {±∞}, there exists a rearrangement summing to T. This is remarkable — by reordering, we can get any sum. Absolutely convergent series are immune: any rearrangement gives the same sum (Dirichlet).
Question 10 of 15
Which test is MOST POWERFUL among the following?
✓ B is correct. The root test is at least as powerful as the ratio test: wherever the ratio test gives a conclusion, so does the root test. But there exist series where the ratio test is inconclusive (L = 1) yet the root test gives L < 1 or L > 1. The root test uses lim sup, which always exists (unlike lim).
Question 11 of 15
The sequence defined by a₁ = √2, aₙ₊₁ = √(2 + aₙ). The sequence converges to:
✓ C is correct. If L = lim aₙ exists, then L = √(2 + L) → L² = 2 + L → L² − L − 2 = 0 → (L−2)(L+1) = 0 → L = 2 (since L > 0). By MCT: (aₙ) is bounded above by 2 (induction: if aₙ < 2 then aₙ₊₁ = √(2+aₙ) < √4 = 2) and increasing. So it converges to 2.
Question 12 of 15
For a sequence (aₙ), which of the following implies convergence? (CSIR NET style)
✓ C is correct. If every subsequence has a sub-subsequence converging to L, then aₙ → L. Proof: If aₙ did not converge to L, there would be a subsequence bounded away from L, contradicting the hypothesis. A and B don't imply convergence ((−1)ⁿ is bounded but diverges). D only says a subsequence converges, not the full sequence.
Question 13 of 15
The series ∑_{n=2}^∞ 1/(n·(ln n)ᵖ) converges when:
✓ B is correct. By the integral test: ∫₂^∞ 1/(x(ln x)ᵖ)dx. Substitute u = ln x: ∫_{ln 2}^∞ u^{-p}du, which converges iff p > 1. So ∑1/(n(ln n)ᵖ) converges iff p > 1. At p=1: ∑1/(n ln n) diverges. At p=2: converges.
Question 14 of 15
A Cauchy sequence in ℚ that does NOT converge in ℚ:
✓ B is correct. ℚ is NOT complete: there exist Cauchy sequences in ℚ that don't converge in ℚ. Example: truncated decimal expansions of √2: 1, 1.4, 1.41, 1.414, … is Cauchy in ℚ but converges to √2 ∉ ℚ. This is exactly why ℝ was constructed: to "complete" ℚ.
Question 15 of 15
If ∑aₙ converges absolutely and (bₙ) is bounded, does ∑aₙbₙ converge?
✓ A is correct. If ∑|aₙ| converges and |bₙ| ≤ M for all n, then ∑|aₙbₙ| ≤ M·∑|aₙ| < ∞. So ∑aₙbₙ converges absolutely. This is a direct comparison. Note: this doesn't require bₙ → 0; any bounded sequence works.

📌 Summary & CSIR NET Tips

Sequence Essentials

  • ε-N definition: ∀ε>0, ∃N: n≥N ⟹ |aₙ−L|<ε
  • Convergent ⟹ bounded (converse fails: (−1)ⁿ)
  • MCT: bounded monotone ⟹ converges
  • Cauchy ⟺ Convergent (in ℝ)
  • limsup = liminf ⟺ convergent

Series Key Facts

  • ∑aₙ conv. ⟹ aₙ → 0 (necessary, not sufficient)
  • ∑1/nᵖ: p>1 conv., p≤1 div.
  • Abs. conv. ⟹ convergent (conv. ⇏ abs. conv.)
  • Root test ≥ Ratio test in power
  • Rearrangement OK for absolutely conv. series

CSIR Exam Q-Types

  • Compute lim sup / lim inf for given sequences
  • Determine convergence using p-series, ratio, root
  • Absolute vs conditional convergence identification
  • ε-N proofs for simple sequences
  • MCT applications: recursive sequences

Common Mistakes

  • aₙ → 0 does NOT imply ∑aₙ converges (∑1/n!)
  • Ratio test L=1 is INCONCLUSIVE (try another test)
  • Bounded sequence may diverge ((−1)ⁿ)
  • Convergent subseq. ≠ convergent sequence
  • Rearrangement changes sum if only conditionally conv.
