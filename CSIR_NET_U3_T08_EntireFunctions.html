Introduction

Entire functions (holomorphic on all of ℂ) include polynomials, eᶻ, sin z, cos z. Their global behavior is far richer than polynomials — they can have infinitely many zeros. The theory of entire functions culminates in Picard's theorems, which are among the deepest results in complex analysis.

Entire Functions

Definition

An entire function is a function holomorphic on all of ℂ. Every entire function has a Taylor series Σaₙzⁿ converging everywhere.

Order of Growth

Order of an Entire Function

The order ρ of entire function f is defined by:

ρ = lim sup_{r→∞} log log M(r) / log r

where M(r) = max_{|z|=r} |f(z)|.

Equivalently, |f(z)| ≤ C·exp(|z|^(ρ+ε)) for all ε>0 but not for ρ replaced by ρ−ε.

Liouville's Theorem (review)

Every bounded entire function is constant. Equivalently, a non-constant entire function is unbounded.

Picard's Theorems

Little Picard Theorem

Every non-constant entire function takes every complex value, with at most one exception.

Example: eᶻ omits 0 (the only exception). sin z, cos z take every value.

f entire, non-constant ⟹ f(ℂ) ⊇ ℂ \ {at most one point}
Great (Big) Picard Theorem

Near an essential singularity z₀, a holomorphic function takes every complex value, with at most one exception, infinitely often.

This dramatically strengthens Casorati-Weierstrass (which only says the image is dense).

Casorati-Weierstrass vs. Picard
  • Casorati-Weierstrass: Near an essential singularity, the image is dense in ℂ.
  • Big Picard: Near an essential singularity, the image is all of ℂ minus at most one point, and each value is taken infinitely often.
Classic Example

e^(1/z) has an essential singularity at z=0. Near z=0: e^(1/z) takes every nonzero value infinitely often (omits only 0). So as z→0 through various sequences, e^(1/z) can approach any nonzero complex number.

Weierstrass Products & Mittag-Leffler

Weierstrass Factorization Theorem

Weierstrass Product Theorem

Given any sequence {aₙ} in ℂ with |aₙ|→∞, there exists an entire function with zeros exactly at {aₙ} (counted with multiplicity).

More precisely, every entire function f with zeros {aₙ} (and a zero of order m at 0) has the form:

f(z) = z^m · e^(g(z)) · ∏ₙ E(z/aₙ, pₙ)

where g is entire and E(u,p) = (1−u)·exp(u + u²/2 + ··· + uᵖ/p) is the Weierstrass elementary factor.

Classic Example: sin(πz)
sin(πz) = πz · ∏_{n=1}^∞ (1 − z²/n²)

This factorization (Euler) gives the product formula for sine and, comparing with Taylor series, yields Σ1/n²=π²/6.

Hadamard Factorization

Hadamard's Theorem

If f is entire of finite order ρ with zeros {aₙ}, then:

f(z) = z^m · e^(P(z)) · ∏ₙ E(z/aₙ, p)

where P is a polynomial of degree ≤ ρ and p = ⌊ρ⌋. The genus is h = max(p, deg P).

Mittag-Leffler Theorem

Mittag-Leffler

Given any sequence {aₙ}→∞ and prescribed principal parts (finite-order poles), there exists a meromorphic function on ℂ with exactly those poles and principal parts.

f(z) = Σₙ [Pₙ(1/(z−aₙ)) − pₙ(z)] + entire function

where pₙ(z) are polynomials added to ensure convergence.

Meromorphic Functions

Definition

A function is meromorphic on a domain D if it is holomorphic on D except at isolated poles.

Meromorphic functions on ℂ are precisely quotients of entire functions: f = g/h with g, h entire and h ≢ 0.

Argument Principle (Review)

Argument Principle
(1/2πi) ∮_γ f'(z)/f(z) dz = Z − P

Z = number of zeros, P = number of poles (both counted with multiplicity) of f inside γ.

The left side equals the winding number of f(γ) around 0.
