Introduction

Numerical methods introduce errors. Understanding these is essential for CSIR NET: absolute/relative errors, floating-point representation, rounding vs truncation, and how errors propagate. The order of convergence determines how fast a method improves with more work.

Types of Errors

Definitions
Absolute error: |x̃ − x| Relative error: |x̃ − x| / |x| (x ≠ 0) Percentage error: 100 · |x̃ − x| / |x|

x = true value, x̃ = approximate value.

Error TypeSourceExample

Significant Digits

A number x̃ approximates x to n significant digits if |x̃−x|≤0.5×10^(k−n+1) where 10^(k−1)≤|x|<10^k. Example: π≈3.14159 has 6 significant digits.

Floating-Point Arithmetic

Floating-Point Representation
x = ±0.d₁d₂…dₜ × β^e β = base (usually 2 or 10) t = precision (number of digits) e = exponent (e_min ≤ e ≤ e_max)
Machine Epsilon (ε_mach)
ε_mach = β^(1−t) (for rounding), or β^(1−t)/2 Smallest ε > 0 such that fl(1+ε) ≠ 1 IEEE double precision: β=2, t=53 ⟹ ε_mach ≈ 2.2×10⁻¹⁶

All floating-point operations have relative error ≤ ε_mach: |fl(x)−x|/|x| ≤ ε_mach.

Catastrophic Cancellation

Subtracting nearly-equal numbers: if x≈y, then x−y loses significant digits. Example: x=1.000001, y=1.000000, x−y=0.000001 (one significant digit from 7-digit numbers).

Remedy: reformulate to avoid subtraction of nearly-equal quantities (e.g., quadratic formula).

Order of Convergence

Rate / Order of Convergence

A sequence xₙ→x has order of convergence p and rate C if:

|x_{n+1}−x| ≤ C·|xₙ−x|^p (as n→∞) p=1 (linear), p=2 (quadratic), p=1.618… (superlinear/golden)

Condition Number

For a problem f(x): κ=|xf′(x)/f(x)| = relative sensitivity of output to input. Large κ → ill-conditioned. For Ax=b: κ(A)=‖A‖·‖A⁻¹‖.

Summary

Absolute/Relative Error

|x̃−x| and |x̃−x|/|x|. Relative error amplified for subtraction of nearly-equal numbers.

Round-off vs Truncation

Round-off: finite precision. Truncation: finite approximation of infinite process.

Machine Epsilon

ε_mach = β^(1−t) ≈ 2.2×10⁻¹⁶ (IEEE double). Fundamental limit of float arithmetic.

Catastrophic Cancellation

x−y loses digits when x≈y. Avoid by algebraic reformulation.

Convergence Order

p=2 (Newton), p≈1.618 (secant), p=1 (bisection, fixed-point).

Condition Number

κ(A)=‖A‖‖A⁻¹‖. Large κ → ill-conditioned → small errors amplified.
