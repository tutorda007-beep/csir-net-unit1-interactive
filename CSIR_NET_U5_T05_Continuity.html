Introduction

Topology studies properties preserved under homeomorphisms — continuous bijections with continuous inverses. Two homeomorphic spaces are "topologically identical". Topological invariants (like compactness, connectedness, homotopy type) are unchanged by homeomorphisms.

Topological Continuity

Continuous Map (Topological)

f: X→Y is continuous if f⁻¹(V) is open in X for every open V ⊆ Y.

Equivalently: f⁻¹(F) is closed for every closed F, or f preserves convergence of sequences (in metric spaces).

Continuous Bijection vs Homeomorphism

A continuous bijection need NOT be a homeomorphism. Example: f: [0,1)→S¹ (circle) by f(t)=e^(2πit) is continuous and bijective but NOT a homeomorphism (f⁻¹ is not continuous: S¹ is compact, [0,1) is not).

Homeomorphisms

Homeomorphism

f: X→Y is a homeomorphism if f is bijective, f is continuous, and f⁻¹ is continuous. Write X ≅ Y.

Non-homeomorphisms

[0,1] ≇ (0,1): removing any point from [0,1] can disconnect it (endpoints), while removing any point from (0,1) also disconnects. BUT: [0,1] is compact and (0,1) is not. So [0,1] ≇ (0,1). Also: ℝ ≇ ℝ² (ℝ\{pt} disconnects, ℝ²\{pt} does not).

Compact + Bijective Continuous ⟹ Homeomorphism

If f: X→Y is continuous, bijective, X is compact, and Y is Hausdorff, then f is a homeomorphism.

Proof: f(F) is closed for any closed F⊆X (compact → compact → closed in Hausdorff Y). So f⁻¹ is continuous.

ℝ ≇ ℝⁿ (n≥2): Proof via Invariant

ℝ\{p} is disconnected. ℝⁿ\{p} for n≥2 is connected (path connected: go around the point). Since homeomorphisms preserve connectedness, ℝ ≇ ℝⁿ for n≥2.

Open & Closed Maps

Open Map

f: X→Y is an open map if f(U) is open in Y for every open U ⊆ X.

Bijective + continuous + open map ⟹ homeomorphism.

Closed Map

f: X→Y is a closed map if f(F) is closed in Y for every closed F ⊆ X.

Every continuous map from a compact space to a Hausdorff space is a closed map.

Open Mapping Theorem (Functional Analysis)

Every surjective bounded linear operator between Banach spaces is an open map. This powerful theorem implies: bijective bounded linear operator between Banach spaces has bounded inverse.

Closed Graph Theorem

A linear map T: X→Y between Banach spaces is continuous iff its graph {(x,Tx)} is closed in X×Y.

Summary

Continuous

f⁻¹(open)=open. Equivalently: f⁻¹(closed)=closed or sequential continuity in metric spaces.

Homeomorphism

Bijective, f and f⁻¹ continuous. Topological isomorphism. X≅Y means same topology.

Compact+Hausdorff

Continuous bijection from compact to Hausdorff is automatically a homeomorphism.

Invariants

Compactness, connectedness, # components, separability preserved by homeomorphisms.

Open Map

f(U) open for all open U. OMT: surjective bounded linear between Banach spaces is open.

ℝ≇ℝⁿ

Removing a point from ℝ disconnects it; ℝⁿ\{pt} (n≥2) remains connected. Key invariant argument.
