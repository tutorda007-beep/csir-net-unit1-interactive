Why Eigenvalues?

Eigenvalues capture the "stretching factors" of a linear transformation. When T has enough eigenvectors to form a basis, the matrix becomes diagonal in that basis — making computations like matrix powers and exponentials trivial.

Key Questions

  • What values of λ make (A − λI) singular?
  • For each λ, what is the eigenspace?
  • When can A be diagonalized?
  • What is the minimal polynomial of A?

Definitions

Definition — Eigenvalue & Eigenvector

Let A be an n×n matrix (or T: V→V a linear operator). A scalar λ ∈ 𝔽 is an eigenvalue of A if there exists a non-zero vector v (the eigenvector) such that:

Av = λv ⟺ (A − λI)v = 0 ⟺ A − λI is singular

The eigenspace for eigenvalue λ is E_λ = ker(A − λI) = {v : Av = λv}. It includes the zero vector but eigenvectors are non-zero by definition.

Key Observations

  • v is an eigenvector iff v ∈ E_λ\{0} for some eigenvalue λ
  • If v is an eigenvector for λ, so is αv for any α ≠ 0
  • Eigenspaces are subspaces (always contain 0, but eigenvectors exclude 0)
  • Eigenvectors for distinct eigenvalues are linearly independent
  • λ = 0 is an eigenvalue iff A is singular (non-invertible)
Geometric Interpretation (ℝ²)

Av = λv means the transformation T: x↦Ax leaves the direction of v unchanged. If λ > 0, the vector is stretched by factor λ. If λ < 0, it is reflected and stretched. If λ = 1, v is fixed. If λ = 0, v is collapsed to 0.

Characteristic Polynomial

Definition

The characteristic polynomial of A is:

p_A(λ) = det(A − λI) = det(λI − A) (up to sign)

This is a polynomial of degree n in λ. The eigenvalues are exactly the roots of p_A(λ).

Computing Characteristic Polynomial

For a 2×2 matrix A = [[a,b],[c,d]]:

p_A(λ) = det(A−λI) = (a−λ)(d−λ) − bc = λ² − (a+d)λ + (ad−bc) = λ² − tr(A)λ + det(A)
Key Relations (any n×n matrix)
  • Trace = sum of eigenvalues: tr(A) = λ₁ + λ₂ + ... + λₙ
  • Determinant = product of eigenvalues: det(A) = λ₁ · λ₂ · ... · λₙ
  • Coefficients of char. poly. are symmetric functions of eigenvalues
Fundamental Theorem of Algebra Consequence

Over ℂ, p_A(λ) always has exactly n roots (counting multiplicity). Over ℝ, complex roots occur in conjugate pairs. Every real matrix has real characteristic polynomial, but may have complex eigenvalues.

Example: 3×3 Matrix

Example

A = [[2,1,0],[0,2,0],[0,0,3]]. Then det(A−λI) = (2−λ)²(3−λ). Eigenvalues: λ₁=2 (multiplicity 2), λ₂=3 (multiplicity 1).

tr(A) = 2+2+3 = 7 = 2+2+3 ✓. det(A) = 4·3 = 12 = 2·2·3 ✓.

Algebraic and Geometric Multiplicity

Definitions

Algebraic multiplicity (AM) of eigenvalue λ: multiplicity of λ as a root of p_A(λ).

Geometric multiplicity (GM) of eigenvalue λ: dim(E_λ) = dim(ker(A−λI)) = nullity(A−λI).

Key Inequality
1 ≤ GM(λ) ≤ AM(λ)

The geometric multiplicity is at least 1 (since λ is an eigenvalue, there is at least one eigenvector) and at most the algebraic multiplicity.

Significance

GM = AM for all λ

A is diagonalizable. There exists a basis of eigenvectors. P⁻¹AP = D (diagonal).

GM < AM for some λ

A is NOT diagonalizable. The "missing" eigenvectors lead to Jordan blocks. A requires Jordan canonical form.

Common Exam Trap

For a 3×3 matrix with eigenvalues 1,1,2 (AM: 2 for λ=1, 1 for λ=2), you must check if GM(1) = 1 or 2. If GM(1)=1, the matrix is NOT diagonalizable. If GM(1)=2, it IS diagonalizable.

Diagonalization

Definition — Diagonalizable

A is diagonalizable if A = PDP⁻¹ for some invertible P and diagonal D. Equivalently, A has n linearly independent eigenvectors (forming the columns of P).

Criteria for Diagonalizability

An n×n matrix A is diagonalizable iff any ONE of:

  1. A has n linearly independent eigenvectors
  2. For every eigenvalue λ: GM(λ) = AM(λ)
  3. The minimal polynomial of A has no repeated roots
  4. V = E_{λ₁} ⊕ E_{λ₂} ⊕ ... ⊕ E_{λₖ} (eigenspace direct sum = V)
Diagonalization Procedure
  1. Find char. poly. p_A(λ) = det(A−λI)
  2. Find eigenvalues (roots of p_A)
  3. For each eigenvalue λᵢ, solve (A−λᵢI)v = 0 to find eigenvectors
  4. If total eigenvectors = n, form P = [v₁|v₂|...|vₙ] and D = diag(λ₁,...,λₙ)
  5. Verify: A = PDP⁻¹

Applications of Diagonalization

Matrix Powers

Aⁿ = PDⁿP⁻¹ = P·diag(λ₁ⁿ,...,λₖⁿ)·P⁻¹. Computing high powers becomes trivial.

Matrix Exponential

e^A = Pe^D P⁻¹ = P·diag(e^λ₁,...,e^λₖ)·P⁻¹. Used in solving systems of ODEs.

Important Classes of Diagonalizable Matrices
  • Symmetric real matrices are always diagonalizable over ℝ (Spectral Theorem)
  • Matrices with n distinct eigenvalues are always diagonalizable
  • Hermitian matrices are diagonalizable over ℂ
  • Unitary and orthogonal matrices are diagonalizable over ℂ

Cayley-Hamilton Theorem

Cayley-Hamilton Theorem

Every square matrix satisfies its own characteristic equation:

p_A(A) = 0

If p_A(λ) = λⁿ + c_{n−1}λⁿ⁻¹ + ... + c₁λ + c₀, then Aⁿ + c_{n−1}Aⁿ⁻¹ + ... + c₁A + c₀I = 0.

Example (2×2)

A = [[1,2],[3,4]]. p_A(λ) = λ²−5λ−2. By Cayley-Hamilton: A²−5A−2I = 0.

This gives A² = 5A + 2I — expresses A² in terms of A and I.

Applications of Cayley-Hamilton

Computing A⁻¹

If c₀ = det(A) ≠ 0: from p_A(A)=0, Aⁿ + ... + c₀I = 0, so c₀I = −Aⁿ − ... − c₁A. Multiply by A⁻¹: c₀A⁻¹ = −Aⁿ⁻¹ − ... − c₁I. Thus A⁻¹ = (−1/c₀)(Aⁿ⁻¹ + ... + c₁I).

Reducing High Powers

Use p_A(A)=0 to write Aⁿ = −c_{n−1}Aⁿ⁻¹ − ... − c₀I. Iteratively reduce any power Aᵏ (k≥n) to a polynomial of degree < n in A.

Minimal Polynomial

Definition

The minimal polynomial m_A(λ) of A is the monic polynomial of smallest degree such that m_A(A) = 0.

  • m_A(λ) divides p_A(λ) (characteristic polynomial)
  • Every eigenvalue of A is a root of m_A(λ)
  • The roots of m_A(λ) are exactly the eigenvalues (but with possibly smaller multiplicities)

Key Theorem

A is diagonalizable ⟺ m_A(λ) has no repeated roots ⟺ m_A splits into distinct linear factors over 𝔽.

Examples
  • A = diag(1,2,2): p_A = (λ−1)(λ−2)², m_A = (λ−1)(λ−2). A is diagonalizable (m_A has no repeated roots).
  • A = [[1,1],[0,1]]: p_A = (λ−1)², m_A = (λ−1)². A is NOT diagonalizable (m_A has repeated root).
  • A = 2I: p_A = (λ−2)³, m_A = (λ−2). A is diagonalizable (trivially).
