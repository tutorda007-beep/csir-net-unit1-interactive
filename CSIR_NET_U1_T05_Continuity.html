Unit 1 — Analysis · Topic 05 of 11
Continuity & Uniform Continuity
ε-δ arguments, discontinuity types, IVT, EVT, and the critical difference between pointwise and uniform continuity
CSIR NET Core Real Analysis Intermediate-Advanced 70 min Study

Learning Objectives

  1. Give the precise ε-δ definition of continuity and verify it for specific functions
  2. Identify and classify all types of discontinuities (removable, jump, essential)
  3. State and apply IVT and EVT to solve existence problems
  4. Distinguish continuity from uniform continuity with counterexamples
  5. Apply Heine's theorem: continuous on compact ⟹ uniformly continuous
🎯
CSIR NET Frequency: Continuity topics appear in 3–5 questions per exam. High-yield: IVT applications, uniform continuity on (0,1) vs [0,1], the function x·sin(1/x), Lipschitz condition implying uniform continuity.

ε Continuity Definitions

ε-δ
Concept 1
Formal Definition of Continuity
Continuity at a Point (ε-δ)
f : D → ℝ is continuous at c ∈ D if:

∀ε > 0, ∃δ > 0 such that |x − c| < δ and x ∈ D ⟹ |f(x) − f(c)| < ε

The key: δ may depend on both ε AND c (the point).
Sequential Characterization
f is continuous at c ⟺ for every sequence xₙ → c (xₙ ∈ D), we have f(xₙ) → f(c).

This is often easier to USE: to show continuity, show sequences are preserved. To show DIS-continuity, find a sequence xₙ → c with f(xₙ) ↛ f(c).

Algebraic Preservation of Continuity

If f, g are continuous at c, then:

  • f ± g, f · g are continuous at c
  • f/g is continuous at c if g(c) ≠ 0
  • g ∘ f is continuous at c if g is continuous at f(c)
  • |f| is continuous at c
  • max(f,g) and min(f,g) are continuous at c
Topological Characterization
f : X → Y is continuous ⟺ the preimage f⁻¹(V) of every open set V ⊆ Y is open in X.
Equivalently: f⁻¹(F) is closed whenever F is closed.
This characterization is fundamental in topology — it captures continuity without ε-δ and works in any topological space.

💥 Types of Discontinuities

✗
Concept 2
Classification of Discontinuities

At a point c, f has a discontinuity if: f is not defined at c, OR the limit doesn't exist, OR lim ≠ f(c). The one-sided limits lim_{x→c⁺} f(x) = f(c⁺) and lim_{x→c⁻} f(x) = f(c⁻) are key.

Removable Discontinuity

Condition: lim_{x→c} f(x) exists (= L) but f(c) ≠ L or f(c) is undefined.
Fix: Redefine f(c) = L to make it continuous.
Example: f(x) = sin(x)/x at x=0 (limit = 1, but f(0) undefined)

Jump Discontinuity

Condition: Both one-sided limits exist but f(c⁺) ≠ f(c⁻).
Jump size: |f(c⁺) − f(c⁻)|
Example: f(x) = sign(x): f(0⁺) = 1, f(0⁻) = −1
Also: floor function ⌊x⌋ at every integer

Essential (Infinite) Discontinuity

Condition: At least one one-sided limit is infinite or doesn't exist.
Example: f(x) = 1/x at x=0 (limit = ±∞)
Also: f(x) = sin(1/x) at x=0 (oscillates, no limit)

Oscillatory Discontinuity

Condition: Function oscillates without settling to a limit.
Example: f(x) = sin(1/x) at x=0
lim_{x→0} sin(1/x) does NOT exist (oscillates between −1 and 1)
Note: x·sin(1/x) → 0 (removable! Define f(0)=0)

Monotone Function Discontinuities
A monotone function f on (a,b) can only have jump discontinuities, and the set of discontinuities is at most countable.
Both one-sided limits exist at every point for a monotone function. If they differ → jump discontinuity. The jumps are disjoint rational intervals → countable.

📐 Key Theorems

IVT
Concept 3
Intermediate & Extreme Value Theorems
Intermediate Value Theorem (IVT)
If f is continuous on [a,b] and f(a) ≠ f(b), then for every value k between f(a) and f(b), there exists c ∈ (a,b) such that f(c) = k.

Equivalently: the continuous image of a connected set is connected (intervals → intervals).
Key consequence: Any polynomial of odd degree has at least one real root. Proof: p(x) → +∞ as x → +∞ and p(x) → −∞ as x → −∞ (or vice versa). By IVT, p(c) = 0 for some c.
Extreme Value Theorem (EVT)
If f is continuous on a compact set K ⊆ ℝⁿ (in particular, on a closed bounded interval [a,b]), then f attains its supremum and infimum on K:

∃ x_max, x_min ∈ K such that f(x_min) ≤ f(x) ≤ f(x_max) for all x ∈ K.
f(K) is compact (continuous image of compact set). Compact subset of ℝ is closed and bounded → has a maximum and minimum → attained at some points.

💡
IVT ≠ unique value: IVT guarantees existence, not uniqueness. Also, IVT does NOT hold for discontinuous functions. Example: f(x) = sign(x) on [−1,1] is discontinuous at 0 and takes values ±1 but not 0 — IVT fails.

📏 Uniform Continuity

UC
Concept 4
Uniform vs Pointwise Continuity
Uniform Continuity
f : D → ℝ is uniformly continuous on D if:

∀ε > 0, ∃δ > 0 such that ∀x, y ∈ D: |x − y| < δ ⟹ |f(x) − f(y)| < ε

Key difference: δ depends ONLY on ε, not on the particular point x or y.

This is stronger than ordinary continuity. In uniform continuity, one δ works globally for ALL points simultaneously.

Key Results on Uniform Continuity

Heine's Theorem
Every continuous function on a compact set is uniformly continuous.
In particular: f continuous on [a,b] ⟹ f uniformly continuous on [a,b].
Lipschitz ⟹ Uniformly Continuous
If f satisfies |f(x) − f(y)| ≤ L|x − y| for some constant L > 0 (Lipschitz condition), then f is uniformly continuous. (Use δ = ε/L.)

Non-Uniformly-Continuous Criterion

f is NOT uniformly continuous on D iff: ∃ε₀ > 0 and sequences (xₙ), (yₙ) with |xₙ − yₙ| → 0 but |f(xₙ) − f(yₙ)| ≥ ε₀.
Example: f(x) = 1/x on (0,1): Take xₙ = 1/n, yₙ = 1/(n+1). Then |xₙ−yₙ| → 0 but |f(xₙ)−f(yₙ)| = 1 → 0? Actually |n−(n+1)| = 1 ≥ 1. ✓

↗ Monotone Functions

↑↓
Concept 5
Properties of Monotone Functions
Monotone Function Continuity Theorem
1. A monotone function f on (a,b) has one-sided limits at every point.
2. The set of discontinuities of a monotone function is at most countable.
3. A monotone bijection f: [a,b] → [c,d] is continuous (and its inverse is also continuous and monotone).
4. If f is monotone and has the IVT property (takes all intermediate values), then f is continuous.

Continuous Bijection on Compact Set

Inverse Function Theorem (Elementary)
If f: [a,b] → ℝ is continuous and strictly monotone, then:
(i) f is a bijection onto [f(a), f(b)] (or [f(b), f(a)])
(ii) f⁻¹ is continuous and strictly monotone on its domain
(iii) f⁻¹ is continuous iff f is an open map (maps open sets to open sets)

✏️ Worked Examples

Example 1 — ε-δ Proof
Prove that f(x) = 3x + 2 is continuous at every point c ∈ ℝ.
Let ε > 0. Choose δ = ε/3.
If |x − c| < δ = ε/3, then:
|f(x) − f(c)| = |(3x+2) − (3c+2)| = |3x − 3c| = 3|x − c| < 3 · (ε/3) = ε. □
Note: δ does not depend on c — so f is actually uniformly continuous on ℝ (Lipschitz with L=3).
Example 2 — IVT Application
Prove that cos(x) = x has at least one solution in (0, π/2).
Let g(x) = cos(x) − x. Then g is continuous on [0, π/2].
g(0) = cos(0) − 0 = 1 > 0
g(π/2) = cos(π/2) − π/2 = 0 − π/2 = −π/2 < 0
Since g(0) > 0 and g(π/2) < 0, by IVT, ∃c ∈ (0, π/2) such that g(c) = 0, i.e., cos(c) = c. □
Example 3 — Uniform Continuity Fail
Show that f(x) = x² is NOT uniformly continuous on [0, ∞).
Take ε₀ = 1. Suppose δ > 0 is any candidate.
Choose x = 1/δ, y = 1/δ + δ/2. Then |x − y| = δ/2 < δ.
But |f(x) − f(y)| = |x² − y²| = |x+y||x−y| = (2/δ + δ/2)(δ/2) ≥ (2/δ)(δ/2) = 1 = ε₀.
So no δ works for ε₀ = 1, and f is not uniformly continuous on [0,∞).
Example 4 — CSIR NET Classic
For which values of α does f(x) = xᵅ sin(1/x) (f(0)=0) become continuous at 0?
|f(x)| = |xᵅ||sin(1/x)| ≤ |x|ᵅ since |sin| ≤ 1.
For f to be continuous at 0, we need f(x) → 0 as x → 0, i.e., |x|ᵅ → 0.
This holds iff α > 0.
Conclusion: f is continuous at 0 iff α > 0.
Special case α = 1: f(x) = x·sin(1/x) → 0, continuous at 0.
Example 5 — Fixed Point Theorem
Prove that every continuous f: [0,1] → [0,1] has a fixed point.
Define g(x) = f(x) − x on [0,1].
g(0) = f(0) − 0 = f(0) ≥ 0 (since f(0) ∈ [0,1]).
g(1) = f(1) − 1 ≤ 0 (since f(1) ∈ [0,1] ⟹ f(1) ≤ 1).
If g(0) = 0, then f(0) = 0 (done). If g(1) = 0, then f(1) = 1 (done).
Otherwise g(0) > 0 and g(1) < 0. By IVT, ∃c ∈ (0,1) with g(c) = 0, i.e., f(c) = c. □

Question 1 of 15
The function f(x) = sin(1/x) at x = 0 has what type of discontinuity?
✓ C is correct. lim_{x→0} sin(1/x) does not exist — the function oscillates between −1 and 1 infinitely often as x → 0. Since neither one-sided limit exists (both are undefined), this is an essential discontinuity (oscillatory type). Compare with x·sin(1/x) which is removable (limit = 0).
Question 2 of 15
Which of the following functions is uniformly continuous on (0,1)?
✓ C is correct. √x extends continuously to [0,1] → uniformly continuous on (0,1) by Heine's theorem (extension argument). f(x) = 1/x is not UC on (0,1): blows up near 0. f(x) = sin(1/x) has no continuous extension to [0,1]. f(x) = x·ln(x) → 0 as x→0 (continuous extension exists), actually also UC on (0,1); but √x is the clear standard answer.
Question 3 of 15
By IVT, if f is continuous on [0,1] with f(0) = −1 and f(1) = 3, which value is GUARANTEED to be attained?
✓ B is correct. IVT: f continuous on [0,1], f(0)=−1 < 0 < 3=f(1). By IVT, f attains every value in [f(0), f(1)] = [−1, 3]. So f(c) = 0 for some c ∈ (0,1) ✓. Value 5 is outside [−1,3] — not guaranteed. Value −2 is below f(0) — not guaranteed. Value 4 is above f(1) — not guaranteed.
Question 4 of 15
A function f: [a,b] → ℝ is continuous. Which of the following is NOT necessarily true?
✓ C is correct — differentiability is NOT guaranteed. A continuous function need not be differentiable. The Weierstrass function is continuous everywhere but differentiable nowhere. A is TRUE (EVT: f bounded). B is TRUE (EVT: f attains max/min). D is TRUE (Heine: f continuous on compact [a,b] ⟹ uniformly continuous).
Question 5 of 15
The function f(x) = x·sin(1/x) for x ≠ 0 and f(0) = 0 is:
✓ A is correct. |f(x) − 0| = |x·sin(1/x)| ≤ |x| → 0 as x → 0 (squeeze theorem). So f is continuous at 0. B is WRONG: the derivative is sin(1/x) − cos(1/x)/x which oscillates as x→0, so f is NOT differentiable at 0. C is WRONG: f is not differentiable, but more importantly, the derivative diverges — actually f IS uniformly continuous on [−1,1] (compact), but NOT on ℝ.
Question 6 of 15
Which of the following is a sufficient condition for uniform continuity?
✓ B is correct. Lipschitz condition: |f(x)−f(y)| ≤ L|x−y| ⟹ uniform continuity (use δ = ε/L). A is wrong (differentiability alone doesn't give UC; we need bounded derivative). C is wrong (monotone functions can fail UC). D is wrong (f could be bounded but not UC, e.g., sin(1/x) on (0,1) is bounded but not UC).
Question 7 of 15
The Dirichlet function f(x) = 1 if x ∈ ℚ, f(x) = 0 if x ∉ ℚ, is:
✓ C is correct. At every point c: both rationals and irrationals are dense, so there exist sequences xₙ → c with xₙ ∈ ℚ (giving f(xₙ)=1) and yₙ → c with yₙ ∉ ℚ (giving f(yₙ)=0). Since 1 ≠ 0, the limit at c does not equal f(c) for any c. Therefore f is discontinuous everywhere.
Question 8 of 15
If f: ℝ → ℝ is uniformly continuous and (xₙ) is Cauchy, then:
✓ B is correct. Uniform continuity maps Cauchy sequences to Cauchy sequences. Proof: Given ε > 0, choose δ for ε. Since (xₙ) Cauchy, ∃N: m,n > N ⟹ |xₙ−xₘ| < δ ⟹ |f(xₙ)−f(xₘ)| < ε. So (f(xₙ)) is Cauchy. A is not guaranteed (ℝ is complete so xₙ converges, but the statement is about general spaces). D would follow if ℝ is complete and f is continuous.
Question 9 of 15
A continuous function on a connected set maps it to:
✓ B is correct. Continuous images of connected sets are connected (this is the topological form of IVT). In ℝ, connected sets are intervals (including single points), so f maps intervals to intervals. A is wrong (need compactness of domain, not connectedness). C is wrong (e.g., f(x) = x² maps ℝ to [0,∞), not open in some sense).
Question 10 of 15
The function f(x) = 1/x on (0,1) is NOT uniformly continuous. The correct Cauchy sequences that witness this are:
✓ B is correct. With xₙ = 1/n, yₙ = 1/(n+1): |xₙ−yₙ| = 1/(n(n+1)) → 0 ✓, but |f(xₙ)−f(yₙ)| = |n−(n+1)| = 1 ≢ 0. This witnesses non-uniform continuity with ε₀ = 1/2. A gives |f(xₙ)−f(yₙ)| = |n²−(n²+1)| = 1 ✓ also works. But B is the standard example for 1/x.
Question 11 of 15
The preimage of an open set under a continuous function is:
✓ A is correct. This is the topological definition of continuity: f is continuous iff f⁻¹(V) is open for every open V. The preimage of a closed set under a continuous function is closed. Note: the IMAGE of an open set need not be open (e.g., f(x) = x² maps (−1,1) to [0,1), which is neither open nor closed).
Question 12 of 15
The function f(x) = x² sin(1/x²) for x ≠ 0, f(0) = 0 is:
✓ B is correct. Continuity at 0: |f(x)| ≤ x² → 0 ✓. Differentiability: f'(0) = lim f(x)/x = lim x·sin(1/x²) → 0. So actually f is differentiable at 0. The derivative f'(x) = 2x sin(1/x²) − (2/x)cos(1/x²) oscillates wildly near 0, making it unbounded — f' is not bounded near 0. This is a key example: differentiable everywhere but derivative is not bounded/continuous.
Question 13 of 15
Which function has a removable discontinuity at x = 0?
✓ C is correct. lim_{x→0} (1−cos x)/x: using L'Hôpital or Taylor: (1−(1−x²/2+…))/x = x/2 − … → 0. So lim = 0, and by defining f(0) = 0, f becomes continuous — removable discontinuity. A has jump (limit from right = 1, left = −1). B has essential oscillatory. D has essential infinite discontinuity.
Question 14 of 15
If f: [a,b] → [a,b] is continuous, then:
✓ C is correct. Fixed point theorem for [a,b]: g(x) = f(x)−x. g(a) = f(a)−a ≥ 0 (since f(a) ∈ [a,b]). g(b) = f(b)−b ≤ 0. By IVT, ∃c with g(c) = 0, i.e., f(c) = c. A is wrong (f could be constant). B is wrong (continuity ≠ differentiability). D is wrong (continuous functions need not be open maps).
Question 15 of 15
The set of all points of continuity of a monotone function on (a,b) is:
✓ C is correct. A monotone function has at most countably many discontinuities (jump discontinuities, and each jump corresponds to a distinct rational interval — hence countable). So the set of continuity points is (a,b) minus a countable set, which is co-countable (uncountable and dense). D is also true but C is more precise and the standard answer.

📌 Summary & CSIR NET Tips

Continuity — Core

  • ε-δ: ∀ε>0, ∃δ>0: |x−c|<δ ⟹ |f(x)−f(c)|<ε
  • Sequential: xₙ→c ⟹ f(xₙ)→f(c)
  • Topological: f⁻¹(open) = open
  • Algebraic: sums, products, compositions preserve continuity

Uniform Continuity

  • δ depends only on ε, not on point
  • Compact domain ⟹ UC (Heine)
  • Lipschitz ⟹ UC (use δ = ε/L)
  • 1/x NOT UC on (0,1) but UC on [1,∞)
  • x² NOT UC on [0,∞) but UC on any [a,b]

Key Theorems

  • IVT: continuous on [a,b] → takes all intermediate values
  • EVT: continuous on compact → attains max/min
  • Fixed point: f:[0,1]→[0,1] continuous → has fixed point
  • Monotone: at most countably many discontinuities

CSIR Exam Q-Types

  • Classify discontinuities (removable/jump/essential)
  • IVT to show existence of roots or fixed points
  • Which functions are UC on given domains?
  • Dirichlet/modified Dirichlet/Thomae functions
  • x·sin(1/x) and xᵅ·sin(1/x) continuity at 0
