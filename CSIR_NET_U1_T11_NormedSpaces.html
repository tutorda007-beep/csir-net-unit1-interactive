Introduction

A normed linear space combines the algebraic structure of a vector space with the geometric structure of a metric (via the norm). When complete, it becomes a Banach space — the natural setting for most of functional analysis. The four fundamental theorems of functional analysis all concern Banach and Hilbert spaces.

Why Normed Spaces?

  • Generalize ℝⁿ to infinite dimensions
  • Study differential/integral equations as linear maps
  • Quantify "size" of functions and operators
  • Foundation for quantum mechanics (Hilbert spaces)

CSIR NET Focus

  • Norm vs inner product: which implies which
  • Identifying Banach spaces
  • Operator norm and bounded operators
  • Four fundamental theorems (HB, UBP, OMT, CGT)

Hierarchy

Normed Linear Spaces

Definition — Norm

A norm on a vector space V (over ℝ or ℂ) is a function ‖·‖: V → ℝ satisfying for all x,y ∈ V, α ∈ 𝔽:

  1. Non-negativity: ‖x‖ ≥ 0 and ‖x‖ = 0 ⟺ x = 0
  2. Scalar multiplication: ‖αx‖ = |α|·‖x‖
  3. Triangle inequality: ‖x + y‖ ≤ ‖x‖ + ‖y‖

A normed linear space (NLS) is a pair (V, ‖·‖).

Norm Induces Metric

Every norm induces a metric: d(x,y) = ‖x−y‖. This metric satisfies translation invariance: d(x+z, y+z) = d(x,y), and homogeneity: d(αx, αy) = |α|·d(x,y).

Equivalent Norms

Definition

Two norms ‖·‖₁ and ‖·‖₂ on V are equivalent if ∃ c₁,c₂ > 0 such that c₁‖x‖₁ ≤ ‖x‖₂ ≤ c₂‖x‖₁ for all x ∈ V.

Theorem — Finite Dimensions

On a finite-dimensional vector space, all norms are equivalent. Completeness depends only on the vector space, not the choice of norm.

On infinite-dimensional spaces, norms can be inequivalent. E.g., on C[0,1], ‖f‖₁ = ∫|f| and ‖f‖∞ = max|f| are not equivalent.

Series in Normed Spaces

Absolutely Convergent Series

A series Σxₙ in a normed space is absolutely convergent if Σ‖xₙ‖ < ∞.

Key theorem: A normed space is a Banach space ⟺ every absolutely convergent series converges (in the space). This is extremely useful for proving completeness.

Banach Spaces

Definition — Banach Space

A Banach space is a normed linear space (V, ‖·‖) that is complete with respect to the induced metric d(x,y) = ‖x−y‖.

Proving Completeness

Method 1: Direct

Take an arbitrary Cauchy sequence (xₙ). Find a candidate limit x. Show ‖xₙ − x‖ → 0 and x ∈ V.

Method 2: Absolutely Convergent Series

Show that Σ‖xₙ‖ < ∞ implies Σxₙ converges in V. If this holds, V is Banach.

Subspaces of Banach Spaces

Theorem

A closed subspace of a Banach space is a Banach space. A finite-dimensional subspace of a normed space is always closed. An infinite-dimensional subspace may or may not be closed.

Quotient Spaces

Quotient Norm

If M is a closed subspace of Banach space X, the quotient space X/M with norm ‖[x]‖ = inf_{m∈M} ‖x−m‖ is a Banach space.

Linear Operators

Definition — Bounded Linear Operator

T: X → Y (X,Y normed spaces) is a bounded linear operator if:

  • T is linear: T(αx+βy) = αT(x)+βT(y)
  • T is bounded: ‖T‖ = sup{‖Tx‖_Y / ‖x‖_X : x ≠ 0} < ∞

The operator norm ‖T‖ = sup_{‖x‖≤1} ‖Tx‖ = sup_{‖x‖=1} ‖Tx‖.

Theorem — Continuity ↔ Boundedness

For a linear operator T: X → Y, the following are equivalent:

  1. T is continuous everywhere on X
  2. T is continuous at 0
  3. T is bounded (‖T‖ < ∞)
  4. ∃ C > 0: ‖Tx‖ ≤ C‖x‖ for all x ∈ X
Unbounded Operators

On infinite-dimensional spaces, linear operators can be unbounded (discontinuous). Example: differentiation d/dx on C¹[0,1] ⊂ C[0,1] with sup-norm is unbounded — take fₙ(x) = sin(nx)/n, then ‖fₙ‖ → 0 but ‖f'ₙ‖ = ‖cos(nx)‖ = 1.

Space of Bounded Operators

Theorem

B(X,Y) = {T: X→Y : T bounded linear} with operator norm is a normed space. If Y is a Banach space, then B(X,Y) is a Banach space.

In particular, X* = B(X,𝔽) (the dual space of X) is always a Banach space.

Linear Functionals & Dual Spaces

Definition — Linear Functional

A linear functional on a normed space X is a linear map f: X → 𝔽 (where 𝔽 = ℝ or ℂ). A bounded linear functional satisfies |f(x)| ≤ C‖x‖ for some C.

The dual space X* = B(X,𝔽) with norm ‖f‖ = sup_{‖x‖≤1} |f(x)| is always a Banach space.

Reflexive Spaces

Definition

X is reflexive if the canonical embedding J: X → X** defined by J(x)(f) = f(x) is surjective. Every reflexive Banach space is weakly sequentially compact.

Reflexive: Hilbert spaces, Lᵖ (1<p<∞), ℓᵖ (1<p<∞). Non-reflexive: ℓ¹, ℓ∞, C[0,1], L¹, L∞.

Hahn-Banach Theorem

Hahn-Banach Theorem (Extension Form)

Let M be a subspace of a normed space X and f ∈ M* (bounded linear functional on M). Then there exists F ∈ X* (bounded linear functional on X) such that:

  • F(x) = f(x) for all x ∈ M (F extends f)
  • ‖F‖_{X*} = ‖f‖_{M*} (norm is preserved)

Important Corollaries

Corollaries
  1. For any x₀ ≠ 0 in X, there exists F ∈ X* with ‖F‖=1 and F(x₀) = ‖x₀‖.
  2. X* separates points: if x ≠ y, then there exists F ∈ X* with F(x) ≠ F(y).
  3. ‖x‖ = sup{|F(x)| : F ∈ X*, ‖F‖ = 1} (norm realized by functionals).
  4. If F(x₀) = 0 for all F ∈ X*, then x₀ = 0.
Separation Theorem (Geometric Hahn-Banach)

If A and B are disjoint convex sets in X with A open, then there exists a continuous linear functional F and c ∈ ℝ such that F(a) < c ≤ F(b) for all a ∈ A, b ∈ B. (Separating hyperplane theorem.)

Key Point

Hahn-Banach does NOT require completeness — it works on any normed space, even incomplete ones. This is one reason why completeness and Hahn-Banach are separated as distinct theorems.

Uniform Boundedness Principle

Banach-Steinhaus (Uniform Boundedness Principle)

Let X be a Banach space and Y a normed space. Let {Tα} ⊆ B(X,Y) be a family of bounded linear operators. If sup_α ‖Tα x‖ < ∞ for each fixed x ∈ X (pointwise bounded), then sup_α ‖Tα‖ < ∞ (uniformly bounded).

Proof Idea

Uses Baire Category Theorem: X = ⋃ₙ Aₙ where Aₙ = {x : sup_α‖Tαx‖ ≤ n}. Since X is complete (Baire), some Aₙ contains an open ball B(x₀,r). Any x ∈ B(0,r) can be written as (x₀+x) − x₀, so ‖Tαx‖ ≤ ‖Tα(x₀+x)‖ + ‖Tαx₀‖ ≤ 2n. Thus ‖Tα‖ ≤ 2n/r for all α.

Consequence: Divergence Phenomenon

If pointwise boundedness fails (i.e., sup_α‖Tαx‖ = ∞ for SOME x), then by Baire, the set of "bad" x is actually a dense G_δ set. So "most" x are bad. Example: Fourier series of continuous functions can diverge at every point of a dense set.

Application: Pointwise Convergence

Corollary

If Tₙx → Tx for all x ∈ Banach space X (pointwise), and each Tₙ is bounded, then T is bounded and ‖T‖ ≤ lim inf ‖Tₙ‖. Also, sup_n ‖Tₙ‖ < ∞.

Open Mapping & Closed Graph Theorems

Open Mapping Theorem

Let X and Y be Banach spaces and T ∈ B(X,Y) be surjective. Then T is an open map: T maps open sets to open sets.

Bounded Inverse Theorem (Corollary)

If T: X → Y is a bijective bounded linear operator between Banach spaces, then T⁻¹: Y → X is also bounded. (Bijective bounded linear map ⟹ bounded inverse.)

Completeness Is Essential

Both Banach space hypotheses are needed. Without completeness, bounded bijections can have unbounded inverses. The inclusion i: (ℓ², ‖·‖₁) → (ℓ², ‖·‖₂) has no bounded inverse when ‖·‖₁ and ‖·‖₂ are inequivalent on infinite dimensions.

Closed Graph Theorem

Let X and Y be Banach spaces and T: X → Y a linear operator. T is bounded ⟺ T has a closed graph, i.e., if xₙ → x and Txₙ → y, then Tx = y.

Using CGT in Practice

To prove T is bounded: assume xₙ → x and Txₙ → y, then use the properties of T to show Tx = y. This is often easier than directly bounding ‖Tx‖/‖x‖.

Inner Product Spaces & Hilbert Spaces

Definition — Inner Product

An inner product on V (over ℂ) is ⟨·,·⟩: V×V → ℂ satisfying for all x,y,z ∈ V, α ∈ ℂ:

  1. Conjugate symmetry: ⟨x,y⟩ = ⟨y,x⟩* (overline = complex conjugate)
  2. Linearity in first argument: ⟨αx+βy,z⟩ = α⟨x,z⟩ + β⟨y,z⟩
  3. Positive definiteness: ⟨x,x⟩ ≥ 0 with equality iff x = 0

The induced norm: ‖x‖ = √⟨x,x⟩. A Hilbert space is a complete inner product space.

Cauchy-Schwarz Inequality

Cauchy-Schwarz Inequality

|⟨x,y⟩| ≤ ‖x‖·‖y‖, with equality iff x and y are linearly dependent.

In ℝⁿ: (Σxᵢyᵢ)² ≤ (Σxᵢ²)(Σyᵢ²). In L²: (∫fg)² ≤ (∫f²)(∫g²).

Parallelogram Law

Parallelogram Identity

In an inner product space: ‖x+y‖² + ‖x−y‖² = 2(‖x‖² + ‖y‖²)

A normed space comes from an inner product ⟺ the parallelogram law holds. So ℓ¹, ℓ∞, L¹, L∞ are NOT Hilbert spaces (they fail the parallelogram law).

Orthogonality & Projections

Orthogonality

x ⊥ y if ⟨x,y⟩ = 0. Pythagorean theorem: x ⊥ y ⟹ ‖x+y‖² = ‖x‖² + ‖y‖².

M⊥ = {x ∈ H : ⟨x,m⟩ = 0 for all m ∈ M} is always a closed subspace.

Projection Theorem

Let M be a closed subspace of a Hilbert space H. Then every x ∈ H has a unique decomposition x = m + m⊥ with m ∈ M and m⊥ ∈ M⊥. The map x ↦ m is the orthogonal projection onto M.

Riesz Representation Theorem

Every bounded linear functional f on a Hilbert space H is of the form f(x) = ⟨x,y⟩ for a unique y ∈ H, and ‖f‖ = ‖y‖. Thus H* ≅ H (Hilbert spaces are self-dual).
