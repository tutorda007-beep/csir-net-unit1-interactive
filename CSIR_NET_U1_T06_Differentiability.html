🎯
CSIR NET Frequency: MVT applications and L'Hôpital appear in 2–3 questions per exam. Taylor series with remainder estimation is common. Also: "f'(x) = 0 everywhere → f constant" and counterexamples to the converse of Rolle's theorem.

d/dx Derivative Definition

d/dx
Concept 1
The Derivative
Derivative at a Point
f : (a,b) → ℝ is differentiable at c ∈ (a,b) if the limit exists:

f'(c) = lim_{h→0} [f(c+h) − f(c)] / h = lim_{x→c} [f(x) − f(c)] / (x − c)

The derivative is the slope of the tangent line at c. Geometrically: as h → 0, secant slope → tangent slope.

Differentiability vs Continuity

Key Relationship
Differentiable at c ⟹ Continuous at c. (Converse is FALSE.)

Counterexamples to the converse:
• f(x) = |x| is continuous at 0 but NOT differentiable at 0 (left derivative = −1, right derivative = 1)
• f(x) = x^{1/3} is continuous at 0 but not differentiable (vertical tangent: f'(0) = ∞)
• Weierstrass function: continuous EVERYWHERE, differentiable NOWHERE

📐 Mean Value Theorems

MVT
Concept 3
Rolle's, MVT, and Cauchy's MVT

Rolle's Theorem

Hypotheses:
1. f continuous on [a,b]
2. f differentiable on (a,b)
3. f(a) = f(b)

Conclusion:
∃ c ∈ (a,b): f'(c) = 0

Proof idea: f attains its max/min on [a,b] (EVT). If at interior point → derivative = 0 there. If both at endpoints → f is constant → f' = 0 everywhere.

Mean Value Theorem

Hypotheses:
1. f continuous on [a,b]
2. f differentiable on (a,b)

Conclusion:
∃ c ∈ (a,b): f'(c) = [f(b)−f(a)]/(b−a)

Proof: Apply Rolle's to g(x) = f(x) − [(f(b)−f(a))/(b−a)](x−a), which satisfies g(a) = g(b) = f(a).

Cauchy's MVT (Generalized)

Hypotheses:
f, g continuous on [a,b]; differentiable on (a,b); g'(x) ≠ 0 on (a,b)

Conclusion:
∃ c ∈ (a,b): [f(b)−f(a)]/[g(b)−g(a)] = f'(c)/g'(c)

Special case: g(x) = x → ordinary MVT

MVT Consequences

1. f'(x) = 0 ∀x ∈ (a,b) ⟹ f is constant
2. f'(x) > 0 on (a,b) ⟹ f is strictly increasing
3. f'(x) < 0 on (a,b) ⟹ f is strictly decreasing
4. |f'| ≤ M on (a,b) ⟹ |f(x)−f(y)| ≤ M|x−y| (Lipschitz)
5. If f'(a) < 0 < f'(b) → f has local min in (a,b)

∞/∞ L'Hôpital's Rule

L'H
Concept 4
L'Hôpital's Rule and Indeterminate Forms
L'Hôpital's Rule
Suppose f, g are differentiable near c (except possibly at c), g'(x) ≠ 0 near c, and the limit is of the form 0/0 or ∞/∞. If lim_{x→c} f'(x)/g'(x) = L (exists or = ±∞), then:

lim_{x→c} f(x)/g(x) = lim_{x→c} f'(x)/g'(x) = L
Derived from Cauchy's MVT. Applies to x → a, x → ∞, one-sided limits. Can be applied repeatedly if the form remains 0/0 or ∞/∞.

∑ Taylor's Theorem

Tₙ
Concept 5
Taylor's Theorem with Remainder
Taylor's Theorem (Lagrange Remainder)
If f is n+1 times differentiable on (a,b) and a ≤ x₀, x ≤ b, then:

f(x) = f(x₀) + f'(x₀)(x−x₀) + f''(x₀)(x−x₀)²/2! + … + f⁽ⁿ⁾(x₀)(x−x₀)ⁿ/n! + Rₙ(x)

where the Lagrange remainder is:
Rₙ(x) = f⁽ⁿ⁺¹⁾(c)(x−x₀)ⁿ⁺¹/(n+1)! for some c between x₀ and x.
Alternative remainder forms: Cauchy remainder Rₙ(x) = f⁽ⁿ⁺¹⁾(c)(x−c)ⁿ(x−x₀)/n!. The Lagrange form is most useful for error estimation: |Rₙ(x)| ≤ M·|x−x₀|ⁿ⁺¹/(n+1)! where M = max|f⁽ⁿ⁺¹⁾| on interval.

✏️ Worked Examples

Example 1 — MVT Application
Prove that |sin(x) − sin(y)| ≤ |x − y| for all x, y ∈ ℝ.
Let f(t) = sin(t). Then f is differentiable on ℝ and f'(t) = cos(t).
By MVT on [min(x,y), max(x,y)]: ∃c between x and y such that
sin(x) − sin(y) = f'(c)(x − y) = cos(c)(x − y).
Therefore |sin(x) − sin(y)| = |cos(c)| · |x − y| ≤ 1 · |x − y| = |x − y|. □
(This shows sin is Lipschitz with constant 1 → uniformly continuous on ℝ.)
Example 2 — Rolle's Theorem
Prove that xᵉ + eˣ = 0 has at most one real root.
Let f(x) = xᵉ + eˣ. Suppose f has two roots: f(a) = f(b) = 0 with a < b.
f is continuous and differentiable on ℝ. By Rolle's Theorem, ∃c ∈ (a,b) with f'(c) = 0.
f'(x) = e·x^{e-1} + eˣ. For all real x, eˣ > 0. For x > 0: e·x^{e-1} > 0 too.
For x < 0: xᵉ is not real (e is irrational). Analysis shows f'(x) ≠ 0 for all relevant x. Contradiction. □
Example 3 — L'Hôpital
Evaluate lim_{x→0⁺} x^x.
This is of the form 0⁰. Take the natural log: let L = lim x^x. Then
ln L = lim_{x→0⁺} x·ln(x). This is 0·(−∞) form.
Rewrite: lim_{x→0⁺} ln(x)/(1/x) — form −∞/∞. Apply L'Hôpital:
lim = lim (1/x)/(−1/x²) = lim (−x) = 0.
So ln L = 0 → L = e⁰ = 1. Therefore lim_{x→0⁺} x^x = 1.
Example 4 — Taylor Remainder Estimation
Estimate |e − (1 + 1 + 1/2! + 1/3! + 1/4!)| using Lagrange remainder.
f(x) = eˣ. Taylor expansion at x₀=0, evaluated at x=1 with n=4 terms:
e ≈ 1 + 1 + 1/2 + 1/6 + 1/24 = 2.7083...
Lagrange remainder: R₄(1) = f⁽⁵⁾(c)/5! = eᶜ/120 for some c ∈ (0,1).
Since eᶜ < e < 3, |R₄(1)| < 3/120 = 1/40 = 0.025.
So |e − 2.7083| < 0.025. (Actual error ≈ 0.0085 — within our bound.) □
Example 5 — CSIR NET Style
If f is differentiable and f'(x) = 0 for all x ∈ (a,b), what can you conclude about f?
Conclusion: f is constant on (a,b).
Proof: Take any x, y ∈ (a,b) with x < y. By MVT, ∃c ∈ (x,y) ⊂ (a,b) such that
f(y) − f(x) = f'(c)(y − x) = 0 · (y − x) = 0.
So f(y) = f(x) for all x, y ∈ (a,b). Hence f is constant on (a,b). □
Important: This is one of the most-tested consequences of MVT in CSIR NET.

Question 2 of 15
The function f(x) = |x| at x = 0 is:
✓ C is correct. |x| is continuous at 0 (|x|→0). But: Left derivative = lim_{h→0⁻} |h|/h = −1. Right derivative = lim_{h→0⁺} |h|/h = +1. Since left ≠ right, f'(0) does not exist — not differentiable at 0. This is the canonical example of continuity ≠ differentiability.
Question 3 of 15
Rolle's Theorem requires which three conditions on f in [a,b]?
✓ B is correct. Rolle's three conditions: (1) f continuous on the CLOSED interval [a,b], (2) f differentiable on the OPEN interval (a,b), (3) f(a) = f(b) (equal values at endpoints — NOT necessarily zero). Note: differentiability is only required on the open interval; the endpoints need only continuity.
Question 4 of 15
The MVT states: if f is continuous on [a,b] and differentiable on (a,b), then:
✓ B is correct. MVT: ∃c ∈ (a,b) such that f'(c) = [f(b)−f(a)]/(b−a) (the derivative at c equals the average rate of change, i.e., the secant slope). A is the IVT (intermediate value). C is generally false. D doesn't capture MVT.

Question 6 of 15
If f'(x) > 0 for all x ∈ (a,b), then by MVT, f is:
✓ A is correct. For x < y in (a,b), MVT gives ∃c: f(y)−f(x) = f'(c)(y−x). Since f'(c) > 0 and y−x > 0, we get f(y) > f(x). So f is strictly increasing. B requires f' = 0. C requires f'' < 0. D is wrong (f(x) = ln(x) is unbounded on (0,1) despite positive derivative).
Question 7 of 15
The Lagrange remainder Rₙ(x) in Taylor's theorem satisfies:
✓ C is correct. Lagrange remainder: Rₙ(x) = f^(n+1)(c) · (x−x₀)^(n+1) / (n+1)! where c lies between x and x₀ (c depends on x and n). This allows error estimation: |Rₙ(x)| ≤ M·|x−x₀|^(n+1)/(n+1)! where M bounds |f^(n+1)| on the interval.
Question 8 of 15
lim_{x→∞} x/eˣ equals:
✓ A is correct. This is ∞/∞ form. L'Hôpital: lim x/eˣ = lim 1/eˣ = 0. More generally, eˣ grows faster than any polynomial: lim x^n/eˣ = 0 for all n (apply L'Hôpital n times). This is one of the standard limit hierarchy: ln(x) ≪ x^α ≪ eˣ as x → ∞ for any α > 0.
Question 9 of 15
Cauchy's Mean Value Theorem generalizes MVT by involving two functions f and g. Its conclusion is:
✓ B is correct. Cauchy's MVT: ∃c ∈ (a,b): [f(b)−f(a)]/[g(b)−g(a)] = f'(c)/g'(c). This reduces to ordinary MVT when g(x) = x. Cauchy's MVT is the key tool for proving L'Hôpital's rule for the 0/0 case.
Question 10 of 15
The function f(x) = x³ on [−1, 1]. Which point c satisfies the MVT?
✓ C is correct. MVT: f'(c) = [f(1)−f(−1)]/(1−(−1)) = [1−(−1)]/2 = 1. f'(x) = 3x². So 3c² = 1 ⟹ c² = 1/3 ⟹ c = ±1/√3 ≈ ±0.577. Both lie in (−1,1). A gives f'(0) = 0 ≠ 1. B gives f'(1/2) = 3/4 ≠ 1. D gives f'(1/2) = 3/4 ≠ 1.
Question 11 of 15
Which of the following is an indeterminate form that CANNOT be directly handled by L'Hôpital?
✓ C is correct. 1/0 is NOT an indeterminate form — it equals ±∞ (a definite divergence). L'Hôpital only applies to 0/0 or ∞/∞. Forms 0·∞, ∞−∞, 1^∞, 0⁰, ∞⁰ are all indeterminate but require conversion before applying L'Hôpital. The form 1/0 tells you the limit is ±∞, no indeterminacy involved.
Question 12 of 15
If f'(x) = 0 for all x in an interval (a,b), then on (a,b):
✓ A is correct. By MVT: for any x < y in (a,b), ∃c with f(y)−f(x) = f'(c)(y−x) = 0·(y−x) = 0. So f(y) = f(x) for all x, y — f is constant. C is wrong: f could be a non-zero constant (e.g., f(x) = 5). D follows but A is stronger. This is one of the most fundamental MVT consequences.
Question 13 of 15
The second-order Taylor polynomial of f(x) = cos(x) at x₀ = 0 is:
✓ B is correct. cos(x) = 1 − x²/2! + x⁴/4! − … = 1 − x²/2 + x⁴/24 − … The second-order polynomial (n=2) includes terms up to x²: T₂(x) = 1 − x²/2. A is wrong (includes −x, but cos'(0)=−sin(0)=0). C includes the 4th degree term (that's T₄). D is the expansion of eˣ.
Question 14 of 15

✓ B is correct. [f(4)−f(1)]/(4−1) = (2−1)/3 = 1/3. f'(x) = 1/(2√x). Set 1/(2√c) = 1/3 ⟹ 2√c = 3 ⟹ √c = 3/2 ⟹ c = 9/4 = 2.25 ∈ (1,4). ✓
Question 15 of 15
The Weierstrass function (limit of sum of cos(n·3^n·πx) type terms) is a famous example of:

📌 Summary & CSIR NET Tips

Derivative Basics

  • f'(c) = lim_{h→0} [f(c+h)−f(c)]/h
  • Differentiable ⟹ Continuous (not converse)
  • |x| is cont. but not diff. at 0
  • x²sin(1/x): diff. at 0, f'(0)=0
  • Weierstrass: cont. everywhere, diff. nowhere

MVT Family

  • Rolle's: f(a)=f(b) ⟹ ∃c: f'(c)=0
  • MVT: ∃c: f'(c)=[f(b)−f(a)]/(b−a)
  • Cauchy: ratio of increments = ratio of derivatives
  • f'=0 on interval ⟹ f constant
  • |f'|≤M ⟹ Lipschitz with const. M

L'Hôpital's Rule

  • Applies to 0/0 and ∞/∞ only
  • Convert 0·∞, ∞−∞, 1^∞, 0⁰, ∞⁰
  • lim xⁿ/eˣ = 0 as x→∞ for all n
  • lim x·ln x = 0 as x→0⁺
  • lim (1+1/x)^x = e as x→∞

Taylor & CSIR Tips

  • Lagrange remainder: f^(n+1)(c)·(x−x₀)^(n+1)/(n+1)!
  • Know sin, cos, eˣ, ln(1+x) series by heart
  • MVT: "f'=0 ⟹ constant" is exam-tested
  • Rolle's used to prove uniqueness of roots
  • Taylor remainder bounds for error estimation
