Unit 1 — Analysis · Topic 08 of 11
Riemann Integration & Improper Integrals
Darboux sums, Riemann integrability criterion, FTC, and convergence of improper integrals via comparison tests
CSIR NET Core Integration Theory Intermediate 75 min Study

Learning Objectives

  1. Define Darboux upper and lower sums and the Riemann integral
  2. State the Riemann integrability criterion: f is integrable iff oscillation can be made small
  3. Identify which functions are Riemann integrable (continuous, monotone, bounded with finite discontinuities)
  4. State and apply both parts of the Fundamental Theorem of Calculus
  5. Evaluate and test convergence of improper integrals using comparison, limit comparison, and p-test
🎯
CSIR NET Frequency: Improper integrals appear in 2–3 questions per exam. Key: ∫₁^∞ 1/xᵖ dx, ∫₀^1 1/xᵖ dx convergence conditions (p<1 and p<1 respectively), Gamma function Γ(n) = (n−1)!, comparison test applications.

⊢ Partitions & Darboux Sums

∑
Concept 1
Partitions and Darboux Upper/Lower Sums
Partition
A partition P of [a,b] is a finite set {x₀, x₁, …, xₙ} with a = x₀ < x₁ < … < xₙ = b.

The mesh (norm) of P: ‖P‖ = max₁≤ᵢ≤ₙ (xᵢ − xᵢ₋₁) = length of the largest subinterval.
Darboux Upper and Lower Sums
For f : [a,b] → ℝ bounded and partition P:
Mᵢ = sup_{x∈[xᵢ₋₁,xᵢ]} f(x),   mᵢ = inf_{x∈[xᵢ₋₁,xᵢ]} f(x)

Upper sum: U(f,P) = Σ Mᵢ·(xᵢ − xᵢ₋₁)
Lower sum: L(f,P) = Σ mᵢ·(xᵢ − xᵢ₋₁)

Always: L(f,P) ≤ ∫f ≤ U(f,P) for any partition P.

∫ Riemann Integrability

R
Concept 2
Riemann Integral and Integrability Criterion
Riemann Integrability
A bounded function f : [a,b] → ℝ is Riemann integrable if:

Upper integral = inf_P U(f,P) = Lower integral = sup_P L(f,P)

The common value is ∫_a^b f(x) dx. Equivalently: ∀ε > 0, ∃ partition P: U(f,P) − L(f,P) < ε.
Key Integrability Theorems
1. Continuous ⟹ Integrable: If f is continuous on [a,b], then f is Riemann integrable.
2. Monotone ⟹ Integrable: If f is monotone (increasing or decreasing) on [a,b], then f is Riemann integrable.
3. Bounded + Finite Discontinuities ⟹ Integrable: If f is bounded on [a,b] and has only finitely many discontinuities, then f is Riemann integrable.
4. Lebesgue Criterion: f is Riemann integrable iff f is bounded and the set of discontinuities has measure zero.

Properties of Riemann Integral
If f, g are Riemann integrable on [a,b]:
1. αf + βg is integrable: ∫(αf+βg) = α∫f + β∫g (linearity)
2. fg is integrable; |f| is integrable
3. |∫_a^b f| ≤ ∫_a^b |f| (triangle inequality)
4. ∫_a^b f = ∫_a^c f + ∫_c^b f for any c ∈ [a,b]
5. If f ≤ g on [a,b], then ∫f ≤ ∫g (monotonicity)
6. Mean Value Theorem for Integrals: ∃c ∈ (a,b): ∫_a^b f(x)dx = f(c)(b−a)

⟺ Fundamental Theorem of Calculus

FTC
Concept 3
Both Parts of the FTC
FTC — Part 1 (Antiderivative)
Let f be Riemann integrable on [a,b] and define F(x) = ∫_a^x f(t) dt for x ∈ [a,b].

Then F is continuous on [a,b]. Furthermore, if f is continuous at c ∈ [a,b], then F is differentiable at c with F'(c) = f(c).
F'(c) = lim_{h→0} [F(c+h)−F(c)]/h = lim_{h→0} (1/h)∫_c^{c+h} f(t) dt = f(c) (by continuity of f at c).
FTC — Part 2 (Evaluation)
If f is continuous on [a,b] and F is any antiderivative of f (i.e., F' = f), then:

∫_a^b f(x) dx = F(b) − F(a) = [F(x)]_a^b
This is the link between differentiation and integration — the two fundamental operations of calculus. It allows evaluation of definite integrals via antiderivatives.

∞ Improper Integrals

∞
Concept 4
Types and Definitions of Improper Integrals
Type I — Infinite Limits
∫_a^∞ f(x) dx = lim_{b→∞} ∫_a^b f(x) dx (if limit exists and is finite → converges)

∫_{-∞}^∞ f(x) dx = lim_{a→−∞, b→∞} ∫_a^b f(x) dx
(Note: must check BOTH limits separately)
Type II — Unbounded Integrand
If f(x) → ±∞ as x → c (a singularity at c ∈ [a,b]):

∫_a^c f(x) dx = lim_{ε→0⁺} ∫_a^{c−ε} f(x) dx    (left-sided singularity)

🧪 Convergence Tests for Improper Integrals

⚖
Concept 5
Comparison, Limit Comparison, Absolute Convergence
Comparison Test for Improper Integrals
If 0 ≤ f(x) ≤ g(x) for x ≥ a:
• ∫_a^∞ g(x) dx converges ⟹ ∫_a^∞ f(x) dx converges
• ∫_a^∞ f(x) dx diverges ⟹ ∫_a^∞ g(x) dx diverges
Limit Comparison Test
If f, g > 0 and lim_{x→∞} f(x)/g(x) = L (with 0 < L < ∞), then ∫f and ∫g either both converge or both diverge.
Absolute vs Conditional Convergence
∫_a^∞ f(x) dx is absolutely convergent if ∫|f| converges.
∫_a^∞ f(x) dx is conditionally convergent if ∫f converges but ∫|f| diverges.

Absolute convergence ⟹ convergence (same as series).
Example: ∫₀^∞ sin(x)/x dx is conditionally convergent (∫|sin(x)/x| diverges).

Convergence Test Strategy

StepAction
1Identify the singularity type: at ∞ or at a finite point
2Try to compute the integral directly (antiderivative)
3If not directly computable, try comparison with a simpler function
4For "similar growth" use limit comparison; near singularity compare with 1/xᵖ
5Check both absolute convergence AND conditional convergence if needed

✏️ Worked Examples

Example 1 — Darboux Sum Computation
For f(x) = x on [0,1] with partition P = {0, 1/3, 2/3, 1}, compute U(f,P) and L(f,P).
On [0, 1/3]: M₁ = 1/3, m₁ = 0. On [1/3, 2/3]: M₂ = 2/3, m₂ = 1/3. On [2/3, 1]: M₃ = 1, m₃ = 2/3.
U(f,P) = (1/3)(1/3) + (2/3)(1/3) + (1)(1/3) = (1/9 + 2/9 + 3/9) = 6/9 = 2/3.
L(f,P) = (0)(1/3) + (1/3)(1/3) + (2/3)(1/3) = (0 + 1/9 + 2/9) = 3/9 = 1/3.
True value: ∫₀¹ x dx = 1/2. Check: 1/3 < 1/2 < 2/3. ✓
Gap: U − L = 1/3 → 0 as partition is refined.
Example 2 — FTC Application
Let F(x) = ∫₀^x sin(t²) dt. Find F'(x).
By FTC Part 1, since f(t) = sin(t²) is continuous, F'(x) = f(x) = sin(x²).
More complex version: G(x) = ∫₀^{x²} sin(t) dt.
By chain rule: G'(x) = sin(x²) · d/dx(x²) = sin(x²) · 2x.
Example 3 — Improper Integral Type I
Determine whether ∫₁^∞ 1/(x² + 1) dx converges.
∫₁^b 1/(x²+1) dx = [tan⁻¹(x)]₁^b = tan⁻¹(b) − tan⁻¹(1) = tan⁻¹(b) − π/4.
As b → ∞: tan⁻¹(b) → π/2. So ∫₁^∞ = π/2 − π/4 = π/4.
Converges to π/4.
Example 4 — Comparison Test
Does ∫₁^∞ (sin²x)/x² dx converge?
Note: 0 ≤ sin²x ≤ 1 for all x. So 0 ≤ sin²(x)/x² ≤ 1/x².
∫₁^∞ 1/x² dx = [−1/x]₁^∞ = 1 (converges).
By comparison test: 0 ≤ sin²(x)/x² ≤ 1/x² and ∫1/x² converges ⟹ ∫sin²(x)/x² converges. □
Example 5 — Improper Type II (Singularity at 0)
Evaluate ∫₀^1 1/√x dx.
f(x) = 1/√x = x^{−1/2} has a singularity at x = 0. This is p-type with p = 1/2 < 1 → should converge.
∫₀^1 x^{−1/2} dx = lim_{ε→0⁺} ∫_ε^1 x^{−1/2} dx = lim_{ε→0⁺} [2√x]_ε^1 = lim_{ε→0⁺} (2 − 2√ε) = 2.
Converges to 2.

Question 2 of 15
The Dirichlet function f(x) = 1 (x∈ℚ), f(x) = 0 (x∉ℚ) on [0,1] is:
✓ C is correct. For any partition P: in each subinterval, both rationals and irrationals are dense. So Mᵢ = 1 (sup = 1) and mᵢ = 0 (inf = 0) for every subinterval. Hence U(f,P) = 1 and L(f,P) = 0 for every P. So U−L = 1 ≠ 0 — not integrable. (However, it is Lebesgue integrable with value 0.)
Question 3 of 15
The integral ∫₁^∞ 1/xᵖ dx converges iff:
✓ C is correct. ∫₁^b x^{−p} dx = [x^{1−p}/(1−p)]₁^b (p≠1). As b→∞: if p>1, then 1−p<0, so b^{1−p}→0 → integral = 1/(p−1). If p<1, then 1−p>0, so b^{1−p}→∞ → diverges. If p=1, ∫₁^∞ 1/x dx = ln(b)→∞ → diverges. So converges iff p>1.

Question 6 of 15
If f is continuous on [a,b], then the Mean Value Theorem for Integrals states:
✓ B is correct. MVT for Integrals: if f is continuous on [a,b], ∃c ∈ (a,b) such that ∫_a^b f(x)dx = f(c)(b−a). Geometrically: there is a rectangle with base [a,b] and height f(c) with the same area as the region under f. Proof: use IVT on F(x) = ∫_a^x f(t)dt, then MVT on F gives F'(c) = f(c) = (F(b)−F(a))/(b−a).
Question 7 of 15
∫₀^∞ e^{−x²} dx equals:
✓ C is correct. The Gaussian integral: ∫₋∞^∞ e^{−x²} dx = √π (proved by squaring and converting to polar coordinates). So ∫₀^∞ e^{−x²} dx = √π/2 (by symmetry). This is related to the Gamma function: Γ(1/2) = √π. This integral comes up frequently in probability (normal distribution) and CSIR NET.
Question 8 of 15
The integral ∫₀^1 1/xᵖ dx converges iff:
✓ A is correct. ∫_ε^1 x^{−p} dx = [x^{1−p}/(1−p)]_ε^1 (p≠1). As ε→0⁺: if p<1, then 1−p>0, ε^{1−p}→0 → converges; value = 1/(1−p). If p>1, ε^{1−p}→∞ → diverges. If p=1, ∫_ε^1 1/x dx = −ln ε → ∞ → diverges. Note: REVERSED compared to ∫₁^∞ case!
Question 9 of 15
Which function is NOT Riemann integrable on [0,1]?
✓ C is correct. The Dirichlet function is discontinuous EVERYWHERE on [0,1]. By Lebesgue's criterion, R-integrable iff set of discontinuities has measure zero. Here, the set of all rationals in [0,1] is dense (not measure zero — wait, it IS measure zero since ℚ is countable). Actually, by Lebesgue criterion: Dirichlet is NOT R-integrable because U=1, L=0 always, not because of measure — it's the uncountably many irrational discontinuities... Question 10 of 15
The Gamma function Γ(n) = ∫₀^∞ x^{n−1}e^{−x} dx satisfies:
✓ B is correct. Γ satisfies the functional equation Γ(n+1) = n·Γ(n) (proved by integration by parts). With Γ(1) = ∫₀^∞ e^{−x}dx = 1, we get Γ(n) = (n−1)! for positive integers. A is wrong: Γ(n) = (n−1)! not n!. C is wrong: Γ(1/2) = √π (related to Gaussian integral). D is wrong: Γ extends to all positive reals (and beyond).
Question 11 of 15
To show ∫₁^∞ 1/(x² + √x) dx converges, the best comparison is:
✓ B is correct. For x ≥ 1: x² + √x ≥ x², so 1/(x²+√x) ≤ 1/x². Since ∫₁^∞ 1/x² dx = 1 (converges, p=2>1), by comparison test ∫₁^∞ 1/(x²+√x) dx converges. Alternatively, limit comparison with 1/x²: lim x²/(x²+√x) = 1 (finite nonzero), both behave the same.
Question 12 of 15
∫₀^∞ sin(x)/x dx (Dirichlet integral) is:
✓ B is correct. ∫₀^∞ sin(x)/x dx = π/2 (Dirichlet integral, classical result). However, ∫₀^∞ |sin(x)/x| dx diverges (since |sin x|/x ≥ C/n on each interval [nπ,(n+1)π] and ∑1/n diverges). So this integral is CONDITIONALLY convergent — it converges but not absolutely.
Question 13 of 15
If f is bounded on [a,b] and f has finitely many discontinuities, then:
✓ A is correct. A bounded function with finitely many discontinuities on [a,b] is Riemann integrable. The finite discontinuity set has measure zero (Lebesgue criterion). One can show that around each discontinuity, one can choose a small interval contributing arbitrarily small amount to U−L.
Question 14 of 15
The antiderivative F(x) = ∫_a^x f(t) dt is always:
✓ B is correct. FTC Part 1: F is always continuous on [a,b] (even if f is just integrable, not continuous). F'(x) = f(x) at points where f is continuous. At discontinuities of f, F may still be differentiable (if the limit exists) but it's not guaranteed. The minimum guarantee is: F continuous everywhere, F' = f wherever f is continuous.
Question 15 of 15
∫₀^∞ x^{n−1}e^{−x} dx = Γ(n). For n = 1/2, Γ(1/2) equals:
✓ C is correct. Γ(1/2) = ∫₀^∞ x^{−1/2}e^{−x} dx. Substitute x = t²: dx = 2t dt. Integral becomes ∫₀^∞ t^{−1}e^{−t²}·2t dt = 2∫₀^∞ e^{−t²} dt = 2·(√π/2) = √π. So Γ(1/2) = √π. This is a key value: Γ(3/2) = (1/2)·Γ(1/2) = √π/2, Γ(5/2) = (3/2)·√π/2 = 3√π/4, etc.

📌 Summary & CSIR NET Tips

Riemann Integral

  • L(f,P) ≤ ∫f ≤ U(f,P) always
  • Integrable iff U−L can be made <ε
  • Continuous ⟹ integrable
  • Monotone ⟹ integrable
  • Dirichlet function: NOT integrable
  • Lebesgue: integrable iff discon. set has measure 0

FTC

  • Part 1: F(x)=∫_a^x f → F'=f where f continuous
  • Part 2: ∫_a^b f = F(b)−F(a) if F'=f
  • MVT for Integrals: ∫f = f(c)(b−a) for some c
  • Chain rule + FTC: d/dx[∫_a^{g(x)} f] = f(g(x))g'(x)

Improper Integrals

  • ∫₁^∞ 1/xᵖ: converges iff p>1
  • ∫₀^1 1/xᵖ: converges iff p<1
  • Γ(n) = ∫₀^∞ x^{n-1}e^{-x}dx = (n-1)!
  • Γ(1/2) = √π
  • ∫₀^∞ e^{-x²}dx = √π/2

CSIR Exam Tips

  • p-test: most common improper integral question
  • Comparison test: 1/(x²+√x) ≤ 1/x²
  • Dirichlet integral ∫sin(x)/x: conditional (π/2)
  • Gamma function recurrence tested frequently
  • FTC Part 2 evaluation: most computation questions
