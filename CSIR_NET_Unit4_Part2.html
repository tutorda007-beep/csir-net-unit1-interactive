Argument Principle Intermediate

If f meromorphic inside C (closed contour), N = zeros (counted with multiplicity), P = poles (counted with multiplicity): (1/2ŌÄi) ‚ąģ_C f'(z)/f(z) dz = N ‚ąí P Equivalently: (1/2ŌÄ) őĒ_C arg f(z) = N ‚ąí P Winding number interpretation: As z traverses C once, f(z) winds N‚ąíP times around the origin. Rouch√©'s Theorem Basic

If f and g are analytic inside and on C, and |g(z)| < |f(z)| for all z on C, then f and f+g have the same number of zeros inside C (counted with multiplicity). Mnemonic: "the big dog dominates ‚ÄĒ adding a small perturbation doesn't change the zero count."
