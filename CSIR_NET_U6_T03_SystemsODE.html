Introduction

Systems of first-order ODEs x′=Ax+b arise from higher-order ODEs and from physical systems with multiple interacting components. The eigenvalue structure of A completely determines the behavior of solutions.

Matrix Form x′ = Ax

Linear System

A first-order linear system: x′ = Ax + b(t), where x∈ℝⁿ, A is n×n matrix, b(t) is a vector function.

Homogeneous: x′=Ax. Solution space is n-dimensional vector space.

General solution: x(t) = xₕ(t) + x_p(t).

Converting Higher-Order to System

y^(n) + aₙ₋₁y^(n−1) + … + a₀y = 0: let x₁=y, x₂=y′, …, xₙ=y^(n−1). Then:

x′ = [0 1 0 … 0; 0 0 1 … 0; ⋮; −a₀ −a₁ … −aₙ₋₁] x (companion matrix)
Fundamental Matrix

A fundamental matrix Φ(t) has n linearly independent solution columns. det Φ ≠ 0. General solution: x(t) = Φ(t)c for constant vector c.

Wronskian = det Φ satisfies W′ = tr(A)·W (analogue of Abel's formula).

Eigenvalue Method

Eigenvalue Solutions

For x′=Ax, try x=e^(λt)v where Av=λv (v is eigenvector of A).

Eigenvalue typeSolution
Real distinct λ₁≠λ₂ with eigenvectors v₁,v₂c₁e^(λ₁t)v₁ + c₂e^(λ₂t)v₂
Complex λ=α+βi with eigenvector v=p+qie^(αt)[c₁(p cos βt − q sin βt) + c₂(p sin βt + q cos βt)]
Repeated λ, deficient (Jordan block)e^(λt)v + te^(λt)w, where (A−λI)w=v (generalized eigenvector)
Example: 2×2 System

x′=[3 1; 0 3]x. λ=3 (double). Eigenvector: (A−3I)v=0 → v=[1;0]. Generalized: (A−3I)w=v → w=[0;1]. Solution: x=e^(3t)(c₁[1;0]+c₂([0;1]+t[1;0])).

Matrix Exponential

Definition
e^(At) = I + At + (At)²/2! + (At)³/3! + … = Σ (At)ⁿ/n!

The unique solution to x′=Ax, x(0)=x₀ is x(t)=e^(At)x₀.

Computing e^(At)
  • Diagonalizable A=PDP⁻¹: e^(At) = Pe^(Dt)P⁻¹ where e^(Dt)=diag(e^(λ₁t),…,e^(λₙt)).
  • Jordan form A=PJP⁻¹: e^(Jt) is block diagonal with e^(λt)[I+Nt+N²t²/2!+…] per block (N = nilpotent part).
  • Cayley-Hamilton: e^(At) can be expressed as polynomial in A of degree ≤ n−1.
Example: Diagonal A

A=diag(2,−1). e^(At)=diag(e^(2t),e^(−t)). x₀=(x₀,y₀)ᵀ: x(t)=(x₀e^(2t), y₀e^(−t))ᵀ.

Properties of Matrix Exponential
  • d/dt e^(At) = Ae^(At) = e^(At)A
  • e^(A·0) = I
  • e^(A(s+t)) = e^(As)e^(At) (when A commutes with itself)
  • If AB=BA: e^(A+B) = e^A e^B
  • det(e^A) = e^(tr A)

Phase Portraits & Stability

For x′=Ax in 2D, the phase portrait near the origin (equilibrium) is classified by the eigenvalues of A:

EigenvaluesTypeStability
Real, distinct, same sign (−)Stable nodeAsymptotically stable
Real, distinct, same sign (+)Unstable nodeUnstable
Real, opposite signsSaddle pointUnstable
Complex α±βi, α<0Stable spiralAsymptotically stable
Complex α±βi, α>0Unstable spiralUnstable
Pure imaginary ±βiCenterStable (not asympt.)
Repeated real λ<0 (deficient)Stable degenerate nodeAsymptotically stable
Lyapunov Stability

Equilibrium x*=0 is stable if nearby solutions stay near. Asymptotically stable if they also converge to 0. For linear systems: stable iff all eigenvalues have non-positive real part (and purely imaginary eigenvalues are semi-simple).

Summary

x′=Ax

n-dim solution space. Fundamental matrix Φ: det Φ≠0. General solution: x=Φ(t)c.

Eigenvalue Method

Av=λv ⟹ x=e^(λt)v. Real/complex/repeated eigenvalues give distinct solution forms.

Matrix Exponential

e^(At)=Σ(At)ⁿ/n!. Solution: x(t)=e^(At)x₀. det(e^A)=e^(tr A).

Phase Portraits

Classified by Re(λ) and Im(λ): nodes, spirals, saddle, center. Stable iff all Re(λ)<0.

Stability

Asympt. stable ↔ all Re(λᵢ)<0. Stable ↔ Re(λᵢ)≤0 and simple imaginary.

Jordan Blocks

Repeated eigenvalue, deficient matrix: use generalized eigenvectors; solution involves te^(λt) factors.
