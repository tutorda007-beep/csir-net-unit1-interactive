What Are Metric Spaces?

A metric space is a set X equipped with a distance function d: X×X → ℝ that captures the notion of "how far apart" two points are. This powerful abstraction unifies ℝⁿ, function spaces, sequence spaces, and discrete sets under one framework.

Why Metric Spaces?

  • Unify analysis on ℝ, ℝⁿ, and abstract spaces
  • Define continuity without coordinates
  • Prove existence theorems (fixed points)
  • Foundation for functional analysis

CSIR NET Importance

  • 2–4 questions per paper on this topic
  • Completeness: Baire category, Cantor intersection
  • Compactness: sequential compactness equivalence
  • Contraction mapping: fixed point uniqueness

Metric Spaces: Definitions & Examples

Definition — Metric Space

A metric space is a pair (X, d) where X is a non-empty set and d: X×X → ℝ satisfies for all x, y, z ∈ X:

  1. Non-negativity: d(x,y) ≥ 0, with d(x,y) = 0 ⟺ x = y
  2. Symmetry: d(x,y) = d(y,x)
  3. Triangle inequality: d(x,z) ≤ d(x,y) + d(y,z)

Subspaces

Definition

If (X,d) is a metric space and Y ⊆ X, then (Y, d|_{Y×Y}) is a metric subspace. The restricted metric inherits all metric axioms.

Key Fact

A subspace of a complete metric space need not be complete. Example: ℚ ⊂ ℝ is not complete since (1, 1.4, 1.41, 1.414, ...) → √2 ∉ ℚ. However, a closed subspace of a complete metric space is complete.

Equivalent Metrics

Two metrics d₁ and d₂ on X are equivalent if they generate the same topology (same open sets). On ℝⁿ, the Euclidean, taxicab, and sup-norm metrics are all equivalent.

d₁, d₂ equivalent ⟺ ∃ c₁,c₂>0: c₁·d₁(x,y) ≤ d₂(x,y) ≤ c₂·d₁(x,y) for all x,y

Diameter and Bounded Sets

Definition

diam(A) = sup{d(x,y) : x,y ∈ A}. A set A is bounded if diam(A) < ∞.

Open and Closed Sets

Definition — Open Ball

The open ball of radius r > 0 centered at x ∈ X is: B(x,r) = {y ∈ X : d(x,y) < r}

The closed ball: B̄(x,r) = {y ∈ X : d(x,y) ≤ r}

Definition — Open/Closed Sets
  • Open set: G ⊆ X is open if ∀ x ∈ G, ∃ r > 0 such that B(x,r) ⊆ G
  • Closed set: F ⊆ X is closed if X \ F is open, equivalently if F contains all its limit points

Properties of Open/Closed Sets

Open Sets

  • ∅ and X are open
  • Arbitrary unions of open sets are open
  • Finite intersections of open sets are open
  • Every open ball is open

Closed Sets

  • ∅ and X are closed
  • Arbitrary intersections of closed sets are closed
  • Finite unions of closed sets are closed
  • Every closed ball is closed

Characterization of Closure

Dense and Nowhere Dense Sets

Definitions
  • Dense: A ⊆ X is dense if cl(A) = X (every open ball meets A)
  • Nowhere dense: A is nowhere dense if int(cl(A)) = ∅ (closure has empty interior)
  • Separable: X is separable if it has a countable dense subset (e.g., ℚ is dense in ℝ)

Convergence and Cauchy Sequences

Definition — Convergence

A sequence (xₙ) in (X,d) converges to x ∈ X if: ∀ ε > 0, ∃ N ∈ ℕ such that n ≥ N ⟹ d(xₙ, x) < ε

Write: xₙ → x or lim xₙ = x

Properties of Limits
  • Limits in metric spaces are unique (by triangle inequality)
  • Convergent sequences are bounded
  • xₙ → x ⟺ d(xₙ, x) → 0 (in ℝ)
  • A is closed ⟺ xₙ ∈ A, xₙ → x implies x ∈ A
Definition — Cauchy Sequence

A sequence (xₙ) in (X,d) is Cauchy if: ∀ ε > 0, ∃ N such that m,n ≥ N ⟹ d(xₙ, xₘ) < ε

Key Relationship

Every convergent sequence is Cauchy. The converse is NOT always true — it holds if and only if X is complete.

  • In ℝ: Cauchy ⟺ convergent (ℝ is complete)
  • In ℚ: Cauchy ⇏ convergent (e.g., rational approximations to √2 are Cauchy but converge to irrational)

Subsequences

Theorem
  • If xₙ → x, then every subsequence also converges to x
  • If a Cauchy sequence has a convergent subsequence xₙₖ → x, then the full sequence xₙ → x

Continuity via Sequences

Sequential Characterization

f: (X,d_X) → (Y,d_Y) is continuous at x₀ ⟺ xₙ → x₀ implies f(xₙ) → f(x₀)

f is continuous ⟺ f⁻¹(G) is open in X for every open G ⊆ Y

f is continuous ⟺ f⁻¹(F) is closed in X for every closed F ⊆ Y

Common Error

On ℝⁿ (n ≥ 2), a sequence (xₙ) in ℝⁿ converges ⟺ each coordinate sequence converges. But there's no natural ordering — limit points in ℝⁿ come from all directions.

Complete Metric Spaces

Definition — Completeness

A metric space (X,d) is complete if every Cauchy sequence in X converges to a limit in X.

Theorem — Closed Subsets of Complete Spaces

If (X,d) is complete and F ⊆ X is closed, then (F,d) is complete.

Conversely, if Y ⊆ X is a complete subspace, then Y is closed in X.

Cantor's Intersection Theorem

Let (X,d) be a complete metric space. Let F₁ ⊇ F₂ ⊇ F₃ ⊇ ... be a nested sequence of non-empty closed sets with diam(Fₙ) → 0. Then ⋂Fₙ consists of exactly one point.

Proof Sketch

Pick xₙ ∈ Fₙ. For m,n ≥ N, xₘ,xₙ ∈ Fₙ, so d(xₘ,xₙ) ≤ diam(Fₙ) → 0. Thus (xₙ) is Cauchy. By completeness xₙ → x. Since Fₙ is closed, x ∈ Fₙ for all n, so x ∈ ⋂Fₙ. Uniqueness follows from diam(Fₙ) → 0.

Baire Category Theorem

Baire Category Theorem

A complete metric space cannot be written as a countable union of nowhere dense sets. Equivalently: a countable intersection of open dense sets is dense.

Applications of Baire Category Theorem
  • ℝ is uncountable (each singleton {x} is nowhere dense)
  • There exists a continuous function nowhere differentiable
  • Banach-Steinhaus theorem (uniform boundedness principle)

Completion of a Metric Space

Theorem — Completion

Every metric space (X,d) has a unique (up to isometry) completion (X̃, d̃) — a complete metric space containing X as a dense subspace. Example: ℝ is the completion of ℚ.

Compact Metric Spaces

Definition — Compactness

(X,d) is compact if every open cover has a finite subcover: if X ⊆ ⋃_{α} Gα (each Gα open), then there exist finitely many α₁,...,αₙ with X ⊆ Gα₁ ∪ ... ∪ Gαₙ.

Equivalent Characterizations

Theorem — Equivalences in Metric Spaces

For a metric space (X,d), the following are equivalent:

  1. (X,d) is compact (every open cover has finite subcover)
  2. (X,d) is sequentially compact (every sequence has a convergent subsequence)
  3. (X,d) is complete and totally bounded (∀ε>0, X is covered by finitely many ε-balls)
  4. Every infinite subset of X has a limit point in X

Totally Bounded Sets

Definition

A set A ⊆ X is totally bounded if for every ε > 0, there exist finitely many points x₁,...,xₙ ∈ X such that A ⊆ ⋃ B(xᵢ,ε). Every totally bounded set is bounded, but not conversely (in infinite dimensions).

Important Contrast

In ℝⁿ: compact ⟺ closed and bounded (Heine-Borel). In infinite-dimensional normed spaces (like C[a,b]): closed and bounded does NOT imply compact. The closed unit ball in C[0,1] is not compact.

Properties of Compact Spaces

  • Compact spaces are complete and totally bounded
  • Closed subsets of compact spaces are compact
  • Compact subsets of metric spaces are closed and bounded
  • Continuous image of a compact space is compact
  • A continuous function on a compact space is uniformly continuous (Heine's theorem)
  • A continuous real-valued function on a compact space attains its max and min (EVT)

Connectedness

Definition — Connected Space

(X,d) is connected if it cannot be written as the union of two disjoint non-empty open sets. Equivalently, the only subsets of X that are both open and closed (clopen) are ∅ and X.

Definition — Path-Connected

(X,d) is path-connected if for every x,y ∈ X, there exists a continuous γ: [0,1] → X with γ(0) = x and γ(1) = y.

Path-connected ⟹ Connected. The converse is false (topologist's sine curve is connected but not path-connected).

Connected Subsets of ℝ

Theorem

A subset E ⊆ ℝ is connected ⟺ E is an interval (bounded, unbounded, open, closed, or half-open).

Properties

Preserved Under Maps

  • Continuous image of connected set is connected
  • Continuous image of path-connected set is path-connected
  • This proves IVT: continuous f: [a,b] → ℝ, f(a) < c < f(b) ⟹ ∃ x with f(x)=c

Connected Components

  • Every point belongs to a unique maximal connected subset called its connected component
  • Connected components are closed
  • Discrete metric: every singleton is a component
Key Theorem

If {Aα} is a family of connected sets with a common point (⋂ Aα ≠ ∅), then ⋃ Aα is connected.

The closure of a connected set is connected. The product of connected sets is connected.

Discrete Metric Space

On any set X with the discrete metric d(x,y) = 0 (x=y), 1 (x≠y): every singleton is open and closed. So X is connected ⟺ X has exactly one point. X is always complete and totally bounded only if finite. X is compact ⟺ X is finite.

Banach Fixed Point Theorem

Definition — Contraction Mapping

T: (X,d) → (X,d) is a contraction if there exists k ∈ [0,1) such that: d(T(x), T(y)) ≤ k · d(x,y) for all x,y ∈ X

The constant k is called the contraction constant or Lipschitz constant.

Banach Fixed Point Theorem (Contraction Mapping Theorem)

Let (X,d) be a complete metric space and T: X → X a contraction with constant k. Then:

  1. T has a unique fixed point x* ∈ X (i.e., T(x*) = x*)
  2. For any x₀ ∈ X, the sequence xₙ₊₁ = T(xₙ) converges to x*
  3. Error estimate: d(xₙ, x*) ≤ kⁿ/(1−k) · d(x₁, x₀)
Proof Sketch
  1. Show (xₙ) is Cauchy: d(xₙ,xₘ) ≤ kⁿ⁺¹/(1−k) · d(x₁,x₀) → 0
  2. By completeness, xₙ → x* for some x* ∈ X
  3. By continuity of T (contractions are continuous): T(x*) = T(lim xₙ) = lim T(xₙ) = lim xₙ₊₁ = x*
  4. Uniqueness: if T(y*) = y*, then d(x*,y*) = d(T(x*),T(y*)) ≤ k·d(x*,y*), so d(x*,y*) = 0

Applications

Solving Equations

Write f(x) = 0 as x = T(x) where T is a contraction. Picard's existence theorem for ODEs uses this on function spaces.

Iterative Methods

Newton's method, fixed-point iterations for solving f(x)=x. The convergence rate is geometric with ratio k.

Common Traps
  • Contractions on non-complete spaces may have no fixed point. f(x) = x/2 + 1/x on (0,∞) with metric |x-y| has no fixed point if domain is wrong.
  • Non-expansive maps (k=1) need not have fixed points. f(x) = x+1 on ℝ.
  • k must be strictly < 1. A map with d(Tx,Ty) < d(x,y) for x≠y (k=1 weak) may lack fixed points.
Example

T: [0,1] → [0,1] defined by T(x) = cos(x)/2 + 1/4 is a contraction (|T'(x)| = |sin(x)|/2 ≤ 1/2 < 1). By BFP, there's a unique x* ∈ [0,1] with cos(x*)/2 + 1/4 = x*. Starting from x₀ = 0: x₁ = 0.75, x₂ ≈ 0.659, x₃ ≈ 0.679, converging to x* ≈ 0.6739...

Metric Spaces for CSIR NET
📐

Metric Axioms

Non-negativity, symmetry, triangle inequality. d(x,y)=0 ⟺ x=y.

🔵

Open/Closed Sets

Open: contains ε-ball around each point. Closed: contains all limit points (sequential criterion).

📊

Cauchy vs Convergent

Convergent ⟹ Cauchy always. Cauchy ⟹ convergent only in complete spaces.

✅

Completeness

Cantor intersection: nested closed sets with diam→0 meet at exactly one point. Closed ⊆ complete = complete.

🔷

Compactness

Equivalent: open cover, sequential, complete + totally bounded. In ℝⁿ: closed + bounded.

🔗

Connectedness

No non-trivial clopen sets. Path-connected ⟹ connected. Subsets of ℝ: connected ⟺ interval.

🎯

Banach FPT

Complete + contraction k<1 ⟹ unique fixed point. Error: kⁿ/(1−k)·d(x₁,x₀).

🏛️

Baire Category

Complete ⟹ not a countable union of nowhere dense sets. Countable ∩ open dense = dense.
