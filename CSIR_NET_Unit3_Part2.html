Definition & Standard Pairs

Definition Basic

ℒ{f(t)} = F(s) = ∫₀^∞ e^(−st) f(t) dt (convergent for s > a = abscissa of convergence) Inverse: f(t) = ℒ⁻¹{F(s)} = (1/2πi) ∫_{c−i∞}^{c+i∞} e^(st) F(s) ds (Bromwich integral) In practice: use partial fractions + tables

Existence: f piecewise continuous and of exponential order |f(t)| ≤ Me^(at) → F(s) exists for s > a.

Key Transform Pairs Basic

ℒ{1} = 1/s (s > 0) ℒ{tⁿ} = n!/s^(n+1) (s > 0) ℒ{e^(at)} = 1/(s−a) (s > a) ℒ{sin bt} = b/(s²+b²) (s > 0) ℒ{cos bt} = s/(s²+b²) (s > 0) ℒ{sinh bt}= b/(s²−b²) (s > |b|) ℒ{cosh bt}= s/(s²−b²) (s > |b|)

Solving IVPs via Laplace Intermediate

Procedure for y''+py'+qy=g(t), y(0)=y₀, y'(0)=y₁: Step 1: Take ℒ of both sides (use IC directly): (s²Y−sy₀−y₁)+p(sY−y₀)+qY = G(s) Y(s)(s²+ps+q) = G(s)+sy₀+y₁+py₀ Step 2: Solve for Y(s) algebraically. Step 3: Use partial fractions to find ℒ⁻¹{Y(s)}. Advantage: ICs incorporated automatically!

Convolution & Special Functions Intermediate

Convolution: (f*g)(t) = ∫₀ᵗ f(τ)g(t−τ)dτ ℒ{f*g} = F(s)·G(s) (Convolution Theorem) Dirac delta: δ(t−a) ℒ{δ(t−a)} = e^(−as) ∫f(t)δ(t−a)dt = f(a) (sifting property) Heaviside (unit step): u(t−a) = 0 if t<a, 1 if t≥a ℒ{u(t−a)} = e^(−as)/s

Impulse response: h(t) = ℒ⁻¹{1/(s²+ps+q)} — solution with zero ICs and δ(t) forcing.

Power Series Method (Ordinary Points)

Power Series at Ordinary Point Basic

For y''+P(x)y'+Q(x)y=0, x₀ ordinary: Try y = Σ_{n=0}^∞ aₙ(x−x₀)ⁿ 1. Substitute into ODE 2. Collect terms by power of (x−x₀) 3. Set each coefficient to 0 → recurrence Radius of convergence ≥ distance from x₀ to nearest singular point (in complex plane)

Example at x₀=0: y''+y=0: recurrence aₙ₊₂=−aₙ/((n+1)(n+2)). Gives cos x and sin x.

Frobenius Method (Regular Singular Points) Intermediate

For x²y''+xP̃(x)y'+Q̃(x)y=0 with x=0 regular: Try y = xʳ Σ_{n=0}^∞ aₙxⁿ = Σ aₙx^(n+r) Indicial equation: r(r−1)+p₀r+q₀ = 0 (p₀=P̃(0), q₀=Q̃(0)) Roots r₁ ≥ r₂ Cases: r₁−r₂ ∉ ℤ: two independent Frobenius series r₁=r₂: second solution has ln x term r₁−r₂ ∈ ℤ⁺: second solution may have ln x
Special Equations

Bessel's Equation Advanced

x²y'' + xy' + (x²−n²)y = 0 Solutions: Jₙ(x) (Bessel of 1st kind), Yₙ(x) (Bessel of 2nd kind, singular at 0) J₀(x) = Σ (−1)ᵏ x²ᵏ / (2²ᵏ(k!)²) = 1 − x²/4 + x⁴/64 − ··· Jₙ(x) = Σ (−1)ᵏ(x/2)^(2k+n) / (k!(n+k)!) Recurrence: Jₙ₋₁+Jₙ₊₁ = (2n/x)Jₙ Orthogonality: ∫₀¹ xJₙ(αx)Jₙ(βx)dx = 0, α≠β

Zeros: J₀ has zeros at 2.405, 5.520, 8.654, … (used in BVPs on disks).

Legendre's Equation Advanced

(1−x²)y'' − 2xy' + n(n+1)y = 0 For n = non-negative integer: polynomial solution Pₙ(x) (Legendre polynomial) P₀=1, P₁=x, P₂=(3x²−1)/2, P₃=(5x³−3x)/2 Rodrigues' formula: Pₙ(x) = (1/2ⁿn!) dⁿ/dxⁿ [(x²−1)ⁿ] Orthogonality: ∫₋₁¹ Pₘ(x)Pₙ(x)dx = 2δₘₙ/(2n+1)

Generating function: (1−2xh+h²)^(−1/2) = Σ Pₙ(x)hⁿ.

Ordinary vs Singular PointBasic
For the ODE x(x−1)y'' + 2y' + xy = 0, which of the following is a regular singular point?
Ax = 0 only
Bx = 1 only
CBoth x = 0 and x = 1
DNeither x = 0 nor x = 1
✓ Correct Answer: (C) — Both x = 0 and x = 1
Divide by x(x−1): y''+2/(x(x−1))·y'+1/(x−1)·y=0. Singular points at x=0, x=1. Check x=0: xP=2x/(x(x−1))=2/(x−1)→−2 (analytic); x²Q=x²/(x−1)→0 (analytic) → x=0 regular singular. Check x=1: (x−1)P=2(x−1)/(x(x−1))=2/x→2 (analytic); (x−1)²Q=(x−1)²/(x−1)=(x−1)→0 (analytic) → x=1 regular singular. Both are regular singular points.

Linear Systems: Matrix Method

System Form & Solution Intermediate

System: X' = AX, X = [x,y]ᵀ, A = 2×2 matrix Step 1: Find eigenvalues λ from det(A−λI)=0. Step 2: For each λ, find eigenvector v: (A−λI)v=0. Step 3: Write solution: X(t) = C₁e^(λ₁t)v₁ + C₂e^(λ₂t)v₂ (distinct real λ) X(t) = C₁e^(λt)v + C₂e^(λt)(tv+w) (repeated λ) where (A−λI)w = v (generalised eigenvector) Complex λ=α±βi: use real and imaginary parts

Matrix Exponential e^(At) Advanced

e^(At) = Σ (At)ⁿ/n! = fundamental matrix Φ(t) Computed via: e^(At) = P·e^(Dt)·P⁻¹ (if A=PDP⁻¹) e^(Jt) = e^(λt)[I + (Jₒt) + (Jₒt)²/2 + ...] (Jordan form J, nilpotent part Jₒ) Non-homogeneous: X'=AX+F(t) X(t) = e^(At)X₀ + ∫₀ᵗ e^(A(t−s))F(s)ds

The fundamental matrix Φ(t)=e^(At) satisfies Φ(0)=I and Φ'=AΦ.

Lyapunov StabilityAdvanced
For the non-linear system x'=y, y'=−sin x, the equilibrium (0,0) is:
AAsymptotically stable
BUnstable
CStable but not asymptotically stable
DCannot be determined
✓ Correct Answer: (C) — Stable but not asymptotically stable
This is the pendulum equation (x=θ). Lyapunov function: V(x,y)=y²/2+(1−cos x)≥0 (energy). V̇=yy'+(sin x)x'=y(−sin x)+(sin x)y=0. Since V̇=0, V is constant (energy conserved). All trajectories are closed curves → solutions are periodic. Therefore (0,0) is stable but not asymptotically stable. Linearisation gives pure imaginary eigenvalues ±i (center), consistent.
