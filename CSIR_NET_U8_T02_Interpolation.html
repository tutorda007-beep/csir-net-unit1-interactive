Introduction

Given n+1 data points (x₀,y₀),…,(xₙ,yₙ) with distinct xᵢ, there exists a unique polynomial of degree ≤n passing through all points. Lagrange and Newton forms compute this polynomial; finite differences give computationally efficient formulas for equally-spaced nodes.

Lagrange Interpolation

Existence & Uniqueness

For n+1 distinct nodes x₀,…,xₙ, there is a unique polynomial Pₙ(x) of degree ≤n with Pₙ(xᵢ)=yᵢ for all i.

Lagrange Form
Pₙ(x) = Σ_{i=0}^n yᵢ · Lᵢ(x) Lᵢ(x) = Π_{j≠i} (x−xⱼ)/(xᵢ−xⱼ) (Lagrange basis polynomials)

Properties: Lᵢ(xⱼ)=δᵢⱼ (Kronecker delta), Σᵢ Lᵢ(x)≡1.

Example (3 points)

x₀=0,y₀=1; x₁=1,y₁=3; x₂=2,y₂=7. L₀=(x−1)(x−2)/((0−1)(0−2))=(x−1)(x−2)/2.

L₁=(x−0)(x−2)/((1−0)(1−2))=−x(x−2). L₂=x(x−1)/2.

P(x)=1·(x−1)(x−2)/2 + 3·(−x)(x−2) + 7·x(x−1)/2 = x²+x+1.

Newton Divided Differences

Divided Difference Table
f[xᵢ] = yᵢ f[xᵢ,xᵢ₊₁] = (f[xᵢ₊₁]−f[xᵢ])/(xᵢ₊₁−xᵢ) f[xᵢ,xᵢ₊₁,xᵢ₊₂] = (f[xᵢ₊₁,xᵢ₊₂]−f[xᵢ,xᵢ₊₁])/(xᵢ₊₂−xᵢ)
Newton's Divided Difference Formula
Pₙ(x) = f[x₀] + f[x₀,x₁](x−x₀) + f[x₀,x₁,x₂](x−x₀)(x−x₁) + … + f[x₀,…,xₙ](x−x₀)…(x−xₙ₋₁)

f[x₀,…,xₙ] = leading coefficient = coefficient of xⁿ in Pₙ. Adding a new point xₙ₊₁ only requires computing one new divided difference — easy to update.

Symmetric property

f[x₀,x₁,…,xₙ] is symmetric in its arguments: f[x₀,x₁]=f[x₁,x₀]. This means the formula gives the same polynomial regardless of node ordering.

Finite Differences (Equal Spacing)

Operators (step size h)
Forward: Δfᵢ = fᵢ₊₁ − fᵢ, Δ²fᵢ = fᵢ₊₂−2fᵢ₊₁+fᵢ Backward: ∇fᵢ = fᵢ − fᵢ₋₁ Central: δfᵢ = fᵢ₊½ − fᵢ₋½
Newton Forward Interpolation (equal spacing)
Pₙ(x₀+sh) = Σ_{k=0}^n C(s,k) Δᵏf₀ where C(s,k) = s(s−1)…(s−k+1)/k! (binomial coefficient in s)

Best for interpolation near x₀ (beginning of table).

Newton Backward Interpolation
Pₙ(xₙ+sh) = Σ_{k=0}^n (−1)^k C(−s,k) ∇ᵏfₙ

Best for interpolation near xₙ (end of table).

Interpolation Error

Error Formula
f(x) − Pₙ(x) = f^(n+1)(ξ)/(n+1)! · ω_{n+1}(x) ω_{n+1}(x) = (x−x₀)(x−x₁)…(x−xₙ) where ξ ∈ (min xᵢ, max xᵢ) depends on x.
Chebyshev Nodes (Minimize Error)

To minimize max|ω_{n+1}(x)| on [a,b], choose Chebyshev nodes:

xᵢ = (a+b)/2 + (b−a)/2 · cos((2i+1)π/(2(n+1))), i=0,…,n This gives: max_x |ω_{n+1}(x)| = (b−a)^(n+1) / (2^(2n+1))
Runge's Phenomenon

For equally-spaced nodes on [−1,1], the error can grow as n→∞ for functions like f(x)=1/(1+25x²). Chebyshev nodes avoid this. Alternatively: use splines.

Cubic Splines

Instead of one degree-n polynomial, use piecewise cubic polynomials (splines) on each subinterval, joined smoothly. Natural cubic spline: S″(x₀)=S″(xₙ)=0. Avoids Runge's phenomenon.

Summary

Uniqueness

n+1 distinct points ⟹ unique poly of degree ≤n. Both Lagrange and Newton give the same polynomial.

Lagrange Form

Pₙ=ΣyᵢLᵢ(x). Lᵢ(xⱼ)=δᵢⱼ, ΣLᵢ≡1. Conceptually simple.

Newton Form

Uses divided differences. Easy to update when new points added. Nested (Horner) form is efficient.

Finite Differences

Equal spacing: Δᵏf₀ replaces divided differences. Forward/backward formulas near x₀/xₙ.

Error Bound

f(x)−Pₙ(x)=f^(n+1)(ξ)/(n+1)!·ω_{n+1}(x). Minimized with Chebyshev nodes.

Cubic Splines

Piecewise cubic, C² smooth. Avoids Runge's phenomenon for large n.
