Introduction

Completeness — the property that every Cauchy sequence converges — is one of the most fundamental properties of metric spaces. The Baire Category Theorem is a powerful tool saying complete metric spaces cannot be "small", with applications to functional analysis, measure theory, and descriptive set theory.

Complete Metric Spaces

Complete Metric Space

A metric space (X,d) is complete if every Cauchy sequence in X converges to a point of X.

Recall: (xₙ) is Cauchy iff ∀ε>0, ∃N: m,n≥N ⟹ d(xₙ,xₘ)<ε.

Nested Sets / Cantor Intersection Theorem

(X,d) complete iff: for every sequence of non-empty closed sets F₁⊇F₂⊇… with diam(Fₙ)→0, we have ⋂Fₙ = {single point}.

Completion

Every metric space (X,d) has a completion (X̃, d̃) — a complete metric space containing X as a dense subspace. The completion is unique up to isometry. Example: ℝ is the completion of ℚ.

Completeness is NOT Topological

Completeness depends on the metric, not just the topology. (0,1) and ℝ are homeomorphic but (0,1) is incomplete and ℝ is complete. However: a Gδ subset of a complete space is completely metrizable (Alexandrov's theorem).

Baire Category Theorem

Nowhere Dense / Category Definitions

A set A ⊆ X is nowhere dense if int(cl(A)) = ∅ (its closure has empty interior).

A is meager (or first category) if it is a countable union of nowhere dense sets.

A is comeager (or second category / residual) if X \ A is meager.

Baire Category Theorem

If X is a complete metric space (or a locally compact Hausdorff space), then:

X is not meager in itself: X cannot be written as a countable union of nowhere dense sets.

Equivalently: a countable intersection of dense open sets is dense.

Applications

Application 1: ℝ is uncountable

If ℝ = {x₁, x₂,…}, then ℝ = ⋃{xₙ}, each singleton {xₙ} nowhere dense. But ℝ is complete, so by BCT it cannot be a countable union of nowhere dense sets. Contradiction.

Application 2: Continuous nowhere differentiable functions exist

The set of functions in C[0,1] differentiable at some point is meager. Hence "most" (comeager set of) continuous functions are nowhere differentiable.

Application 3: Uniform Boundedness (Banach-Steinhaus)

If {Tₙ} are continuous linear maps on Banach space X with sup_n |Tₙ(x)| < ∞ for each x, then sup_n ‖Tₙ‖ < ∞.

Key Corollary

In a complete metric space X, every comeager set is dense. So if Gₙ are open dense sets, then ⋂Gₙ is dense in X.

Totally Bounded & Compact

Totally Bounded

(X,d) is totally bounded if for every ε>0, X can be covered by finitely many balls of radius ε.

Every totally bounded metric space is bounded, but the converse fails (e.g., infinite-dimensional unit ball).

Characterization of Compact Metric Spaces

A metric space (X,d) is compact iff it is complete AND totally bounded.

Equivalently: compact iff every sequence has a convergent subsequence (sequentially compact).

In ℝⁿ: compact ↔ closed and bounded (Heine-Borel).

Sequential Compactness

In metric spaces, compactness, sequential compactness, and countable compactness are all equivalent.

Contraction Mapping Theorem

Contraction

f: X→X is a contraction (with constant k<1) if:

d(f(x), f(y)) ≤ k · d(x,y) for all x,y∈X, 0≤k<1
Banach Fixed Point Theorem (Contraction Mapping Theorem)

If (X,d) is a complete metric space and f: X→X is a contraction with constant k<1, then:

  1. f has a unique fixed point x* (f(x*)=x*).
  2. For any x₀∈X, the sequence x_{n+1} = f(xₙ) converges to x*.
  3. Error estimate: d(xₙ, x*) ≤ kⁿ/(1−k) · d(x₁,x₀).

Applications

  • Existence of ODE solutions: Picard-Lindelöf theorem — the Picard iteration is a contraction on C[t₀−h, t₀+h].
  • Newton's method: convergence near a simple root.
  • Implicit Function Theorem: proof uses Banach fixed point.
  • Integral equations: Volterra equation solutions via iteration.
Lipschitz vs Contraction

f is Lipschitz if d(f(x),f(y)) ≤ L·d(x,y) for some L (not necessarily <1). Contractions are Lipschitz with L<1. Lipschitz maps are uniformly continuous.
