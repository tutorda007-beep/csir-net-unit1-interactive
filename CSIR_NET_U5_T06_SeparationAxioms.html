Introduction

Separation axioms describe how well a topological space can separate points and sets by open sets. The hierarchy T₀ ⊆ T₁ ⊆ T₂ ⊆ T₃ ⊆ T₄ gives progressively stronger separation. Most spaces in analysis (metric spaces) are T₄ (normal). The Urysohn lemma and Tietze extension are fundamental tools in analysis and topology.

Separation Axioms T₀ through T₄

AxiomNameCondition
T₀KolmogorovFor any two distinct points x≠y, ∃ open set containing one but not the other
T₁FréchetFor x≠y, ∃ open set containing x but not y AND one containing y but not x
T₂HausdorffFor x≠y, ∃ disjoint open sets U∋x and V∋y
T₃RegularT₁ + any closed set F and point x∉F can be separated by disjoint open sets
T₄NormalT₁ + any two disjoint closed sets can be separated by disjoint open sets
T₁ Equivalent

X is T₁ iff every singleton {x} is a closed set. Equivalently: for every x, ⋂{U open : x∈U} = {x}.

T₂ (Hausdorff)

In a Hausdorff space: limits of sequences are unique; compact subsets are closed; continuous maps to Hausdorff spaces are closed on compact domains.

Hierarchy

Every metric space is T₄ (normal). T₄ ⟹ T₃ ⟹ T₂ ⟹ T₁ ⟹ T₀. None of these implications reverses in general.

Normal Spaces & Urysohn's Lemma

Normal Space (T₄)

X is normal if it is T₁ and: for any two disjoint closed sets A, B ⊆ X, there exist disjoint open sets U⊇A and V⊇B.

Every metric space is normal. Every compact Hausdorff space is normal.

Urysohn's Lemma

X is normal iff: for any two disjoint closed sets A, B, there exists a continuous function f: X→[0,1] with f≡0 on A and f≡1 on B.

Such f is called an Urysohn function separating A and B.

Proof Idea

Construct f by assigning rational values to nested open sets: U_{p} for p∈ℚ∩[0,1] with A⊆U_p and cl(U_p)⊆U_q for p<q, B⊆X\U_1. Set f(x)=inf{p: x∈U_p}.

Urysohn Metrization Theorem

Every regular (T₃) second-countable topological space is metrizable (can be given a metric inducing the same topology).

Every separable metrizable space embeds into [0,1]^ℕ (Hilbert cube).

Tietze Extension Theorem

Tietze Extension Theorem

X is normal iff: for every closed subset A⊆X and every continuous f: A→[a,b] (or f: A→ℝ), there exists a continuous extension F: X→[a,b] (resp. F: X→ℝ) with F|_A = f.

Equivalently: normal ↔ every continuous function on a closed subspace extends to the whole space.

Comparison with Urysohn

Urysohn: gives ONE specific function separating two closed sets.
Tietze: extends ANY continuous function from a closed subset.
Both require normality; Tietze is derived using Urysohn's lemma.

Partition of Unity

On a paracompact Hausdorff space (e.g., any metric space, any manifold), for every open cover {Uα} there exists a partition of unity {φα} subordinate to the cover: continuous φα≥0, support(φα)⊆Uα, Σφα=1.

Used to: patch local constructions globally, define integrals on manifolds.

Compact Hausdorff Spaces

Every compact Hausdorff space is normal (T₄). Proof: use compactness to build the separating open sets from the T₂ separation of individual points.

Second Countability + T₃ = Metrizable

Urysohn Metrization: second-countable T₃ ⟹ metrizable. In practice: separable metric spaces are second-countable, hence the two notions align.

Summary

T₀–T₄ Hierarchy

T₄⟹T₃⟹T₂⟹T₁⟹T₀. Every metric space is T₄.

Hausdorff (T₂)

Distinct points separated by disjoint opens. Unique limits. Compact subsets closed.

Normal (T₄)

Disjoint closed sets separated by disjoint opens. Metric spaces and compact Hausdorff are normal.

Urysohn Lemma

Normal ↔ ∃ Urysohn function f:X→[0,1] with f=0 on A, f=1 on B (disjoint closed A,B).

Tietze Ext.

Normal ↔ every continuous f on closed A⊆X extends to F:X→ℝ.

Urysohn Metrization

T₃ + second countable ⟹ metrizable ⟹ embeds in Hilbert cube [0,1]^ℕ.
