Introduction

Quadrature (numerical integration) approximates ∫_a^b f(x)dx using function values at selected nodes. Newton-Cotes rules use equally-spaced nodes; Gauss quadrature chooses optimal nodes and weights, achieving high accuracy with few evaluations.

Simpson's Rule (key)
∫_a^b f dx ≈ (b−a)/6 · [f(a) + 4f((a+b)/2) + f(b)] Error: −(b−a)⁵/(2880) · f⁴(ξ) (exact for cubics!)

Composite Rules

Setup

Divide [a,b] into n equal subintervals of width h=(b−a)/n. Apply basic rule on each subinterval and sum.

MethodFormulaGlobal Error
Composite Midpointh·Σf(xᵢ₊½)O(h²)
Composite Trapezoidal(h/2)[f(x₀)+2Σf(xᵢ)+f(xₙ)]O(h²)
Composite Simpson's(h/3)[f₀+4f₁+2f₂+4f₃+…+4f_{n−1}+fₙ]O(h⁴)
Error Formulas
Composite Trapezoidal: E = −(b−a)h²f″(ξ)/12 = O(h²) Composite Simpson's: E = −(b−a)h⁴f⁴(ξ)/180 = O(h⁴)

Simpson's requires n even (pairs of subintervals).

Gauss Quadrature

Gauss-Legendre Quadrature
∫_{-1}^1 f(x)dx ≈ Σ_{i=1}^n wᵢ f(xᵢ) Degree of precision: 2n−1 (exact for polynomials of degree ≤ 2n−1)

The nodes xᵢ are roots of the Legendre polynomial Pₙ(x), and weights wᵢ are determined by exactness conditions.

General Interval
∫_a^b f(x)dx = (b−a)/2 ∫_{-1}^1 f((b+a)/2+(b−a)t/2) dt ≈ (b−a)/2 Σwᵢf(xᵢ)

Numerical Differentiation

Finite Difference Formulas
Forward: f′(x) ≈ [f(x+h)−f(x)]/h, error O(h) Backward: f′(x) ≈ [f(x)−f(x−h)]/h, error O(h) Central: f′(x) ≈ [f(x+h)−f(x−h)]/(2h), error O(h²) Central 2: f″(x) ≈ [f(x+h)−2f(x)+f(x−h)]/h², error O(h²)
Richardson Extrapolation

If D(h) = f′(x)+c₂h²+c₄h⁴+…, then:

D_rich = [4D(h/2)−D(h)]/3 has error O(h⁴)

Cancels the h² error term. Romberg integration applies this to the trapezoidal rule.

Step Size Trade-off

Total error = truncation error + round-off error. As h↓: truncation ↓ but round-off ↑ (subtracting nearly equal numbers). Optimal h minimizes total error: h_opt ≈ √(ε_mach) for central differences.

Summary

Trapezoidal

(h/2)[f₀+2f₁+…+fₙ]. O(h²). Degree 1. Exact for linears.

Simpson's

(h/3)[f₀+4f₁+2f₂+…+fₙ]. O(h⁴). Degree 3. Exact for cubics.

Gauss-Legendre

n points ⟹ degree 2n−1. Nodes=roots of Pₙ. Most efficient per evaluation.

Central Difference

f′≈[f(x+h)−f(x−h)]/(2h), O(h²). f″≈[f(x+h)−2f(x)+f(x−h)]/h², O(h²).

Richardson

D_rich=[4D(h/2)−D(h)]/3. Upgrades O(h²)→O(h⁴).

Step Size

Optimal h balances truncation vs round-off. For central: h_opt≈ε_mach^(1/3).
