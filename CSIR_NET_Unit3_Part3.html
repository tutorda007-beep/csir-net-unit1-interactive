Lagrange's (Quasi-linear) Method

Lagrange's Method Basic

Quasi-linear 1st order PDE: P(x,y,z)p + Q(x,y,z)q = R(x,y,z) where p=∂z/∂x, q=∂z/∂y Lagrange's Auxiliary Equations: dx/P = dy/Q = dz/R From these, find two independent integrals: u(x,y,z)=C₁, v(x,y,z)=C₂ General solution: F(u,v) = 0 or equivalently: u = f(v)

Key: From dx/P=dy/Q=dz/R, extract two first integrals using groupings, multipliers, or direct integration.

Method of Characteristics Intermediate

For a(x,y)u_x + b(x,y)u_y = c(x,y,u): Characteristic ODEs: dx/ds = a(x,y) dy/ds = b(x,y) du/ds = c(x,y,u) Solve characteristic curves (x(s),y(s)). u is constant along characteristics if c=0. IVP: u(x,0)=g(x) Characteristics through (x₀,0) carry initial data

Geometric view: Characteristics are curves along which the PDE reduces to an ODE. Information propagates along them.

Linear & Non-linear First Order PDEs

Linear First Order PDE Basic

Form: a(x,y)u_x + b(x,y)u_y + c(x,y)u = d(x,y) Example: xu_x + yu_y = u Auxiliary: dx/x = dy/y = du/u From dx/x = dy/y: y/x = C₁ From dx/x = du/u: u/x = C₂ General solution: u/x = f(y/x) i.e., u = x·f(y/x)

Homogeneous PDE: d=0. Solution space is a vector space (superposition holds).

Charpit's Method (Non-linear) Advanced

Non-linear: F(x,y,z,p,q) = 0 Charpit's auxiliary equations: dp/(F_x+pF_z) = dq/(F_y+qF_z) = dz/(−pF_p−qF_q) = dx/(−F_p) = dy/(−F_q) Find one relation between p and q satisfying F=0, then integrate dz = p dx + q dy (complete integral). Example: p² + q² = 1 Charpit → q = constant a Then p = √(1−a²), z = x√(1−a²) + ay + b

Lagrange's MethodBasic
The general solution of xp + yq = z is:
Az = f(x+y)
Bz = f(x/y)
Cz = x·f(y/x)
Dz = f(xy)
✓ Correct Answer: (C) — z = x·f(y/x)
Auxiliary equations: dx/x = dy/y = dz/z. From dx/x=dy/y: integrating gives ln y − ln x = C₁ → y/x=C₁. From dx/x=dz/z: ln z − ln x = C₂ → z/x=C₂. Two integrals: u=y/x, v=z/x. General solution: F(y/x, z/x)=0 → z/x = f(y/x) → z = x·f(y/x). This describes all homogeneous functions of degree 1.

Transport EquationBasic
The general solution of u_t + 3u_x = 0 with u(x,0) = sin x is:
Asin(x − 3t)
Bsin(x + 3t)
Ce^(−3t) sin x
Dsin(3x − t)
✓ Correct Answer: (A) — sin(x − 3t)
Transport equation u_t + cu_x = 0 has general solution u=f(x−ct). Here c=3, so u=f(x−3t). Applying IVP: f(x)=sin x. Solution: u = sin(x−3t). The wave travels to the right with speed c=3. Characteristics: x−3t = const (straight lines in x-t plane with slope 1/3).

Auxiliary EquationsMedium
For the PDE (y−z)p + (z−x)q = x−y, which of the following is a first integral?
Ax + y + z
Bx² + y² + z²
Cxy + yz + zx
Dxyz
✓ Correct Answer: (A) — x + y + z
Auxiliary: dx/(y−z)=dy/(z−x)=dz/(x−y). Use multiplier (1,1,1): numerator = (y−z)+(z−x)+(x−y)=0, denominator = dx+dy+dz → d(x+y+z)=0 → x+y+z=C₁. For a second integral, use multiplier (x,y,z): numerator=x(y−z)+y(z−x)+z(x−y)=0, denominator=x dx+y dy+z dz → d(x²+y²+z²)/2=0 → x²+y²+z²=C₂. General solution: F(x+y+z, x²+y²+z²)=0.

Non-linear PDEMedium
A complete integral of p² − q² = x − y is:
Az = (x−a)^(3/2)/3 − (y+a)^(3/2)/3 + b
Bz = ax − ay + b
Cz = (x+a²)^(3/2)/3 − (y−a²)^(3/2)/3 + b
Dz = √(x−a) + √(y+a) + b
✓ Correct Answer: (A) — z = (x−a)^(3/2)/3 − (y+a)^(3/2)/3 + b
Try p²=x+a²... Actually, assume p²=x+f(a) and q²=y+g(a) so that p²−q²=(x+f)−(y+g)=x−y requires f(a)=−g(a)=a. So p²=x+a → p=(x+a)^(1/2), q²=y+a → q=(y+a)^(1/2). But dz=pdx+qdy = √(x+a)dx+√(y+a)dy → z=(x+a)^(3/2)/(3/2)+(y+a)^(3/2)/(3/2)+b. Closer to standard: try p²=x−a, q²=y−a: p²−q²=(x−a)−(y−a)=x−y ✓. Then z=∫√(x−a)dx−∫√(y−a)dy=(x−a)^(3/2)/(3/2)−(y−a)^(3/2)/(3/2)+b = (2/3)(x−a)^(3/2)−(2/3)(y−a)^(3/2)+b, i.e. option (A) with factor adjustment.

Shock Formation (Burgers)Advanced
For u_t + uu_x = 0 with u(x,0) = 1/(1+x²), characteristics emanating from (x₀,0) are:
Ax = x₀ + t/(1+x₀²)
Bx = x₀ + t
Cx = x₀·e^t
Dx = x₀ − t/(1+x₀²)
✓ Correct Answer: (A) — x = x₀ + t/(1+x₀²)
On each characteristic, u = const = u(x₀,0) = 1/(1+x₀²). Characteristic equation: dx/dt = u = 1/(1+x₀²). So x = x₀ + t·u(x₀,0) = x₀ + t/(1+x₀²). Characteristics are straight lines with slope 1/u(x₀). Shocks form when characteristics cross, which happens when ∂x/∂x₀=0 at some finite t. Since u is decreasing for x₀>0, right-moving characteristics have slower speed → characteristics from different x₀ will eventually cross.

Reduction to Canonical Form Advanced

For Au_xx+Bu_xy+Cu_yy = F(x,y,u,p,q): Characteristic equations: A(dy)²−B(dx)(dy)+C(dx)²=0 → dy/dx = (B±√(B²−4AC))/(2A) New variables ξ=ξ(x,y), η=η(x,y): (ξ,η) along characteristic directions Hyperbolic: ξ=const, η=const are 2 characteristics → u_ξη = Φ Parabolic: one repeated characteristic family → u_ηη = Φ Elliptic: complex characteristics → ξ+iη → u_ξξ+u_ηη = Φ

Well-Posedness by Type Intermediate

Elliptic (Laplace/Poisson): Well-posed: Dirichlet or Neumann BVP NOT well-posed as IVP (Cauchy-Kowalewski fails) Parabolic (Heat equation): Well-posed: IVP forward in time t>0 Ill-posed: backward in time (t<0) Hyperbolic (Wave equation): Well-posed: IVP (Cauchy problem) in t>0 Finite speed of propagation Domain of dependence / influence

Hadamard's conditions: Existence, Uniqueness, Continuous dependence on data.

The PDE u_xx + 4u_xy + 4u_yy = 0 is:
AHyperbolic
BParabolic
CElliptic
DType changes with x,y
✓ Correct Answer: (B) — Parabolic
A=1, B=4, C=4. Discriminant Δ=B²−4AC=16−16=0 → Parabolic. The equation factors: (∂/∂x+2∂/∂y)²u=0 → it has a repeated characteristic direction dy/dx=2. This can be reduced to v_ηη=0 (where η=y−2x), giving v=ηf(ξ)+g(ξ) for new variables ξ=y−2x, η=x.

Variable TypeMedium
The PDE xu_xx + u_yy = 0 is hyperbolic where:
Ax > 0
Bx < 0
Cx = 0
DAll x ≠ 0
✓ Correct Answer: (B) — x < 0
A=x, B=0, C=1. Δ=B²−4AC=0−4x=−4x. Δ>0 (hyperbolic) ↔ −4x>0 ↔ x<0. Δ=0 (parabolic) ↔ x=0. Δ<0 (elliptic) ↔ x>0. This is Tricomi's equation (type changes across x=0): elliptic for x>0, parabolic on x=0, hyperbolic for x<0.

Canonical FormMedium
The canonical form of u_xx − u_yy = 0 (hyperbolic) using ξ=x+y, η=x−y is:
Au_ξξ − u_ηη = 0
Bu_ξη = 0
Cu_ξξ + u_ηη = 0
Du_ξ + u_η = 0
✓ Correct Answer: (B) — u_ξη = 0
With ξ=x+y, η=x−y: u_x=u_ξ+u_η, u_y=u_ξ−u_η. u_xx=u_ξξ+2u_ξη+u_ηη; u_yy=u_ξξ−2u_ξη+u_ηη. So u_xx−u_yy=4u_ξη=0 → u_ξη=0. General solution: u=f(ξ)+g(η)=f(x+y)+g(x−y) — D'Alembert's solution for the wave equation.

Well-posednessAdvanced
For the backward heat equation u_t = −u_xx (t < 0), the Cauchy problem is:
AWell-posed for smooth initial data
BIll-posed (solutions can blow up)
CWell-posed only for analytic data
DWell-posed if initial data is periodic
✓ Correct Answer: (B) — Ill-posed (solutions can blow up)
The backward heat equation is ill-posed (Hadamard's example). Consider u(x,0)=sin(nx)/n → u(x,t)=sin(nx)e^(n²|t|)/n. For any t>0 and large n, e^(n²t)/n→∞ even though initial data → 0. Small changes in IC cause arbitrarily large changes in solution — no continuous dependence. This is why heat diffusion is irreversible and "unsmoothing" is unstable.

Mixed TypeAdvanced
The PDE (1−M²)u_xx + u_yy = 0 (where M is Mach number) changes type when:
AM = 0
BM = 1
CM = 2
DM = ∞
✓ Correct Answer: (B) — M = 1
A=1−M², B=0, C=1. Δ=−4(1−M²)=4M²−4. Δ<0 (elliptic) if M<1 (subsonic flow). Δ=0 (parabolic) if M=1 (sonic). Δ>0 (hyperbolic) if M>1 (supersonic flow). This is the linearised potential equation for compressible aerodynamics — the transition at M=1 corresponds to the sound barrier.

The Three Fundamental PDEs

Heat Equation Intermediate

u_t = k·u_xx (x∈[0,L], t>0) BCs: u(0,t)=0, u(L,t)=0 (Dirichlet) IC: u(x,0) = f(x) Separation: u=X(x)T(t) X''/X = T'/(kT) = −λ² (negative!) X_n = sin(nπx/L), λ_n = nπ/L T_n = e^(−k(nπ/L)²t) Solution: u(x,t) = Σ bₙ sin(nπx/L) e^(−kn²π²t/L²) bₙ = (2/L)∫₀ᴸ f(x)sin(nπx/L)dx

Properties: Solution is instantly smooth (infinite speed of propagation); energy ∫u²dx decreases.

Wave Equation Intermediate

u_tt = c²u_xx (x∈[0,L], t>0) BCs: u(0,t)=0, u(L,t)=0 ICs: u(x,0)=f(x), u_t(x,0)=g(x) Separation: u=X(x)T(t) X''/X = T''/(c²T) = −λ² Solution (standing waves): u=Σ(aₙcos(nπct/L)+bₙsin(nπct/L))sin(nπx/L) D'Alembert's solution (infinite domain): u(x,t) = [f(x+ct)+f(x−ct)]/2 + (1/2c)∫_{x−ct}^{x+ct} g(s) ds

Domain of dependence: u(x,t) depends only on initial data in [x−ct, x+ct].

Laplace & Poisson Equations Intermediate

Laplace: ∇²u = u_xx+u_yy = 0 (elliptic, steady-state) Poisson: ∇²u = f(x,y) (with source) Solutions are harmonic functions. Key properties: Mean value property: u(x₀) = (1/|∂B|)∫_{∂B} u dS Maximum principle: max of u on Ω̄ is on ∂Ω Uniqueness: Dirichlet BVP has at most one solution Solution on disk r

Green's function: G(x,ξ) satisfies ∇²G=−δ(x−ξ). Solution: u(x)=∫f(ξ)G(x,ξ)dξ + boundary terms.

Fourier Series & Separation of Variables

Fourier Series Intermediate

f(x) = a₀/2 + Σₙ₌₁^∞ (aₙcos(nπx/L)+bₙsin(nπx/L)) Coefficients (period 2L): a₀ = (1/L)∫₋ₗᴸ f(x)dx aₙ = (1/L)∫₋ₗᴸ f(x)cos(nπx/L)dx bₙ = (1/L)∫₋ₗᴸ f(x)sin(nπx/L)dx Parseval's theorem: (1/L)∫₋ₗᴸ |f(x)|² dx = a₀²/2 + Σ(aₙ²+bₙ²)

Separation of Variables Steps Basic

General procedure: 1. Assume u(x,t) = X(x)·T(t) 2. Substitute into PDE → separate: ODE in X (with BCs) ODE in T 3. Find eigenvalues λₙ from X-BVP 4. Solve T-equation for each λₙ 5. Write u = Σ cₙXₙ(x)Tₙ(t) 6. Apply IC to find coefficients cₙ (using orthogonality of {Xₙ})

Works for: linear, homogeneous PDEs on simple domains (rectangle, disk, ball) with linear homogeneous BCs.

Maximum PrincipleMedium
If u is harmonic (∇²u=0) on a bounded domain Ω with u=0 on ∂Ω, then:
Au = 0 throughout Ω
Bu could be nonzero inside Ω
Cu achieves its max at an interior point
Du is necessarily constant
✓ Correct Answer: (A) — u = 0 throughout Ω
By the strong maximum principle: a non-constant harmonic function cannot achieve its maximum or minimum in the interior of Ω. Since u=0 on ∂Ω, max u on Ω̄ = 0 (achieved on boundary) and min u = 0. Therefore 0 ≤ u ≤ 0 throughout Ω, giving u≡0. This proves uniqueness for the Dirichlet problem: two solutions with the same BC must agree.

Mean Value PropertyAdvanced
If u is harmonic in ℝ², then the value of u at the centre of any circle of radius r is:
AThe maximum of u on the circle
BThe average of u on the circle
CThe minimum of u on the circle
DZero regardless of u
✓ Correct Answer: (B) — The average of u on the circle
Mean Value Property: if u is harmonic at x₀, then u(x₀) = (1/2πr)∮_{|x−x₀|=r} u dS for all r where u is harmonic. In ℝ³: u(x₀)=(1/4πr²)∯ u dS. This is a fundamental property of harmonic functions, equivalent to the maximum principle, and implies harmonic functions are real analytic.
